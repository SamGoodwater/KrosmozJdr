# 2.4.5. Maîtrises et Expertises

**Description** : Explique le système de maîtrise et d'expertise des compétences, langues, outils et métiers, inspiré de D&D.

> **Règles rapides - Maîtrises et Expertises**

## 2.4.5.1. Bonus de maîtrise

Le **bonus de maîtrise** reflète votre degré de compétence générale et augmente avec votre niveau.

### Calcul

**Formule** : Bonus de maîtrise = 1 + (Niveau / 4) arrondi à l'inférieur

**Progression par niveau** :
- Niveau 1-3 : **+1**
- Niveau 4-7 : **+2**
- Niveau 8-11 : **+3**
- Niveau 12-15 : **+4**
- Niveau 16-19 : **+5**
- Niveau 20 : **+6**

### Application

Le bonus de maîtrise s'ajoute à :
- **Tests de compétences maîtrisées** : Si vous maîtrisez une compétence, ajoutez votre bonus de maîtrise
- **Jets de sauvegarde maîtrisés** : Si vous maîtrisez une sauvegarde, ajoutez votre bonus de maîtrise
- **Certaines aptitudes** : Certaines aptitudes utilisent le bonus de maîtrise
- **Réserve de Wakfu** : Le bonus de maîtrise détermine votre réserve de Wakfu

### Exemple

Un personnage niveau 10 avec :
- Force +3
- Maîtrise en Athlétisme
- Bonus de maîtrise : +3

**Test d'Athlétisme** : 1d20 + 3 (Force) + 3 (Maîtrise) = 1d20 + 6

## 2.4.5.2. Expertise

L'**expertise** double votre bonus de maîtrise pour une compétence spécifique.

### Conditions d'obtention

Vous pouvez obtenir une expertise aux niveaux :
- **Niveau 9** : Première expertise
- **Niveau 15** : Deuxième expertise
- **Niveau 20** : Troisième expertise

**Total** : Maximum **3 expertises** au niveau 20

### Fonctionnement

**Exigence** : Vous devez déjà **maîtriser** la compétence pour obtenir l'expertise.

**Effet** : Votre bonus de maîtrise est **doublé** pour cette compétence.

### Exemple

Un personnage niveau 15 avec :
- Agilité +4
- Maîtrise en Discrétion
- Expertise en Discrétion
- Bonus de maîtrise : +4

**Test de Discrétion** : 1d20 + 4 (Agilité) + 8 (Expertise : 2 × 4) = 1d20 + 12

### Choix stratégiques

L'expertise est un choix important :
- **Renforce vos forces** : Doublez vos compétences les plus importantes
- **Compense vos faiblesses** : Expertisez une compétence que vous utilisez souvent mais avec une caractéristique faible
- **Crée des spécialisations** : Devenez expert dans un domaine spécifique

## 2.4.5.3. Compétences

Les compétences représentent l'entraînement dans des domaines spécifiques.

### Liste des compétences

#### Force
- **Athlétisme** : Sauter, escalader, nager, soulever
- **Intimidation** : Effrayer, menacer, imposer

#### Agilité
- **Acrobaties** : Équilibre, voltige, roulade
- **Discrétion** : Se cacher, se faufiler
- **Escamotage** : Voler, tricher, crocheter

#### Intelligence
- **Arcanes** : Connaissance de la magie
- **Histoire** : Connaissance de l'histoire
- **Investigation** : Examiner, analyser, déduire
- **Nature** : Connaissance de la nature
- **Religion** : Connaissance des religions

#### Sagesse
- **Dressage** : Manipuler les animaux
- **Médecine** : Soigner, diagnostiquer
- **Perception** : Remarquer, écouter, observer
- **Perspicacité** : Détecter les mensonges, comprendre les motivations
- **Survie** : Suivre des pistes, chasser, se nourrir

#### Chance
- **Persuasion** : Convaincre, négocier
- **Représentation** : Jouer, chanter, danser
- **Supercherie** : Mentir, déguiser, tromper

### Affectation par spécialisation

Chaque spécialisation donne accès à certaines compétences :

**Érudit** : Arcanes, Histoire, Investigation, Religion
**Milicien** : Athlétisme, Intimidation, Perception
**Dévot** : Médecine, Religion, Perspicacité
**Artiste** : Représentation, Persuasion, Supercherie
**Négociant** : Persuasion, Supercherie, Perspicacité
**Explorateur** : Survie, Perception, Nature
**Sylvain** : Nature, Dressage, Survie
**Artisan** : Investigation, Perception (pour identifier les matériaux)
**Voleur** : Discrétion, Escamotage, Supercherie

**Note** : Cette liste est indicative. Consultez la bibliothèque pour les détails complets.

## 2.4.5.4. Langues et outils

### Langues

Les langues que vous connaissez sont déterminées par :
- **Votre historique** : Votre origine peut vous donner des langues
- **Votre spécialisation** : Certaines spécialisations donnent accès à des langues
- **Vos choix** : Vous pouvez choisir des langues supplémentaires

**Langues de base** :
- **Commun** : Tous les personnages connaissent le commun

**Langues supplémentaires** :
- **Langues régionales** : Selon votre région d'origine
- **Langues anciennes** : Si vous avez étudié
- **Langues spécialisées** : Langues de créatures, langues secrètes

### Outils

Les outils sont des équipements spécialisés utilisés pour :
- **Artisanat** : Créer, réparer, améliorer
- **Exploration** : Détecter, analyser, naviguer
- **Social** : Négocier, persuader, divertir

**Types d'outils** :
- **Outils d'artisanat** : Forge, couture, bijouterie, etc.
- **Outils d'exploration** : Boussole, cartes, outils de crochetage
- **Outils sociaux** : Instruments de musique, outils de jeu

**Maîtrise d'outils** :
- Vous pouvez **maîtriser** des outils
- La maîtrise vous donne un **bonus** lors de leur utilisation
- Certaines spécialisations donnent accès à des outils spécifiques

## 2.4.5.5. Métiers

Les métiers sont des compétences spécialisées qui permettent de créer, réparer ou améliorer des objets.

### Accès aux métiers

Vous pouvez accéder aux métiers via :
- **Votre spécialisation** : Certaines spécialisations donnent accès à des métiers
- **Vos choix** : Vous pouvez choisir des métiers supplémentaires
- **Votre historique** : Votre passé peut vous donner des métiers

### Types de métiers

- **Forge** : Créer des armes, armures, outils
- **Couture** : Créer des vêtements, capes, chapeaux
- **Bijouterie** : Créer des amulettes, anneaux
- **Alchimie** : Créer des potions, poisons
- **Cordonnerie** : Créer des bottes
- **Et bien d'autres...**
