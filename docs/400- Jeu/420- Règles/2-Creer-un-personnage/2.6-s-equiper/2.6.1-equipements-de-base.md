## 2.6.1.1. Armes

Les armes s'utilisent par l'intermédiaire des sorts. Utiliser une arme directement est très rarement bénéfique par rapport aux sorts, mais peut être utile dans certaines situations.

### Types d'armes

| Arme | Dégâts | Coût (PA) | Portée |
|------|--------|-----------|--------|
| **Main nue** | 1d4 + niveau/4 | 2 | 1 |
| **Arc** | 1d4 + niveau/4 | 4 | 1-6 |
| **Bâtons** | 1d4 + niveau/4 | 4 | 1-2 |
| **Baguettes** | 1d4 + niveau/4 | 3 | 1-4 |
| **Dagues** | 1d6 + niveau/4 | 4 | 1 |
| **Épées** | 1d6 + niveau/4 | 4 | 1 |
| **Marteaux** | 1d6 + niveau/4 | 4 | 1 |
| **Pelles** | 1d6 + niveau/4 | 4 | 1-2 |
| **Haches** | 1d6 + niveau/4 | 4 | 1 |
| **Pioches** | 1d8 + niveau/4 | 5 | 1-2 |
| **Faux** | 1d4 + niveau/4 | 5 | 1-2 |
| **Arbalètes** | 1d8 + niveau/4 | 6 | 1-5 |
| **Armes magiques** | 1d12 + niveau/4 | 7 | 1-8 |

### Caractéristiques des armes

- **Dégâts** : Les dégâts augmentent avec le niveau du personnage
- **Coût en PA** : Utiliser une arme coûte des PA
- **Portée** : Distance à laquelle l'arme peut être utilisée
- **Élément** : Les armes peuvent être associées à un élément (Terre, Feu, Air, Eau, Neutre)

### Armes improvisées

Il peut arriver que les personnages n'aient pas leurs armes avec eux et doivent se contenter de ce qu'ils ont sous la main.

**Règles** :
- Une arme improvisée peut être n'importe quel objet que vous pouvez tenir à une ou deux mains
- Exemples : Éclat de verre, pied de table, poêle, roue de chariot, pandawa ivre-mort
- Si l'objet ressemble à une arme réelle, il peut être traité comme telle (ex: pied de table = bâton)
- Le MJ peut décider qu'un personnage qui maîtrise une arme peut utiliser son bonus de maîtrise
- Un objet qui ne ressemble à aucune arme inflige **1d4 dégâts neutres**

**Note** : Pour les détails complets des armes (bonus, restrictions, etc.), consultez la **Bibliothèque → Équipement** sur le site web.

## 2.6.1.2. Équipements

Les équipements principaux se composent de plusieurs types, chacun donnant des bonus spécifiques.

### Types d'équipements

#### Chapeaux

**Bonus possibles** :
- Vitalité : +4 maximum
- Sagesse : +4 maximum
- Compétences : +5 maximum
- Prix par unité : 1 200 kamas

#### Capes

**Bonus possibles** :
- Force : +4 maximum
- Intelligence : +4 maximum
- Chance : +4 maximum
- Agilité : +4 maximum
- Initiative : Illimité
- Points de vie max : Illimité (50 kamas par point)
- Compétences : +5 maximum
- Prix par unité : 1 000 kamas

#### Ceintures

**Bonus possibles** :
- Tacle : +10 maximum
- Fuite : +10 maximum
- Recharge de Wakfu : +3 maximum
- Prix par unité : 300 kamas (Tacle/Fuite) ou 1 500 kamas (Wakfu)

#### Bottes

**Bonus possibles** :
- PM : +3 maximum
- Esquive PM : +5 maximum
- Points de vie max : Illimité (50 kamas par point)
- Prix par unité : 1 000 kamas (PM) ou 300 kamas (Esquive)

**Note** : Les bottes sont très majoritairement vendues par paires.

#### Amulettes

**Bonus possibles** :
- PA : +6 maximum
- Esquive PA : +5 maximum
- Points de vie max : Illimité (50 kamas par point)
- Prix par unité : 1 300 kamas (PA) ou 300 kamas (Esquive)

#### Anneaux

**Bonus possibles** :
- Portée (PO) : +6 maximum
- Nombre d'invocation : +5 maximum
- Points de vie max : Illimité (50 kamas par point)
- Prix par unité : 800 kamas

**Note** : Vous pouvez porter **deux anneaux** (un à chaque main).

### Restrictions

- **Un seul objet de chaque type** à la fois (sauf anneaux : 2 maximum)
- Les bonus sont limités selon le type d'équipement
- Certaines caractéristiques peuvent se retrouver dans plusieurs types d'équipements

## 2.6.1.3. Défenses

### Boucliers

**Bonus possibles** :
- Classe d'armure : +5 maximum
- Résistance fixe (Neutre, Terre, Feu, Air, Eau) : +7 maximum
- Prix par unité : 1 100 kamas (CA) ou 700 kamas (Résistance)

**Utilisation** :
- Les boucliers augmentent votre Classe d'Armure (CA)
- Ils peuvent également offrir des résistances aux dégâts élémentaires
- Un personnage peut porter un seul bouclier

## 2.6.1.4. Outils et Faux

### Outils de métiers

Les outils de métiers sont utilisés pour pratiquer les différents métiers (Forge, Alchimie, etc.).

**Caractéristiques** :
- Nécessaires pour utiliser les compétences de métier
- Peuvent être achetés ou fabriqués
- Chaque métier a ses outils spécifiques

**Note** : Pour plus de détails, consultez la section **4.3. Métiers**.

### Faux

La faux est une arme particulière qui peut également servir d'outil.

**Caractéristiques** :
- Dégâts : 1d4 + niveau/4
- Coût : 5 PA
- Portée : 1-2
- Peut être utilisée comme outil agricole

## 2.6.1.5. Règles générales

### Rareté

Les équipements ont différents niveaux de rareté :
- **Très répandu** : Facile à trouver, prix modéré
- **Commun** : Disponible dans la plupart des villes
- **Rare** : Difficile à trouver, prix élevé
- **Très rare** : Exceptionnel, très cher
- **Légendaire** : Unique ou presque, extrêmement cher

### Niveaux requis

Certains équipements ont un **niveau minimum requis** :
- Niveau 1-3 : Équipements de débutant
- Niveau 4-7 : Équipements intermédiaires
- Niveau 8-11 : Équipements avancés
- Niveau 12-15 : Équipements experts
- Niveau 16-20 : Équipements maîtres

### Cumul des effets

- **Bonus du même type** : Se cumulent (ex: +2 Force cape + +2 Force autre cape = +4 Force total)
- **Limites maximales** : Respectez les limites par type d'équipement
- **Bonus différents** : Se cumulent également (ex: +2 Force cape + +1 Force chapeau = +3 Force total)

### Prix et économie

Les prix sont donnés en **kamas** (monnaie du Monde des Douze) :
- Prix de base : Selon le type d'équipement
- Prix par bonus : Multiplié par la valeur du bonus
- Prix personnalisé : Certains équipements ont des prix fixes selon leurs effets spéciaux

## 2.6.1.6. Recettes

Les équipements peuvent être **fabriqués** grâce aux métiers.

### Métiers de fabrication

- **Forge** : Armes, armures, boucliers
- **Couture** : Capes, chapeaux, ceintures
- **Bijouterie** : Amulettes, anneaux
- **Cordonnerie** : Bottes

### Ressources nécessaires

Chaque équipement nécessite :
- **Ressources de base** : Matériaux bruts (fer, cuir, tissu, etc.)
- **Ressources spéciales** : Matériaux rares ou spécifiques
- **Compétences** : Maîtrise du métier approprié

### Processus de fabrication

1. **Obtenir la recette** : Apprendre ou trouver la recette
2. **Rassembler les ressources** : Collecter les matériaux nécessaires
3. **Fabriquer** : Utiliser la compétence de métier appropriée
4. **Résultat** : Obtenir l'équipement (ou échouer et perdre les ressources)

**Note** : Pour plus de détails sur les métiers, les recettes et les ressources, consultez la section **4.3. Métiers**.

## Équipement de départ

Lorsqu'un joueur ou une joueuse rejoint une aventure, le ou la MJ veut l'équiper pour ne pas qu'iel soit trop en retard sur les autres joueurs/joueuses.

### Panoplies de départ

Des ensembles d'équipements (appelés **panoplies**) adaptés à différents niveaux sont en cours de développement. Ces panoplies offrent :
- Un équipement équilibré pour le niveau
- Des bonus cohérents avec la progression
- Une base solide pour commencer l'aventure

**Note** : Pour plus de détails sur les panoplies, consultez la section **2.6.2. Panoplies**.
