# 4.3.3. Métiers d'artisanat

**Description** : Fonctionnement des métiers de fabrication (tailleur, forgeron, bijoutier, cordonnier, bricoleur…) : recettes, réussite/échec, qualité.

> **Règles rapides - Métiers d'artisanat**
> - **Métiers** : Tailleur, Forgeron, Bijoutier, Cordonnier, Bricoleur, etc.
> - **Recettes** : Achat, apprentissage, découverte, création
> - **Tests** : Jets requis (Intelligence + compétence), seuils d'échec critique
> - **Qualité** : Standard, Rare, Épique (bonus additionnels selon la qualité)
> - **Temps** : Durée variable, assistance possible, ateliers (bonus)

## Contenu

- **4.3.3.1. Recettes** : Comment les obtenir, les apprendre, les créer.
- **4.3.3.2. Tests et réussite** : Jets requis, seuils d'échec critique.
- **4.3.3.3. Qualité** : Grades (standard, rare, épique), bonus additionnels.
- **4.3.3.4. Temps de fabrication** : Durée, assistance, ateliers.

## 4.3.3.1. Recettes

Les recettes sont nécessaires pour fabriquer des objets.

### Obtention des recettes

**Achat** :
- **Marchands** : Acheter des recettes auprès de marchands
- **Prix** : Variable selon la rareté de la recette
- **Disponibilité** : Certaines recettes sont rares

**Apprentissage** :
- **Maîtres** : Apprendre des recettes auprès de maîtres artisans
- **Quêtes** : Obtenir des recettes en accomplissant des quêtes
- **Découverte** : Découvrir des recettes dans des donjons ou des ruines

**Création** :
- **Expérience** : Créer de nouvelles recettes avec l'expérience
- **Expérimentation** : Expérimenter avec différents matériaux
- **Innovation** : Créer des recettes uniques

### Types de recettes

**Recettes de base** :
- **Niveau** : Niveau 1-10
- **Complexité** : Simple
- **Ressources** : Ressources communes

**Recettes avancées** :
- **Niveau** : Niveau 11-20
- **Complexité** : Modérée
- **Ressources** : Ressources rares

**Recettes épiques** :
- **Niveau** : Niveau 21-30
- **Complexité** : Complexe
- **Ressources** : Ressources épiques

**Recettes légendaires** :
- **Niveau** : Niveau 31+
- **Complexité** : Très complexe
- **Ressources** : Ressources légendaires

## 4.3.3.2. Tests et réussite

La fabrication nécessite des tests de compétence pour réussir.

### Niveau requis

**Règle** : Pour fabriquer un objet, il faut avoir au moins le **niveau de l'objet lui-même**.

**Exemples** :
- **Objet niveau 5** : Nécessite niveau 5 en métier
- **Objet niveau 15** : Nécessite niveau 15 en métier
- **Objet niveau 30** : Nécessite niveau 30 en métier

### Test de fabrication

**Formule** : 1d20 + Modificateur de caractéristique (selon métier) + Bonus de maîtrise + Bonus d'outils + Autres modificateurs

**DD** : Niveau de l'objet + 5

**Exemples** :
- **Objet niveau 10** : DD 15
- **Objet niveau 20** : DD 25
- **Objet niveau 30** : DD 35

### Caractéristiques par métier

**Bijoutier** : Agilité
**Cordonnier** : Intelligence
**Tailleur** : Agilité
**Sculpteur d'armes** : Chance
**Forgeur d'armes** : Force
**Forgeur de Boucliers** : Force
**Bricoleur** : Intelligence
**Boulanger** : Intelligence
**Boucher** : Force
**Poissonnier** : Chance

### Échec critique

**Seuil** : Échec critique sur un 1 naturel

**Conséquences** :
- **Ressources perdues** : Toutes les ressources sont perdues
- **Outils endommagés** : Les outils peuvent être endommagés
- **Blessure** : Possibilité de se blesser
- **Atelier endommagé** : L'atelier peut être endommagé

## 4.3.3.3. Qualité

Les objets fabriqués ont différents niveaux de qualité.

### Grades de qualité

**Standard** :
- **Réussite normale** : Objet de qualité standard
- **Caractéristiques** : Caractéristiques de base
- **Valeur** : Valeur standard

**Rare** :
- **Réussite élevée** : Objet de qualité rare
- **Caractéristiques** : Caractéristiques améliorées
- **Valeur** : Valeur augmentée

**Épique** :
- **Réussite très élevée** : Objet de qualité épique
- **Caractéristiques** : Caractéristiques significativement améliorées
- **Valeur** : Valeur très élevée

**Légendaire** :
- **Réussite critique** : Objet de qualité légendaire
- **Caractéristiques** : Caractéristiques exceptionnelles
- **Valeur** : Valeur inestimable

### Bonus additionnels

**Qualité supérieure** :
- **Bonus** : Bonus supplémentaires aux caractéristiques
- **Effets** : Effets spéciaux uniques
- **Durabilité** : Durabilité améliorée

**Exemples** :
- **Arme rare** : +1 dégâts supplémentaires
- **Armure épique** : +1 CA supplémentaire
- **Objet légendaire** : Effets spéciaux uniques

## 4.3.3.4. Temps de fabrication

La fabrication prend du temps selon la complexité de l'objet.

### Durée

**Règle** : Le temps de fabrication est égal à **2 fois le niveau de l'objet en heures**.

**Exemples** :
- **Objet niveau 5** : 10 heures
- **Objet niveau 10** : 20 heures
- **Objet niveau 20** : 40 heures

**Modificateurs** :
- **Niveau élevé** : Réduction du temps avec un niveau élevé
- **Outils** : Meilleurs outils = temps réduit
- **Expérience** : Plus d'expérience = temps réduit

### Assistance

**Aide d'alliés** :
- **Réduction** : Réduction du temps avec l'aide d'alliés
- **Limite** : Maximum 2-3 assistants
- **Compétence** : Les assistants doivent avoir la compétence

**Maîtres** :
- **Guidance** : Guidance d'un maître peut réduire le temps
- **Apprentissage** : Peut apprendre de nouvelles techniques
- **Coût** : Peut nécessiter un paiement

### Ateliers

**Ateliers de base** :
- **Accès** : Accès à un atelier équipé nécessaire
- **Bonus** : Bonus aux tests de fabrication
- **Coût** : Peut nécessiter un paiement ou une location

**Ateliers spécialisés** :
- **Bonus** : Bonus supplémentaires aux tests
- **Rareté** : Plus rares et chers
- **Accès** : Nécessitent souvent une réputation ou des relations

**Ateliers personnels** :
- **Construction** : Possibilité de construire son propre atelier
- **Coût** : Coût initial élevé
- **Avantages** : Accès permanent, pas de coûts récurrents
