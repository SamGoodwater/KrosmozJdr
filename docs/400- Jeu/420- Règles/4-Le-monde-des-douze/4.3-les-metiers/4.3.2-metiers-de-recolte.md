# 4.3.2. Métiers de récolte

**Description** : Règles pour herboriste, pêcheur, mineur, bûcheron, chasseur, etc. : zones de récolte, difficultés, risques.

> **Règles rapides - Métiers de récolte**
> - **Métiers** : Paysan, Herboriste, Pêcheur, Mineur, Bûcheron, Chasseur
> - **Zones** : Biomes spécifiques, ressources disponibles selon la zone
> - **Tests** : Compétences requises, difficultés variables, risques possibles
> - **Rendement** : Quantité obtenue, bonus d'équipe (travail en groupe)
> - **Incidents** : Rencontres aléatoires, pièges naturels

## 4.3.2.1. Zones de récolte

Chaque métier de récolte a ses zones spécifiques où les ressources peuvent être trouvées.

### Paysan

**Zones** :
- **Champs** : Céréales, légumes, fruits
- **Prairies** : Herbes, plantes médicinales
- **Fermes** : Zones cultivées

**Ressources** :
- **Céréales** : Blé, orge, avoine
- **Légumes** : Carottes, pommes de terre, choux
- **Fruits** : Pommes, poires, baies
- **Herbes** : Herbes culinaires et médicinales

### Bûcheron

**Zones** :
- **Forêts** : Zones boisées
- **Bois** : Zones avec des arbres
- **Plantations** : Zones d'arbres cultivés

**Ressources** :
- **Bois commun** : Bois de base
- **Bois rares** : Bois de qualité supérieure
- **Résine** : Résine d'arbres
- **Écorce** : Écorce d'arbres

### Mineur

**Zones** :
- **Mines** : Mines souterraines
- **Carrières** : Carrières à ciel ouvert
- **Montagnes** : Zones montagneuses

**Ressources** :
- **Minerais** : Fer, cuivre, argent, or
- **Pierres** : Pierres précieuses, gemmes
- **Cristaux** : Cristaux magiques
- **Métaux rares** : Métaux précieux

### Chasseur

**Zones** :
- **Forêts** : Zones sauvages
- **Plaines** : Zones ouvertes
- **Montagnes** : Zones montagneuses

**Ressources** :
- **Viande** : Viande de gibier
- **Peaux** : Peaux d'animaux
- **Cornes** : Cornes et bois
- **Griffes** : Griffes et dents

### Pêcheur

**Zones** :
- **Rivières** : Cours d'eau
- **Lacs** : Plans d'eau
- **Océans** : Zones côtières

**Ressources** :
- **Poissons** : Poissons comestibles
- **Coquillages** : Coquillages et crustacés
- **Algues** : Algues marines
- **Perles** : Perles rares

### Alchimiste (Herboriste)

**Zones** :
- **Forêts** : Zones boisées
- **Prairies** : Zones herbeuses
- **Montagnes** : Zones montagneuses
- **Zones magiques** : Zones avec concentration magique

**Ressources** :
- **Herbes** : Herbes médicinales
- **Plantes** : Plantes magiques
- **Champignons** : Champignons rares
- **Essences** : Essences végétales

## 4.3.2.2. Tests requis

La récolte nécessite des tests de compétence pour réussir.

### Niveau requis

**Règle** : Pour récolter une ressource, il faut avoir au moins le **niveau de la ressource elle-même**.

**Exemples** :
- **Ressource niveau 5** : Nécessite niveau 5 en métier
- **Ressource niveau 15** : Nécessite niveau 15 en métier
- **Ressource niveau 30** : Nécessite niveau 30 en métier

### Test de récolte

**Formule** : 1d20 + Modificateur de caractéristique (selon métier) + Bonus de maîtrise + Bonus d'outils + Autres modificateurs

**DD** : Niveau de la ressource + 3

**Exemples** :
- **Ressource niveau 10** : DD 13
- **Ressource niveau 20** : DD 23
- **Ressource niveau 30** : DD 33

### Caractéristiques par métier

**Paysan** : Intelligence
**Bûcheron** : Force
**Mineur** : Force
**Chasseur** : Agilité
**Pêcheur** : Agilité
**Alchimiste** : Chance

### Risques

**Échec** :
- **Ressource perdue** : La ressource n'est pas récoltée
- **Temps perdu** : Le temps écoulé est perdu
- **Parchemins** : Les parchemins de métier sont consommés

**Échec critique** :
- **Blessure** : Possibilité de se blesser
- **Outils endommagés** : Les outils peuvent être endommagés
- **Ressource détruite** : La ressource peut être rendue inutilisable

## 4.3.2.3. Rendement

Le rendement dépend de plusieurs facteurs.

### Quantité obtenue

**Base** :
- **Réussite normale** : 1 unité de ressource
- **Réussite élevée** : 1-2 unités de ressource
- **Réussite critique** : 2-3 unités de ressource

**Modificateurs** :
- **Niveau** : Plus le niveau est élevé, meilleur est le rendement
- **Outils** : Meilleurs outils = meilleur rendement
- **Zone** : Certaines zones offrent de meilleurs rendements

### Bonus d'équipe

**Coopération** :
- **Aide** : Les alliés peuvent aider à la récolte
- **Bonus** : Bonus aux tests de récolte
- **Rendement** : Augmentation du rendement total

**Spécialisation** :
- **Rôles** : Chaque personnage peut se spécialiser dans un métier
- **Efficacité** : Travail d'équipe plus efficace
- **Répartition** : Répartition des ressources récoltées

---

## 4.3.2.4. Incidents

La récolte peut être interrompue par divers incidents.

### Rencontres aléatoires

**Créatures** :
- **Animaux sauvages** : Rencontres avec des créatures
- **Créatures agressives** : Créatures hostiles
- **Boss** : Rencontres avec des créatures puissantes

**Gestion** :
- **Fuite** : Possibilité de fuir
- **Combat** : Nécessité de combattre
- **Négociation** : Possibilité de négocier

### Pièges naturels

**Types** :
- **Pièges de terrain** : Fissures, sables mouvants
- **Pièges de créatures** : Pièges posés par des créatures
- **Pièges magiques** : Pièges magiques dans les zones magiques

**Détection** :
- **Perception** : Tests de Perception pour détecter
- **Compétences** : Compétences spécifiques selon le type
- **Prévention** : Précautions pour éviter les pièges

### Conditions météorologiques

**Effets** :
- **Pluie** : Réduit l'efficacité de la récolte
- **Neige** : Rendre certaines zones inaccessibles
- **Tempêtes** : Peuvent interrompre la récolte

**Adaptation** :
- **Préparation** : Préparer l'équipement approprié
- **Timing** : Choisir le bon moment pour récolter
- **Risques** : Évaluer les risques avant de récolter
