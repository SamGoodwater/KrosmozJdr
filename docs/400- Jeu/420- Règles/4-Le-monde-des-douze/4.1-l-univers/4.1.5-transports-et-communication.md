# 4.1.5. Transports et communication

**Description** : Moyens de déplacement (terrestres, maritimes, portails, Zaaps, montures publiques) et méthodes de communication longue distance.

> **Règles rapides - Transports et communication**
> - **Zaaps** : Réseau de portails magiques, transport instantané (10-50 kamas), visite préalable requise
> - **Portails magiques** : Créés par mages/artefacts, temporaires ou permanents, coût variable
> - **Transports terrestres** : Montures publiques, caravanes, trotinnettes dragodindes
> - **Transports maritimes/aériens** : Navires, dirigeables, portails temporels
> - **Communication** : Messages magiques, familiers messagers, reliques

## Contenu

- **4.1.5.1. Zaaps et portails** : Fonctionnement, coût, restrictions.
- **4.1.5.2. Transports terrestres** : Montures publiques, caravanes, trotinnettes dragodindes.
- **4.1.5.3. Transports maritimes/aériens** : Navires, dirigeables, portails temporels.
- **4.1.5.4. Communication** : Messages magiques, familiers messagers, reliques.

## 4.1.5.1. Zaaps et portails

Les **Zaaps** et **portails** sont des moyens de transport magique permettant de se déplacer instantanément.

### Zaaps

**Fonctionnement** :
- **Réseau** : Réseau de portails magiques connectant les principales villes
- **Activation** : Nécessite d'avoir visité le Zaap de destination au moins une fois
- **Instantané** : Transport instantané entre deux Zaaps

**Coût** :
- **Standard** : Généralement 10-50 kamas selon la distance
- **Gratuit** : Certains Zaaps sont gratuits dans les grandes villes
- **Réductions** : Certaines guildes ou factions offrent des réductions

**Restrictions** :
- **Visite préalable** : Doit avoir visité le Zaap de destination
- **Disponibilité** : Certains Zaaps peuvent être temporairement indisponibles
- **Sécurité** : Certaines zones peuvent être bloquées en cas de danger

**Localisation** :
- **Grandes villes** : Bonta, Brâkmar, Astrub, Sufokia
- **Villes moyennes** : Nombreuses villes secondaires
- **Zones spéciales** : Certains donjons et zones mystiques

### Portails magiques

**Fonctionnement** :
- **Création** : Créés par des mages puissants ou des artefacts
- **Temporaires** : Certains portails sont temporaires
- **Permanents** : D'autres portails sont permanents

**Coût** :
- **Variable** : Dépend du créateur et de la destination
- **Gratuit** : Certains portails sont gratuits pour les alliés
- **Élevé** : Les portails vers des destinations lointaines coûtent cher

**Restrictions** :
- **Accès** : Certains portails nécessitent une autorisation
- **Stabilité** : Certains portails peuvent être instables
- **Danger** : Certains portails peuvent mener à des zones dangereuses

## 4.1.5.2. Transports terrestres

Les transports terrestres sont les moyens de déplacement les plus courants.

### Montures publiques

**Types** :
- **Dragodindes** : Montures rapides et robustes
- **Chevaux** : Montures classiques et fiables
- **Autres** : Diverses créatures utilisées comme montures

**Coût** :
- **Location** : Généralement 5-20 kamas par jour
- **Achat** : 500-2000 kamas selon la qualité
- **Entretien** : Nourriture et soins nécessaires

**Caractéristiques** :
- **Vitesse** : Variable selon le type de monture
- **Endurance** : Capacité de voyager sur de longues distances
- **Capacité** : Peut porter de l'équipement supplémentaire

### Caravanes

**Fonctionnement** :
- **Groupes** : Voyages en groupe pour la sécurité
- **Routes** : Suivent des routes commerciales établies
- **Protection** : Gardes pour protéger contre les brigands

**Coût** :
- **Passage** : Généralement 50-200 kamas selon la destination
- **Sécurité** : Coût supplémentaire pour la protection
- **Commodités** : Coût supplémentaire pour le confort

**Avantages** :
- **Sécurité** : Protection contre les dangers de la route
- **Confort** : Voyage plus confortable
- **Réseau** : Accès à un réseau de contacts

### Trotinnettes dragodindes

**Fonctionnement** :
- **Véhicules** : Petits véhicules tirés par des dragodindes
- **Rapidité** : Plus rapides que les montures classiques
- **Capacité** : Peuvent transporter plusieurs personnes

**Coût** :
- **Location** : Généralement 20-50 kamas par jour
- **Achat** : 1000-3000 kamas selon la qualité
- **Entretien** : Maintenance régulière nécessaire

**Caractéristiques** :
- **Vitesse** : Très rapides sur les routes
- **Maniabilité** : Bonne maniabilité en terrain varié
- **Capacité** : Peuvent transporter de l'équipement

### Véhicules à traction

**Types** :
- **Calèches** : 500 kamas, transport confortable
- **Carrioles** : 1k kamas, transport rapide
- **Chariots** : 300 kamas, transport de marchandises
- **Charrettes** : 200 kamas, transport basique
- **Traîneaux** : 400 kamas, transport sur neige

**Utilisation** :
- **Maîtrise** : Nécessite la maîtrise du véhicule pour un contrôle optimal
- **Tests** : Tests de compétence pour les situations délicates
- **Entretien** : Maintenance régulière nécessaire

## 4.1.5.3. Transports maritimes/aériens

Les transports maritimes et aériens permettent de voyager sur de longues distances.

### Transports maritimes

**Bateaux à rames** :
- **Barges** : 10k kamas, vitesse 1,4 km/h
- **Barques** : 200 kamas, vitesse 2,5 km/h
- **Utilisation** : Sur les lacs et rivières
- **Courant** : Vitesse augmentée dans le sens du courant

**Bateaux à voiles** :
- **Bateaux à voiles** : 20k kamas, vitesse 3 km/h
- **Drakkars** : 8k kamas, vitesse 5 km/h
- **Galères** : 70k kamas, vitesse 6,5 km/h
- **Navires de guerre** : 50k kamas, vitesse 4 km/h

**Caractéristiques** :
- **Vitesse** : Dépend du type de bateau et des conditions
- **Capacité** : Peuvent transporter de nombreuses personnes et marchandises
- **Sécurité** : Risques de tempêtes et de pirates

### Transports aériens

**Dirigeables** :
- **Fonctionnement** : Ballons remplis de gaz magique
- **Vitesse** : Rapide, peut traverser de grandes distances
- **Coût** : Très cher, réservé aux riches

**Portails temporels** :
- **Fonctionnement** : Portails créés par les Éliotropes
- **Vitesse** : Instantané
- **Coût** : Variable, nécessite un Éliotrope ou un artefact

**Créatures volantes** :
- **Types** : Griffons, dragons, créatures magiques
- **Vitesse** : Très rapide
- **Coût** : Très cher, nécessite une relation spéciale

## 4.1.5.4. Communication

La communication longue distance est essentielle pour rester en contact.

### Messages magiques

**Sorts de communication** :
- **Sends** : Sorts permettant d'envoyer des messages à distance
- **Portée** : Variable selon le niveau du sort
- **Coût** : Nécessite des ressources magiques

**Artefacts** :
- **Reliques** : Certaines reliques permettent la communication
- **Fonctionnement** : Communication instantanée entre deux reliques
- **Rareté** : Très rares et précieux

### Familiers messagers

**Fonctionnement** :
- **Familiers** : Créatures entraînées pour porter des messages
- **Fidélité** : Retournent à leur maître après avoir livré le message
- **Vitesse** : Rapide, peuvent traverser de grandes distances

**Types** :
- **Oiseaux** : Rapides et discrets
- **Créatures magiques** : Peuvent traverser des obstacles
- **Autres** : Diverses créatures selon les régions

**Coût** :
- **Entraînement** : Nécessite du temps et des ressources
- **Entretien** : Nourriture et soins nécessaires
- **Risques** : Peuvent être interceptés ou tués

### Reliques

**Fonctionnement** :
- **Communication** : Certaines reliques permettent la communication
- **Portée** : Variable selon la puissance de la relique
- **Utilisation** : Nécessite une activation magique

**Types** :
- **Reliques de communication** : Spécialement conçues pour la communication
- **Reliques polyvalentes** : Peuvent avoir d'autres fonctions
- **Reliques anciennes** : Artéfacts anciens avec des pouvoirs uniques

**Rareté** :
- **Très rares** : Extrêmement difficiles à obtenir
- **Prix** : Très élevés, souvent inestimables
- **Protection** : Généralement bien protégées

### Autres méthodes

**Courriers** :
- **Messagers** : Personnes payées pour porter des messages
- **Vitesse** : Dépend de la vitesse du messager
- **Coût** : Généralement 10-50 kamas selon la distance

**Pigeons voyageurs** :
- **Fonctionnement** : Oiseaux entraînés pour retourner à leur point d'origine
- **Vitesse** : Rapide mais limitée par la distance
- **Fiabilité** : Peuvent être interceptés ou perdus

**Réseaux** :
- **Guildes** : Réseaux de communication entre les membres
- **Factions** : Réseaux de communication entre les alliés
- **Commerce** : Réseaux de communication commerciaux

## Exemples pratiques

### Exemple 1 : Voyage par Zaap

**Scénario** : Un groupe veut voyager de Bonta à Astrub.

**Situation** : Les deux villes ont des Zaaps connectés.

**Résolution** :
- **Coût** : 20 kamas par personne
- **Temps** : Instantané
- **Restriction** : Doit avoir visité le Zaap d'Astrub au moins une fois

### Exemple 2 : Voyage par caravane

**Scénario** : Un groupe veut voyager de Bonta à Sufokia.

**Situation** : Longue distance, route dangereuse.

**Résolution** :
- **Coût** : 150 kamas par personne
- **Temps** : 1 semaine de voyage
- **Sécurité** : Protection contre les brigands incluse

### Exemple 3 : Communication magique

**Scénario** : Un personnage veut envoyer un message urgent à un allié lointain.

**Situation** : Distance trop grande pour un messager normal.

**Résolution** :
- **Méthode** : Sort de communication ou relique
- **Coût** : Ressources magiques ou utilisation de relique
- **Temps** : Instantané ou très rapide
