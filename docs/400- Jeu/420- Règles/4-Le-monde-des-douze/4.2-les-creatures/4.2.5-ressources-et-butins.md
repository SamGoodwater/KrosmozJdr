## 4.2.5.1. Tables de butin

Chaque famille de créatures laisse des ressources spécifiques lorsqu'elle est vaincue.

### Format générique

**Structure** :
- **Ressources communes** : Toujours présentes
- **Ressources rares** : Présentes avec une certaine probabilité
- **Ressources épiques** : Présentes rarement
- **Ressources légendaires** : Présentes très rarement

**Exemple de table** :

| 1d100 | Ressource | Quantité |
|-------|-----------|----------|
| 1-60 | Ressource commune | 1-3 |
| 61-85 | Ressource rare | 1-2 |
| 86-95 | Ressource épique | 1 |
| 96-100 | Ressource légendaire | 1 |

### Adaptation par famille

**Bouftous** :
- **Commun** : Laine (1-3), Viande (1-2)
- **Rare** : Laine de qualité (1), Corne (1)
- **Épique** : Laine royale (1)
- **Légendaire** : Corne dorée (1)

**Tofus** :
- **Commun** : Peau (1-2), Essence magique faible (1)
- **Rare** : Peau de qualité (1), Essence magique (1)
- **Épique** : Peau royale (1), Essence magique pure (1)
- **Légendaire** : Fragment de Wakfu (1)

**Chafers** :
- **Commun** : Carapace (1-2), Venin (1)
- **Rare** : Carapace renforcée (1), Venin concentré (1)
- **Épique** : Carapace de qualité (1), Venin pur (1)
- **Légendaire** : Carapace légendaire (1)

**Craqueleurs** :
- **Commun** : Écailles (1-3), Viande (2-4)
- **Rare** : Écailles renforcées (1-2), Griffes (1)
- **Épique** : Écailles de qualité (1), Griffes aiguisées (1)
- **Légendaire** : Écailles légendaires (1)

---

## 4.2.5.2. Rareté

Les ressources sont classées par rareté, déterminant leur valeur et leur disponibilité.

### Commun

**Caractéristiques** :
- **Probabilité** : 60-80% de chance d'obtenir
- **Quantité** : Généralement 1-3 unités
- **Valeur** : Faible à moyenne
- **Usage** : Utilisé dans de nombreuses recettes de base

**Exemples** :
- **Laine** : Utilisée en couture
- **Viande** : Utilisée en cuisine
- **Peau** : Utilisée en cordonnerie
- **Écailles** : Utilisées en forge

### Rare

**Caractéristiques** :
- **Probabilité** : 15-25% de chance d'obtenir
- **Quantité** : Généralement 1-2 unités
- **Valeur** : Moyenne à élevée
- **Usage** : Utilisé dans des recettes avancées

**Exemples** :
- **Laine de qualité** : Utilisée en couture avancée
- **Essence magique** : Utilisée en alchimie
- **Carapace renforcée** : Utilisée en forge avancée
- **Griffes** : Utilisées en bijouterie

### Épique

**Caractéristiques** :
- **Probabilité** : 5-10% de chance d'obtenir
- **Quantité** : Généralement 1 unité
- **Valeur** : Élevée à très élevée
- **Usage** : Utilisé dans des recettes épiques

**Exemples** :
- **Laine royale** : Utilisée en couture épique
- **Essence magique pure** : Utilisée en alchimie épique
- **Carapace de qualité** : Utilisée en forge épique
- **Griffes aiguisées** : Utilisées en bijouterie épique

### Légendaire

**Caractéristiques** :
- **Probabilité** : 1-5% de chance d'obtenir
- **Quantité** : Toujours 1 unité
- **Valeur** : Très élevée à inestimable
- **Usage** : Utilisé dans des recettes légendaires

**Exemples** :
- **Corne dorée** : Utilisée en forge légendaire
- **Fragment de Wakfu** : Utilisé en alchimie légendaire
- **Carapace légendaire** : Utilisée en forge légendaire
- **Écailles légendaires** : Utilisées en forge légendaire

## 4.2.5.3. Usage

Les ressources sont utilisées dans différents métiers pour créer des objets.

### Métiers concernés

**Couture** :
- **Ressources** : Laine, Peau, Tissu
- **Objets** : Vêtements, Capes, Chapeaux
- **Qualité** : Dépend de la qualité des ressources

**Forge** :
- **Ressources** : Minerais, Écailles, Carapaces
- **Objets** : Armes, Armures, Outils
- **Qualité** : Dépend de la qualité des ressources

**Bijouterie** :
- **Ressources** : Pierres, Griffes, Essences
- **Objets** : Amulettes, Anneaux, Bijoux
- **Qualité** : Dépend de la qualité des ressources

**Alchimie** :
- **Ressources** : Essences, Venins, Herbes
- **Objets** : Potions, Poisons, Consommables
- **Qualité** : Dépend de la qualité des ressources

**Cordonnerie** :
- **Ressources** : Peau, Cuir, Tissu
- **Objets** : Bottes, Chaussures
- **Qualité** : Dépend de la qualité des ressources

### Recettes clés

**Recettes de base** :
- **Utilisent** : Ressources communes
- **Résultat** : Objets de qualité standard
- **Exemples** : Vêtements basiques, Armes simples

**Recettes avancées** :
- **Utilisent** : Ressources rares
- **Résultat** : Objets de qualité améliorée
- **Exemples** : Vêtements de qualité, Armes améliorées

**Recettes épiques** :
- **Utilisent** : Ressources épiques
- **Résultat** : Objets de qualité épique
- **Exemples** : Vêtements épiques, Armes épiques

**Recettes légendaires** :
- **Utilisent** : Ressources légendaires
- **Résultat** : Objets de qualité légendaire
- **Exemples** : Vêtements légendaires, Armes légendaires

## 4.2.5.4. Valorisation

Les ressources ont une valeur qui varie selon leur rareté et leur qualité.

### Prix

**Ressources communes** :
- **Prix** : 10-50 kamas par unité
- **Marché** : Facilement disponibles
- **Variation** : Peu de variation selon les régions

**Ressources rares** :
- **Prix** : 50-200 kamas par unité
- **Marché** : Disponibles mais plus chères
- **Variation** : Variation selon les régions et la demande

**Ressources épiques** :
- **Prix** : 200-1000 kamas par unité
- **Marché** : Difficiles à trouver
- **Variation** : Grande variation selon les régions

**Ressources légendaires** :
- **Prix** : 1000+ kamas par unité, parfois inestimables
- **Marché** : Très rares, souvent troquées
- **Variation** : Variation extrême, parfois uniques

### Troc

**Utilisation** :
- **Ressources rares** : Peuvent être troquées contre d'autres ressources
- **Ressources épiques** : Souvent troquées plutôt que vendues
- **Ressources légendaires** : Généralement troquées, rarement vendues

**Avantages** :
- **Valeur** : Peuvent avoir plus de valeur en troc
- **Relations** : Créent des relations avec d'autres personnages
- **Accès** : Peuvent donner accès à des objets non disponibles autrement

### Demandes particulières

**Quêtes** :
- **Demandes** : Certaines quêtes demandent des ressources spécifiques
- **Valeur** : Valeur augmentée pour ces quêtes
- **Rareté** : Peuvent être difficiles à obtenir

**Commandes** :
- **Demandes** : Certains PNJ demandent des ressources spécifiques
- **Récompenses** : Récompenses spéciales pour ces ressources
- **Délais** : Peuvent avoir des délais

**Événements** :
- **Demandes** : Certains événements créent une demande temporaire
- **Valeur** : Valeur temporairement augmentée
- **Opportunités** : Opportunités de profit

## Exemples pratiques

### Exemple 1 : Butin d'un Bouftou

**Roll** : 1d100 = 45

**Résultat** :
- **Laine** : 2 unités (ressource commune)
- **Valeur** : 2 × 20 = 40 kamas

### Exemple 2 : Butin d'un Tofu royal

**Roll** : 1d100 = 92

**Résultat** :
- **Peau royale** : 1 unité (ressource épique)
- **Essence magique pure** : 1 unité (ressource épique)
- **Valeur** : 200 + 300 = 500 kamas

### Exemple 3 : Butin d'un Boss

**Roll** : 1d100 = 99

**Résultat** :
- **Ressources épiques** : 2-3 unités
- **Ressource légendaire** : 1 unité
- **Valeur** : 1000+ kamas
