## 4.2.3.1. Sorts naturels

Chaque famille de créatures possède des sorts naturels qui leur sont propres.

### Sorts de Bouftous

**Charge** :
- **Type** : Attaque physique
- **Effet** : Charge sur un ennemi, inflige des dégâts et peut pousser
- **Coût** : 3 PA
- **Portée** : Contact

**Cornes** :
- **Type** : Attaque physique
- **Effet** : Attaque avec les cornes, dégâts augmentés
- **Coût** : 4 PA
- **Portée** : Contact

### Sorts de Tofus

**Bond** :
- **Type** : Déplacement
- **Effet** : Bond sur une case, peut éviter des attaques
- **Coût** : 2 PM
- **Portée** : 3 cases

**Magie mineure** :
- **Type** : Attaque magique
- **Effet** : Projectile magique faible
- **Coût** : 3 PA
- **Portée** : 4 cases

### Sorts de Chafers

**Morsure** :
- **Type** : Attaque physique
- **Effet** : Morsure venimeuse, peut empoisonner
- **Coût** : 3 PA
- **Portée** : Contact

**Vol** :
- **Type** : Déplacement
- **Effet** : Vol sur plusieurs cases
- **Coût** : 4 PM
- **Portée** : 5 cases

### Sorts de Craqueleurs

**Coup de queue** :
- **Type** : Attaque physique
- **Effet** : Attaque en zone, peut toucher plusieurs ennemis
- **Coût** : 5 PA
- **Portée** : Zone de 2 cases

**Rugissement** :
- **Type** : Contrôle
- **Effet** : Intimide les ennemis, peut les faire fuir
- **Coût** : 4 PA
- **Portée** : Zone de 3 cases

## 4.2.3.2. Passifs

Les créatures possèdent des passifs qui leur confèrent des avantages permanents.

### Régénération

**Fonctionnement** :
- **Effet** : Régénère des PV chaque tour
- **Valeurs** : Généralement 5-20 PV par tour
- **Types** : Régénération normale, régénération magique

**Exemples** :
- **Créature régénérante** : 10 PV par tour
- **Boss régénérant** : 20-50 PV par tour
- **Régénération conditionnelle** : Se régénère seulement dans certaines conditions

### Immunité

**Fonctionnement** :
- **Effet** : Immunité à certains types de dégâts ou effets
- **Types** : Immunité au feu, à l'eau, aux états, etc.
- **Limites** : Généralement partielle, rarement totale

**Exemples** :
- **Créature de feu** : Immunité au feu
- **Créature de glace** : Immunité au froid
- **Créature magique** : Immunité partielle à la magie

### Enrage

**Fonctionnement** :
- **Déclencheur** : Se déclenche quand les PV tombent en dessous d'un seuil
- **Effet** : Bonus aux dégâts, parfois aux caractéristiques
- **Durée** : Généralement jusqu'à la fin du combat

**Exemples** :
- **Enrage à 50% PV** : +50% dégâts
- **Enrage à 25% PV** : +100% dégâts, +2 PA
- **Enrage extrême** : +200% dégâts, immunité à certains effets

## 4.2.3.3. Traits d'espèce

Chaque espèce possède des traits innés qui la caractérisent.

### Bonus innés

**Exemples** :
- **Bouftou** : Bonus à la Vitalité, résistance aux dégâts physiques
- **Tofu** : Bonus à l'Agilité, esquive améliorée
- **Chafers** : Bonus à la Force, attaques venimeuses
- **Craqueleurs** : Bonus à la Vitalité et à la Force, taille imposante

### Malus innés

**Exemples** :
- **Créature lourde** : Malus à l'Agilité, difficulté à se déplacer
- **Créature fragile** : Malus à la Vitalité, PV réduits
- **Créature lente** : Malus à l'initiative, PM réduits

### Traits spéciaux

**Exemples** :
- **Vision dans le noir** : Peut voir dans l'obscurité
- **Camouflage** : Bonus à la Discrétion dans certains environnements
- **Résistance naturelle** : Résistance à certains types de dégâts
- **Sens aiguisés** : Bonus à la Perception

## 4.2.3.4. Interactions états/traits

Les états et traits interagissent différemment selon les espèces.

### États positifs

**Bénédiction** :
- **Bouftous** : Bonus aux dégâts et à la défense
- **Tofus** : Bonus à l'Agilité et à l'esquive
- **Boss** : Effets réduits ou immunité

**Protection** :
- **Toutes les créatures** : Réduction des dégâts
- **Créatures magiques** : Effets renforcés
- **Boss** : Effets partiels

### États négatifs

**Ralentissement** :
- **Créatures rapides** : Effet plus prononcé
- **Créatures lentes** : Effet réduit ou nul
- **Boss** : Résistance accrue

**Empoisonnement** :
- **Créatures résistantes** : Effet réduit
- **Créatures vulnérables** : Effet renforcé
- **Créatures immunes** : Aucun effet

**Brûlure** :
- **Créatures de feu** : Aucun effet ou régénération
- **Créatures de glace** : Effet renforcé
- **Créatures normales** : Effet standard

### Traits permanents

**Lourd** :
- **Effet** : Ne peut pas être déplacé facilement
- **Interactions** : Résistant aux sorts de déplacement
- **Malus** : Difficulté sur les surfaces fragiles

**Petite taille** :
- **Effet** : Bonus à la Discrétion
- **Interactions** : Plus difficile à toucher
- **Malus** : Vitesse réduite

**Grande taille** :
- **Effet** : Bonus à l'Intimidation
- **Interactions** : Plus facile à toucher
- **Avantages** : Portée augmentée
