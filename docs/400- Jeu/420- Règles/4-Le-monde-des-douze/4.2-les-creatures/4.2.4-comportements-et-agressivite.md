## 4.2.4.1. Profils comportementaux

Les créatures ont différents niveaux d'agressivité qui déterminent leur réaction initiale.

### Amical

**Description** : La créature est amicale et cherche à aider ou à interagir positivement.

**Caractéristiques** :
- **Réaction** : Approche amicale, pas d'agression
- **Communication** : Peut communiquer ou aider
- **Combat** : N'attaque pas sauf provocation

**Exemples** :
- **Villageois** : Habitants des villages
- **Marchands** : Commerçants itinérants
- **Animaux domestiqués** : Créatures apprivoisées

### Curieux

**Description** : La créature est curieuse et peut s'approcher pour observer.

**Caractéristiques** :
- **Réaction** : Approche prudente, observation
- **Communication** : Peut tenter de communiquer
- **Combat** : Peut fuir si menacée, peut attaquer si provoquée

**Exemples** :
- **Animaux sauvages** : Créatures non agressives
- **Créatures magiques** : Entités curieuses
- **Explorateurs** : Autres aventuriers

### Neutre

**Description** : La créature est neutre et ne réagit que si provoquée.

**Caractéristiques** :
- **Réaction** : Ignore sauf provocation
- **Communication** : Peut communiquer si approchée
- **Combat** : Attaque seulement si menacée

**Exemples** :
- **Animaux sauvages** : Créatures qui évitent les conflits
- **Créatures territoriales** : Défendent leur territoire
- **Gardes** : Neutres sauf si menacés

### Peureux

**Description** : La créature a peur et fuit généralement.

**Caractéristiques** :
- **Réaction** : Fuite immédiate
- **Communication** : Difficile, la créature fuit
- **Combat** : Attaque seulement si acculée

**Exemples** :
- **Petites créatures** : Créatures faibles
- **Créatures chassées** : Proies habituelles
- **Créatures blessées** : Créatures affaiblies

### Agressif

**Description** : La créature est agressive et attaque si approchée.

**Caractéristiques** :
- **Réaction** : Attaque si approchée
- **Communication** : Difficile, la créature attaque
- **Combat** : Attaque activement

**Exemples** :
- **Créatures territoriales** : Défendent agressivement leur territoire
- **Créatures affamées** : Attaquent pour se nourrir
- **Créatures enragées** : Créatures en colère

### Hostile

**Description** : La créature est hostile et attaque immédiatement.

**Caractéristiques** :
- **Réaction** : Attaque immédiate
- **Communication** : Impossible, la créature attaque
- **Combat** : Attaque sans provocation

**Exemples** :
- **Créatures maléfiques** : Entités corrompues
- **Boss** : Gardiens de donjons
- **Créatures enragées** : Créatures complètement enragées

## 4.2.4.2. Réactions contextuelles

Les créatures réagissent différemment selon le contexte et les actions des personnages.

### Actions des personnages

**Approche agressive** :
- **Réaction** : La créature devient plus agressive
- **Neutre → Agressif** : Peut changer de comportement
- **Agressif → Hostile** : Peut devenir complètement hostile

**Approche pacifique** :
- **Réaction** : La créature peut devenir moins agressive
- **Agressif → Neutre** : Peut se calmer
- **Hostile → Agressif** : Peut devenir moins hostile

**Provocation** :
- **Réaction** : La créature devient immédiatement hostile
- **Tous les niveaux** : Peut devenir hostile
- **Conséquences** : Combat immédiat

### Environnement

**Territoire** :
- **Réaction** : Créatures plus agressives sur leur territoire
- **Défense** : Défendent activement leur territoire
- **Fuite** : Peuvent fuir si hors de leur territoire

**Nourriture** :
- **Réaction** : Créatures affamées plus agressives
- **Chasse** : Attaquent pour se nourrir
- **Satiété** : Moins agressives si rassasiées

**Saison** :
- **Réaction** : Comportement varie selon les saisons
- **Hiver** : Plus agressives en raison de la rareté de la nourriture
- **Printemps** : Moins agressives, période de reproduction

### Conditions spéciales

**Blessures** :
- **Réaction** : Créatures blessées peuvent fuir ou devenir plus agressives
- **Fuite** : Fuient si gravement blessées
- **Dernier recours** : Attaquent désespérément si acculées

**Protection** :
- **Réaction** : Créatures protégeant leurs petits sont très agressives
- **Défense** : Défendent activement leur progéniture
- **Risque** : Attaquent même des ennemis plus puissants

## 4.2.4.3. Scripts et tables aléatoires

Le MJ peut utiliser des scripts et des tables pour déterminer les réactions.

### Scripts comportementaux

**Script simple** :
1. **Détection** : La créature détecte les personnages
2. **Évaluation** : Évalue la menace
3. **Réaction** : Réagit selon son profil comportemental
4. **Adaptation** : Adapte sa réaction selon les actions des personnages

**Script complexe** :
1. **Observation** : Observe les personnages
2. **Communication** : Peut tenter de communiquer
3. **Négociation** : Peut négocier ou demander quelque chose
4. **Réaction** : Réagit selon le résultat

### Tables aléatoires

**Table de réaction** :

| 1d20 | Réaction |
|------|----------|
| 1-5 | Hostile |
| 6-10 | Agressif |
| 11-15 | Neutre |
| 16-18 | Curieux |
| 19-20 | Amical |

**Modificateurs** :
- **Approche pacifique** : +2 à +5
- **Approche agressive** : -2 à -5
- **Territoire** : -2
- **Nourriture abondante** : +2
- **Créature blessée** : -2

### Exemples d'utilisation

**Exemple 1 : Rencontre avec un Bouftou** :
- **Roll** : 1d20 = 12
- **Réaction de base** : Neutre
- **Modificateur** : Approche pacifique (+3)
- **Résultat** : Curieux (15)

**Exemple 2 : Rencontre avec un Chafers** :
- **Roll** : 1d20 = 8
- **Réaction de base** : Agressif
- **Modificateur** : Territoire (-2)
- **Résultat** : Agressif (6, reste agressif)

## 4.2.4.4. Domestication/relations

Certaines créatures peuvent être domestiquées ou développer des relations avec les personnages.

### Domestication

**Processus** :
- **Temps** : Nécessite du temps et de la patience
- **Tests** : Tests de Sagesse (Dressage) ou de Chance (Persuasion)
- **Conditions** : Créature doit être compatible

**Créatures domesticables** :
- **Animaux** : Bouftous, Dragodindes, Wabbits
- **Créatures magiques** : Certaines créatures magiques
- **Humanoïdes** : Généralement pas domesticables

**Avantages** :
- **Compagnon** : Créature fidèle et utile
- **Combat** : Peut aider au combat
- **Transport** : Peut servir de monture

### Relations

**Fidélité** :
- **Développement** : Se développe avec le temps et les soins
- **Niveaux** : Neutre, Amical, Fidèle, Dévoué
- **Avantages** : Créature plus obéissante et loyale

**Amitié** :
- **Développement** : Relation positive avec la créature
- **Avantages** : Créature peut aider volontairement
- **Limites** : Créature peut avoir ses propres besoins

**Haine** :
- **Développement** : Relation négative avec la créature
- **Conséquences** : Créature peut devenir hostile
- **Réconciliation** : Difficile mais possible

### Gestion

**Soins** :
- **Nourriture** : Nourrir régulièrement la créature
- **Soins** : Soigner les blessures
- **Attention** : Donner de l'attention et des soins

**Entraînement** :
- **Compétences** : Enseigner des compétences à la créature
- **Ordres** : Apprendre des ordres simples
- **Temps** : Nécessite du temps et de la patience

**Limites** :
- **Nature** : Certaines créatures ne peuvent pas être domestiquées
- **Temps** : Nécessite un investissement en temps
- **Ressources** : Nécessite des ressources pour les soins
