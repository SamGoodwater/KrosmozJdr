# 5.1.2. Création de PNJ et monstres

**Description** : Proposer des gabarits et checklists pour créer rapidement des PNJ, monstres ou boss adaptés au niveau des joueurs.

> **Règles rapides - Création de PNJ et monstres**
> - **Gabarits** : Structure standardisée (caractéristiques, PV, CA, actions)
> - **Balance** : Ajuster PV, dégâts, défenses selon le niveau du groupe
> - **Archétypes** : PNJ sociaux, marchands, ennemis, alliés (profils types)
> - **Boss** : Mécaniques spécifiques, phases, aides (combats épiques)

## Contenu

- **5.1.2.1. Gabarits** : Structure de fiche PNJ/monstre (caracs, actions).
- **5.1.2.2. Balance** : Ajuster les PV, dégâts, défenses selon le groupe.
- **5.1.2.3. Archétypes** : PNJ sociaux, marchands, ennemis, alliés.
- **5.1.2.4. Boss** : Mécaniques spécifiques, phases, aides.

## 5.1.2.1. Gabarits

Les gabarits fournissent une structure standardisée pour créer rapidement des PNJ et monstres.

### Structure de fiche PNJ

**Informations de base** :
- **Nom** : Nom du PNJ
- **Type** : Type de PNJ (marchand, garde, etc.)
- **Niveau** : Niveau approximatif
- **Alignement** : Alignement (si pertinent)

**Caractéristiques** :
- **Force, Intelligence, Sagesse, Agilité, Chance, Vitalité** : Modificateurs
- **PV** : Points de vie
- **CA** : Classe d'armure
- **Résistances** : Résistances aux dégâts

**Compétences** :
- **Compétences maîtrisées** : Liste des compétences
- **Bonus** : Bonus aux tests

**Actions** :
- **Actions de base** : Actions standard
- **Actions spéciales** : Capacités uniques
- **Équipement** : Équipement porté

### Structure de fiche Monstre

**Informations de base** :
- **Nom** : Nom du monstre
- **Type** : Type de créature
- **Taille** : Taille (TP, P, M, G, TG, Gig)
- **Niveau** : Niveau approximatif

**Caractéristiques** :
- **Force, Intelligence, Sagesse, Agilité, Chance, Vitalité** : Modificateurs
- **PV** : Points de vie
- **CA** : Classe d'armure
- **Résistances** : Résistances aux dégâts

**Ressources de combat** :
- **PA** : Points d'Action
- **PM** : Points de Mouvement
- **PO** : Points d'Opportunité
- **Initiative** : Modificateur d'initiative

**Actions** :
- **Attaques** : Attaques de base
- **Sorts** : Sorts naturels
- **Capacités** : Capacités spéciales
- **Passifs** : Passifs uniques

**Butin** :
- **Ressources** : Ressources laissées
- **Rareté** : Rareté des ressources

## 5.1.2.2. Balance

L'équilibrage permet d'adapter les créatures au niveau du groupe.

### Ajuster les PV

**Formule de base** : PV = Vitalité × 5-10 selon la robustesse

**Ajustements** :
- **Créature faible** : Vitalité × 5
- **Créature moyenne** : Vitalité × 7
- **Créature robuste** : Vitalité × 10
- **Boss** : Vitalité × 12-15

**Par niveau** :
- **Niveau 1-5** : 20-50 PV
- **Niveau 6-10** : 50-100 PV
- **Niveau 11-15** : 100-200 PV
- **Niveau 16-20** : 200-400 PV

### Ajuster les dégâts

**Formule de base** : Dégâts = Caractéristique + modificateurs

**Ajustements** :
- **Créature faible** : Dégâts réduits de 20-30%
- **Créature moyenne** : Dégâts standards
- **Créature puissante** : Dégâts augmentés de 20-30%
- **Boss** : Dégâts augmentés de 50-100%

**Par niveau** :
- **Niveau 1-5** : 1d6+mod à 2d6+mod
- **Niveau 6-10** : 2d6+mod à 3d6+mod
- **Niveau 11-15** : 3d6+mod à 4d6+mod
- **Niveau 16-20** : 4d6+mod à 5d6+mod

### Ajuster les défenses

**CA** :
- **Créature sans armure** : 10 + Vitalité
- **Créature avec armure naturelle** : 12-15 + Vitalité
- **Créature avec équipement** : Selon l'équipement + Vitalité

**Résistances** :
- **Créature élémentaire** : Résistance à son élément, vulnérabilité à l'élément opposé
- **Créature magique** : Résistance partielle à la magie
- **Boss** : Résistances multiples

## 5.1.2.3. Archétypes

Les archétypes fournissent des modèles pour créer rapidement des PNJ.

### PNJ sociaux

**Caractéristiques** :
- **Chance élevée** : Pour les tests sociaux
- **Compétences** : Persuasion, Perspicacité, Représentation
- **Comportement** : Amical, communicatif

**Exemples** :
- **Villageois** : Habitants des villages
- **Nobles** : Aristocrates et dirigeants
- **Artistes** : Musiciens, acteurs, conteurs

### Marchands

**Caractéristiques** :
- **Intelligence élevée** : Pour évaluer les objets
- **Compétences** : Persuasion, Perspicacité
- **Comportement** : Neutre, intéressé par le commerce

**Exemples** :
- **Marchands itinérants** : Commerçants voyageurs
- **Marchands spécialisés** : Commerçants d'objets rares
- **Marchands de guildes** : Commerçants de guildes

### Ennemis

**Caractéristiques** :
- **Force ou Intelligence élevée** : Selon le type
- **Compétences** : Combat, Intimidation
- **Comportement** : Agressif, hostile

**Exemples** :
- **Bandits** : Criminels et hors-la-loi
- **Cultistes** : Adeptes de cultes maléfiques
- **Soldats ennemis** : Forces militaires hostiles

### Alliés

**Caractéristiques** :
- **Caractéristiques équilibrées** : Selon le rôle
- **Compétences** : Variées selon le type
- **Comportement** : Amical, coopératif

**Exemples** :
- **Gardiens** : Forces de l'ordre amicales
- **Aventuriers** : Autres aventuriers
- **Mentors** : Guides et enseignants

## 5.1.2.4. Boss

Les boss sont des créatures puissantes avec des mécaniques spéciales.

### Mécaniques spécifiques

**Phases** :
- **Phase 1** : 100-50% PV, capacités de base
- **Phase 2** : 50-25% PV, capacités renforcées
- **Phase 3** : 25-0% PV, capacités désespérées

**Exemples** :
- **Boss niveau 10** : 3 phases avec capacités différentes
- **Boss niveau 20** : 4-5 phases avec mécaniques complexes

### Capacités spéciales

**Enrage** :
- **Déclencheur** : Se déclenche à un seuil de PV
- **Effet** : Bonus aux dégâts, parfois aux caractéristiques
- **Durée** : Jusqu'à la fin du combat

**Invocation** :
- **Déclencheur** : À certaines phases
- **Effet** : Invoque des créatures alliées
- **Gestion** : Nécessite une stratégie pour gérer les invocations

**Régénération** :
- **Déclencheur** : Actif en permanence ou à certaines phases
- **Effet** : Régénère des PV chaque tour
- **Contre** : Nécessite des dégâts élevés pour compenser

### Aides

**Minions** :
- **Fonction** : Créatures plus faibles qui aident le boss
- **Gestion** : Stratégie pour gérer les minions et le boss
- **Équilibrage** : Ajuster selon le nombre de joueurs

**Environnement** :
- **Fonction** : L'environnement peut aider ou gêner
- **Exemples** : Pièges, obstacles, zones dangereuses
- **Utilisation** : Le boss peut utiliser l'environnement

**Mécaniques uniques** :
- **Fonction** : Mécaniques spécifiques au boss
- **Exemples** : Faiblesses, vulnérabilités, mécaniques spéciales
- **Découverte** : Les joueurs doivent découvrir ces mécaniques
