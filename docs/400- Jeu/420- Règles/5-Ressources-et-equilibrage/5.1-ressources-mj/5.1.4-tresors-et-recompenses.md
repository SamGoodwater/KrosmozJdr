# 5.1.4. Trésors et récompenses

**Description** : Modèles de loot, équilibrage des récompenses (Kamas, ressources, équipements, Dofus fragments).

> **Règles rapides - Trésors et récompenses**
> - **Tables de loot** : Butins par type de rencontre (faible/moyen/puissant), par niveau
> - **Équipement** : Distribution selon rareté (Commun → Légendaire), panoplies partielles
> - **Ressources/Kamas** : Quantités recommandées (10-50 faibles, 50-200 moyens, 200-1000 puissants)
> - **Récompenses narratives** : Titres, faveur, alliances (récompenses non-matérielles)

## Contenu

- **5.1.4.1. Tables de loot** : Butins par type de rencontre, par niveau.

## 5.1.4.1. Tables de loot

Les tables de loot déterminent les récompenses selon le type de rencontre.

### Butins par type de rencontre

**Créatures faibles** :
- **Kamas** : 10-50 kamas
- **Ressources** : 1-2 ressources communes
- **Équipement** : Rare, équipement de base si présent

**Créatures moyennes** :
- **Kamas** : 50-200 kamas
- **Ressources** : 1-3 ressources (communes à rares)
- **Équipement** : Occasionnel, équipement de qualité si présent

**Créatures puissantes** :
- **Kamas** : 200-500 kamas
- **Ressources** : 2-4 ressources (rares à épiques)
- **Équipement** : Fréquent, équipement de qualité si présent

**Boss** :
- **Kamas** : 500-2000+ kamas
- **Ressources** : 3-5 ressources (épiques à légendaires)
- **Équipement** : Garanti, équipement épique ou légendaire

### Butins par niveau

**Niveau 1-5** :
- **Kamas** : 20-100 kamas par rencontre
- **Ressources** : 1-2 ressources communes
- **Équipement** : Équipement niveau 1-5

**Niveau 6-10** :
- **Kamas** : 100-300 kamas par rencontre
- **Ressources** : 1-3 ressources (communes à rares)
- **Équipement** : Équipement niveau 6-10

**Niveau 11-15** :
- **Kamas** : 300-800 kamas par rencontre
- **Ressources** : 2-4 ressources (rares à épiques)
- **Équipement** : Équipement niveau 11-15

**Niveau 16-20** :
- **Kamas** : 800-2000+ kamas par rencontre
- **Ressources** : 3-5 ressources (épiques à légendaires)
- **Équipement** : Équipement niveau 16-20

---

## 5.1.4.2. Équipement

La distribution d'équipement doit être équilibrée.

### Distribution

**Fréquence** :
- **Créatures faibles** : 5-10% de chance
- **Créatures moyennes** : 15-25% de chance
- **Créatures puissantes** : 30-50% de chance
- **Boss** : 80-100% de chance

**Qualité** :
- **Créatures faibles** : Équipement standard
- **Créatures moyennes** : Équipement standard à rare
- **Créatures puissantes** : Équipement rare à épique
- **Boss** : Équipement épique à légendaire

### Rareté

**Commun** :
- **Probabilité** : 60-70% des équipements trouvés
- **Niveau** : Niveau approprié au groupe
- **Valeur** : Valeur standard

**Rare** :
- **Probabilité** : 20-30% des équipements trouvés
- **Niveau** : Niveau approprié au groupe
- **Valeur** : Valeur augmentée

**Épique** :
- **Probabilité** : 5-10% des équipements trouvés
- **Niveau** : Niveau approprié au groupe
- **Valeur** : Valeur très élevée

**Légendaire** :
- **Probabilité** : 1-5% des équipements trouvés
- **Niveau** : Niveau élevé
- **Valeur** : Valeur inestimable

### Panoplies partielles

**Distribution** :
- **Pièces individuelles** : Pièces de panoplie trouvées séparément
- **Progression** : Permet aux joueurs de compléter progressivement
- **Récompenses** : Récompenses pour compléter une panoplie

**Gestion** :
- **Suivi** : Suivre les pièces trouvées par les joueurs
- **Opportunités** : Créer des opportunités pour trouver les pièces manquantes
- **Récompenses** : Récompenser la complétion d'une panoplie

## 5.1.4.3. Ressources/Kamas

L'équilibrage des ressources et Kamas est important pour l'économie.

### Quantités recommandées

**Par rencontre** :
- **Créatures faibles** : 20-50 kamas
- **Créatures moyennes** : 50-150 kamas
- **Créatures puissantes** : 150-400 kamas
- **Boss** : 400-1000+ kamas

**Par niveau** :
- **Niveau 1-5** : 30-100 kamas par rencontre
- **Niveau 6-10** : 100-250 kamas par rencontre
- **Niveau 11-15** : 250-600 kamas par rencontre
- **Niveau 16-20** : 600-1500+ kamas par rencontre

### Économie

**Équilibre** :
- **Revenus** : Revenus des rencontres et quêtes
- **Dépenses** : Dépenses pour équipement, consommables, services
- **Équilibre** : Maintenir un équilibre entre revenus et dépenses

**Inflation** :
- **Contrôle** : Contrôler l'inflation des prix
- **Ajustements** : Ajuster les prix selon l'économie
- **Rareté** : Maintenir la rareté des objets précieux

**Troc** :
- **Alternatives** : Encourager le troc pour les objets rares
- **Relations** : Le troc crée des relations
- **Valeur** : Le troc peut avoir plus de valeur que la vente

## 5.1.4.4. Récompenses narratives

Les récompenses narratives enrichissent l'expérience de jeu.

### Titres

**Types** :
- **Titres honorifiques** : Titres accordés par des autorités
- **Titres de réputation** : Titres basés sur la réputation
- **Titres de classe** : Titres liés à la classe
- **Titres personnalisés** : Titres uniques pour des accomplissements

**Effets** :
- **Réputation** : Amélioration de la réputation
- **Relations** : Amélioration des relations avec certains PNJ
- **Opportunités** : Nouvelles opportunités de quêtes

**Exemples** :
- **"Héros de Bonta"** : Titre honorifique
- **"Chasseur de Boss"** : Titre de réputation
- **"Maître Forgeur"** : Titre de classe

### Faveur

**Types** :
- **Faveur de PNJ** : Faveur de personnages importants
- **Faveur de factions** : Faveur de factions ou organisations
- **Faveur divine** : Faveur des dieux

**Effets** :
- **Accès** : Accès à des zones ou ressources
- **Réductions** : Réductions de prix
- **Assistance** : Assistance en cas de besoin

**Gestion** :
- **Progression** : La faveur progresse avec les actions
- **Perte** : La faveur peut être perdue
- **Utilisation** : La faveur peut être utilisée pour des avantages

### Alliances

**Types** :
- **Alliances politiques** : Alliances avec des factions
- **Alliances commerciales** : Alliances avec des guildes
- **Alliances personnelles** : Alliances avec des personnages

**Effets** :
- **Support** : Support dans les conflits
- **Ressources** : Accès à des ressources
- **Protection** : Protection contre les ennemis

**Gestion** :
- **Développement** : Les alliances se développent avec le temps
- **Engagements** : Les alliances créent des engagements
- **Conséquences** : Les alliances ont des conséquences
