# 5.1.3. Rencontres et scénarios

**Description** : Fournir des matrices pour générer des rencontres (combat, social, exploration), ainsi que des idées d'intrigues et d'obstacles.

> **Règles rapides - Rencontres et scénarios**
> - **Tables de rencontres** : Combat (1d20), social, défis environnementaux
> - **Générateur d'intrigues** : Accroches, twists, complications (structure narrative)
> - **Obstacles** : Épreuves physiques, puzzles, défis moraux
> - **Variantes** : Ajustements selon le niveau ou la région (adaptation contextuelle)

## Contenu

- **5.1.3.1. Tables de rencontres** : Combat, social, défis environnementaux.
- **5.1.3.2. Générateur d'intrigues** : Accroches, twists, complications.
- **5.1.3.3. Obstacles** : Épreuves physiques, puzzles, défis moraux.
- **5.1.3.4. Variantes** : Ajustements selon le niveau ou la région.

---

## 5.1.3.1. Tables de rencontres

Les tables de rencontres permettent de générer rapidement des rencontres variées.

### Rencontres de combat

**Table de base** (1d20) :

| 1d20 | Rencontre |
|------|-----------|
| 1-5 | Créatures sauvages (Bouftous, Wabbits, etc.) |
| 6-10 | Bandits ou hors-la-loi |
| 11-13 | Créatures magiques (Tofus, créatures élémentaires) |
| 14-16 | Gardes ou miliciens |
| 17-18 | Créatures puissantes (Craqueleurs, etc.) |
| 19 | Boss ou créature unique |
| 20 | Rencontre spéciale (événement unique) |

**Modificateurs** :
- **Région dangereuse** : +2 à +5
- **Région sûre** : -2 à -5
- **Niveau élevé** : +1 à +3
- **Niveau faible** : -1 à -3

### Rencontres sociales

**Table de base** (1d20) :

| 1d20 | Rencontre |
|------|-----------|
| 1-5 | Marchands itinérants |
| 6-8 | Villageois ou habitants |
| 9-11 | Aventuriers ou explorateurs |
| 12-13 | Nobles ou dirigeants |
| 14-15 | Artistes ou conteurs |
| 16-17 | Prêtres ou sages |
| 18 | Personnage mystérieux |
| 19-20 | Rencontre spéciale (quête, intrigue) |

**Modificateurs** :
- **Ville** : +3 à +5
- **Route** : -2 à -3
- **Temple** : +2 à +4
- **Donjon** : -5 (peu de rencontres sociales)

### Défis environnementaux

**Table de base** (1d20) :

| 1d20 | Défi |
|------|------|
| 1-4 | Obstacle physique (rivière, précipice, mur) |
| 5-8 | Piège mécanique |
| 9-11 | Piège magique |
| 12-14 | Conditions météorologiques (tempête, neige) |
| 15-16 | Phénomène magique (tempête de Wakfu) |
| 17-18 | Créature environnementale (plantes, éléments) |
| 19 | Défi complexe (combinaison) |
| 20 | Défi unique (événement spécial) |

**Modificateurs** :
- **Donjon** : +3 à +5
- **Nature** : +1 à +2
- **Ville** : -2 à -3

## 5.1.3.2. Générateur d'intrigues

Le générateur d'intrigues aide à créer des scénarios variés.

### Accroches

**Table d'accroches** (1d20) :

| 1d20 | Accroche |
|------|----------|
| 1-3 | Quête de récupération (objet volé, personne kidnappée) |
| 4-6 | Quête d'exploration (donjon, ruines, zone mystérieuse) |
| 7-9 | Quête de protection (escorte, défense) |
| 10-12 | Quête d'investigation (mystère, crime, disparition) |
| 13-15 | Quête de réputation (défi, tournoi, performance) |
| 16-17 | Quête de ressources (collecte, fabrication) |
| 18 | Quête épique (Dofus, artefact légendaire) |
| 19-20 | Quête personnelle (histoire du personnage) |

### Twists

**Table de twists** (1d20) :

| 1d20 | Twist |
|------|-------|
| 1-4 | Trahison (allié trahit, commanditaire maléfique) |
| 5-8 | Révélation (vérité cachée, identité secrète) |
| 9-12 | Complication (obstacle inattendu, nouvelle menace) |
| 13-15 | Opportunité (nouvelle piste, allié inattendu) |
| 16-17 | Conséquence (action passée a des conséquences) |
| 18 | Révélation majeure (changement de perspective) |
| 19-20 | Twist unique (événement spécial) |

### Complications

**Table de complications** (1d20) :

| 1d20 | Complication |
|------|--------------|
| 1-4 | Délai serré (contrainte de temps) |
| 5-8 | Conflit moral (choix difficile) |
| 9-12 | Opposition (ennemis puissants, obstacles) |
| 13-15 | Ressources limitées (manque de matériel, argent) |
| 16-17 | Information manquante (nécessite investigation) |
| 18 | Complication majeure (plusieurs problèmes) |
| 19-20 | Complication unique (événement spécial) |

## 5.1.3.3. Obstacles

Les obstacles créent des défis variés pour les personnages.

### Épreuves physiques

**Types** :
- **Escalade** : Parois, falaises, structures
- **Traversée** : Rivières, précipices, zones dangereuses
- **Force** : Portes verrouillées, obstacles lourds
- **Agilité** : Passages étroits, surfaces instables

**Tests** :
- **Escalade** : Athlétisme (Force)
- **Traversée** : Athlétisme (Force) ou Acrobaties (Agilité)
- **Force** : Athlétisme (Force)
- **Agilité** : Acrobaties (Agilité)

**DD** : Variable selon la difficulté (10-25)

### Puzzles

**Types** :
- **Mécaniques** : Mécanismes complexes, énigmes
- **Magiques** : Sorts, runes, portails
- **Logiques** : Énigmes, codes, séquences
- **Environnementaux** : Utilisation de l'environnement

**Résolution** :
- **Investigation** : Tests d'Investigation (Intelligence)
- **Arcanes** : Tests d'Arcanes (Intelligence)
- **Perception** : Tests de Perception (Sagesse)
- **Créativité** : Solutions créatives récompensées

### Défis moraux

**Types** :
- **Choix difficiles** : Dilemmes éthiques
- **Conséquences** : Actions avec conséquences
- **Relations** : Conflits entre personnages
- **Valeurs** : Conflits de valeurs

**Gestion** :
- **Discussion** : Encourager la discussion
- **Conséquences** : Conséquences réelles des choix
- **Roleplay** : Focus sur le roleplay
- **Flexibilité** : Pas de solution unique

## 5.1.3.4. Variantes

Les variantes permettent d'adapter les rencontres au niveau et à la région.

### Ajustements par niveau

**Niveau faible (1-5)** :
- **Créatures** : Créatures faibles à moyennes
- **DD** : DD réduits (10-15)
- **Complexité** : Rencontres simples
- **Récompenses** : Récompenses de base

**Niveau moyen (6-10)** :
- **Créatures** : Créatures moyennes à puissantes
- **DD** : DD standards (15-20)
- **Complexité** : Rencontres modérées
- **Récompenses** : Récompenses améliorées

**Niveau élevé (11-20)** :
- **Créatures** : Créatures puissantes à boss
- **DD** : DD élevés (20-30)
- **Complexité** : Rencontres complexes
- **Récompenses** : Récompenses épiques

### Ajustements par région

**Amakna** :
- **Créatures** : Créatures communes, humanoïdes
- **Rencontres** : Plus de rencontres sociales
- **Ambiance** : Paisible, dynamique

**Sufokia** :
- **Créatures** : Créatures du désert, nomades
- **Rencontres** : Plus de rencontres d'exploration
- **Ambiance** : Mystérieuse, dangereuse

**Frigost** :
- **Créatures** : Créatures de glace, créatures adaptées au froid
- **Rencontres** : Plus de défis environnementaux
- **Ambiance** : Hostile, isolée

**Saharach** :
- **Créatures** : Créatures de feu, créatures volcaniques
- **Rencontres** : Plus de combats
- **Ambiance** : Dangereuse, instable
