# 5.2.3. Sorts et aptitudes

**Description** : Cadre pour calibrer les sorts, aptitudes et capacités (coûts, dégâts, utilité, fréquence).

> **Règles rapides - Sorts et aptitudes**
> - **Coûts** : Actions simples (3-4 PA), fortes (5 PA), bonus (1-2 PA), Wakfu (1-3 points)
> - **Dégâts/effets** : Bornes par niveau (1d6+mod à 5d6+mod), durées maximales
> - **Fréquence** : Limiter les effets puissants (cooldown, Wakfu), équilibrage
> - **Contrôle** : Règles pour entraves, buffs, débuffs (durées et limitations)

## Contenu

- **5.2.3.1. Coûts** : PA/PM/Wakfu recommandés selon puissance.
- **5.2.3.2. Dégâts/effets** : Bornes par niveau, durées maximales.
- **5.2.3.3. Fréquence** : Limiter les effets puissants (cooldown, Wakfu).
- **5.2.3.4. Contrôle** : Règles pour entraves, buffs, débuffs.

---

## 5.2.3.1. Coûts

Les coûts en PA/PM/Wakfu doivent être équilibrés selon la puissance.

### Coûts en PA

**Actions simples** :
- **Coût** : 3-4 PA
- **Exemples** : Attaque de base, sort simple
- **Équilibrage** : Permet plusieurs actions par tour

**Actions fortes** :
- **Coût** : 5 PA
- **Exemples** : Sort en zone, effet important
- **Équilibrage** : Limite à une action forte par tour

**Actions bonus simples** :
- **Coût** : 2 PA
- **Exemples** : Action bonus de base
- **Équilibrage** : Permet des actions bonus supplémentaires

**Actions bonus fortes** :
- **Coût** : 3 PA
- **Exemples** : Action bonus avec effet important
- **Équilibrage** : Limite les actions bonus puissantes

### Coûts en PM

**Déplacement** :
- **Coût** : 1 PM par case
- **Équilibrage** : Permet le positionnement tactique

**Déplacement spécial** :
- **Coût** : Variable selon l'effet
- **Exemples** : Charge, saut, téléportation
- **Équilibrage** : Coût approprié selon la puissance

### Coûts en Wakfu

**Sorts puissants** :
- **Coût** : Variable selon le niveau du sort
- **Équilibrage** : Limite l'utilisation des sorts puissants

**Aptitudes** :
- **Coût** : Variable selon la puissance
- **Équilibrage** : Coût approprié selon l'effet

**Réserve de Wakfu** :
- **Limite** : Quantité limitée de PA de la réserve
- **Équilibrage** : Limite l'utilisation hors combat

---

## 5.2.3.2. Dégâts/effets

Les dégâts et effets doivent être équilibrés selon le niveau.

### Bornes par niveau

**Niveau 1-5** :
- **Dégâts** : 1d6+mod à 2d6+mod
- **Soins** : 1d4+mod à 2d4+mod
- **Effets** : Effets simples, durées courtes

**Niveau 6-10** :
- **Dégâts** : 2d6+mod à 3d6+mod
- **Soins** : 2d4+mod à 3d4+mod
- **Effets** : Effets modérés, durées modérées

**Niveau 11-15** :
- **Dégâts** : 3d6+mod à 4d6+mod
- **Soins** : 3d4+mod à 4d4+mod
- **Effets** : Effets puissants, durées longues

**Niveau 16-20** :
- **Dégâts** : 4d6+mod à 5d6+mod
- **Soins** : 4d4+mod à 5d4+mod
- **Effets** : Effets exceptionnels, durées très longues

### Durées maximales

**Effets temporaires** :
- **Court terme** : 1 tour à 1 minute
- **Moyen terme** : 1 minute à 10 minutes
- **Long terme** : 10 minutes à 1 heure

**Équilibrage** :
- **Puissance** : Durée inversement proportionnelle à la puissance
- **Coût** : Durée proportionnelle au coût
- **Niveau** : Durée proportionnelle au niveau

---

## 5.2.3.3. Fréquence

La fréquence d'utilisation doit être limitée pour les effets puissants.

### Cooldown

**Définition** : Temps de recharge avant réutilisation.

**Types** :
- **Court** : 1-3 tours
- **Moyen** : 4-6 tours
- **Long** : 7-10 tours
- **Très long** : 10+ tours ou repos

**Équilibrage** :
- **Puissance** : Cooldown inversement proportionnel à la puissance
- **Utilité** : Cooldown inversement proportionnel à l'utilité
- **Niveau** : Cooldown inversement proportionnel au niveau

### Réserve de Wakfu

**Limite** :
- **Quantité** : Quantité limitée de PA
- **Régénération** : Régénération limitée
- **Utilisation** : Utilisation limitée hors combat

**Équilibrage** :
- **Puissance** : Coût en Wakfu inversement proportionnel à la puissance
- **Fréquence** : Limite la fréquence d'utilisation
- **Niveau** : Coût proportionnel au niveau

### Limitations supplémentaires

**Par combat** :
- **Limite** : Utilisation limitée par combat
- **Exemples** : Une fois par combat, deux fois par combat
- **Équilibrage** : Limite les effets puissants

**Par jour** :
- **Limite** : Utilisation limitée par jour
- **Exemples** : Une fois par jour, deux fois par jour
- **Équilibrage** : Limite les effets très puissants

---

## 5.2.3.4. Contrôle

Les règles de contrôle doivent être équilibrées pour éviter les abus.

### Entraves

**Types** :
- **Immobilisation** : Vitesse réduite à 0
- **Ralentissement** : Vitesse réduite de moitié
- **Empêchement** : Empêchement de certaines actions

**Équilibrage** :
- **Durée** : Durée limitée
- **Sauvegarde** : Possibilité de sauvegarde
- **Résistance** : Résistance possible

### Buffs

**Types** :
- **Caractéristiques** : Bonus aux caractéristiques
- **Compétences** : Bonus aux compétences
- **Combat** : Bonus aux attaques, CA, etc.

**Équilibrage** :
- **Durée** : Durée limitée
- **Stack** : Limite de stack
- **Coût** : Coût approprié

### Debuffs

**Types** :
- **Caractéristiques** : Malus aux caractéristiques
- **Compétences** : Malus aux compétences
- **Combat** : Malus aux attaques, CA, etc.

**Équilibrage** :
- **Durée** : Durée limitée
- **Sauvegarde** : Possibilité de sauvegarde
- **Résistance** : Résistance possible

### États

**Types** :
- **Stun** : Ne peut pas agir
- **Gel** : Vitesse réduite, dégâts de froid
- **Brûlure** : Dégâts de feu
- **Empoisonnement** : Dégâts de poison

**Équilibrage** :
- **Durée** : Durée limitée
- **Sauvegarde** : Possibilité de sauvegarde
- **Résistance** : Résistance possible
- **Immunité** : Immunité possible après application
