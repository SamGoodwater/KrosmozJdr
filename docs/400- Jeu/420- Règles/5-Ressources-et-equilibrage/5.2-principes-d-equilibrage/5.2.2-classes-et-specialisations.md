# 5.2.2. Classes et spécialisations

**Description** : Règles d'équilibrage des classes (voies, rôles) et spécialisations (aptitudes, capacités), critères pour juger leur puissance.

> **Règles rapides - Classes et spécialisations**
> - **Rôles** : Vérifier que chaque rôle est couvert (Dégâts, Protection, Soin, Amélioration, Entrave, Placement)
> - **Voies élémentaires** : Répartition des sorts (Feu, Eau, Terre, Air), cohérence thématique
> - **Spécialisations** : Puissance des aptitudes/capacités par niveau (progression équilibrée)
> - **Comparaison** : Méthode pour comparer deux classes/spécialisations (critères objectifs)

## Contenu

- **5.2.2.1. Rôles** : Vérifier que chaque rôle (dégâts, protection, soin, etc.) est couvert.
- **5.2.2.2. Voies élémentaires** : Répartition des sorts, cohérence thématique.
- **5.2.2.3. Spécialisations** : Puissance des aptitudes/capacités par niveau.
- **5.2.2.4. Comparaison** : Méthode pour comparer deux classes/spécialisations.

---

## 5.2.2.1. Rôles

Chaque classe doit pouvoir remplir au moins un rôle dans le groupe.

### Rôles principaux

**Dégâts** :
- **Fonction** : Infliger des dégâts aux ennemis
- **Classes** : Classes orientées dégâts (Iop, Cra, etc.)
- **Équilibrage** : Dégâts moyens par niveau appropriés

**Protection** :
- **Fonction** : Protéger les alliés
- **Classes** : Classes orientées protection (Pandawa, Sacrieur, etc.)
- **Équilibrage** : CA et PV appropriés, capacités de protection

**Soin** :
- **Fonction** : Soigner les alliés
- **Classes** : Classes orientées soin (Eniripsa, etc.)
- **Équilibrage** : Capacités de soin appropriées

**Support** :
- **Fonction** : Soutenir les alliés (buffs, débuffs)
- **Classes** : Classes orientées support (Enutrof, etc.)
- **Équilibrage** : Effets de support appropriés

**Contrôle** :
- **Fonction** : Contrôler le champ de bataille
- **Classes** : Classes orientées contrôle (Sram, etc.)
- **Équilibrage** : Capacités de contrôle appropriées

### Vérification de couverture

**Méthode** :
- **Liste** : Lister tous les rôles
- **Vérification** : Vérifier que chaque rôle est couvert par au moins une classe
- **Équilibrage** : S'assurer que les rôles sont équilibrés

**Exemples** :
- **Dégâts** : Iop, Cra, Xélor
- **Protection** : Pandawa, Sacrieur
- **Soin** : Eniripsa
- **Support** : Enutrof, Féca
- **Contrôle** : Sram, Ecaflip

## 5.2.2.2. Voies élémentaires

Les voies élémentaires déterminent la répartition des sorts et la cohérence thématique.

### Répartition des sorts

**Voies élémentaires** :
- **Feu** : Sorts de feu, dégâts, brûlure
- **Eau** : Sorts d'eau, soins, gel
- **Terre** : Sorts de terre, protection, contrôle
- **Air** : Sorts d'air, mobilité, foudre
- **Neutre** : Sorts neutres, polyvalence

**Répartition** :
- **Équilibre** : Chaque voie doit avoir des sorts équilibrés
- **Niveaux** : Sorts de tous les niveaux pour chaque voie
- **Variété** : Variété de sorts pour chaque voie

### Cohérence thématique

**Thèmes** :
- **Feu** : Agressif, destructeur, passionné
- **Eau** : Soignant, fluide, adaptatif
- **Terre** : Protecteur, stable, résistant
- **Air** : Mobile, rapide, libre
- **Neutre** : Équilibré, polyvalent, universel

**Application** :
- **Sorts** : Les sorts doivent correspondre au thème
- **Capacités** : Les capacités doivent correspondre au thème
- **Esthétique** : L'esthétique doit correspondre au thème

---

## 5.2.2.3. Spécialisations

Les spécialisations ajoutent de la variété et de la personnalisation.

### Puissance des aptitudes

**Par niveau** :
- **Niveau 1-5** : Aptitudes de base, effets simples
- **Niveau 6-10** : Aptitudes améliorées, effets modérés
- **Niveau 11-15** : Aptitudes avancées, effets puissants
- **Niveau 16-20** : Aptitudes légendaires, effets exceptionnels

**Équilibrage** :
- **Coût** : Coût approprié selon la puissance
- **Fréquence** : Fréquence appropriée selon la puissance
- **Effets** : Effets appropriés selon le niveau

### Puissance des capacités

**Par niveau** :
- **Niveau 1-5** : Capacités de base, effets simples
- **Niveau 6-10** : Capacités améliorées, effets modérés
- **Niveau 11-15** : Capacités avancées, effets puissants
- **Niveau 16-20** : Capacités légendaires, effets exceptionnels

**Équilibrage** :
- **Passifs** : Effets passifs appropriés
- **Actifs** : Effets actifs avec coûts appropriés
- **Contextuels** : Effets contextuels avec conditions appropriées

### Rôles dans les spécialisations

**1er Rôle** :
- **Définition** : Personne jouant la classe dans cette orientation
- **Sorts** : Sorts en accord avec le rôle
- **Caractéristiques** : Caractéristiques en accord avec le rôle
- **Puissance** : Puissance maximale pour ce rôle

**2nd Rôle** :
- **Définition** : Personne jouant la classe d'une autre façon
- **Sorts** : Sorts partiellement en accord avec le rôle
- **Caractéristiques** : Caractéristiques partiellement en accord
- **Puissance** : Puissance modérée pour ce rôle

**Pas de Rôle** :
- **Définition** : Personne ayant choisi d'ignorer ce gameplay
- **Sorts** : Sorts non orientés vers ce rôle
- **Caractéristiques** : Caractéristiques non orientées vers ce rôle
- **Puissance** : Puissance minimale pour ce rôle

## 5.2.2.4. Comparaison

La comparaison permet d'évaluer l'équilibre entre classes et spécialisations.

### Méthode de comparaison

**Critères** :
- **Dégâts** : Dégâts moyens par tour
- **Défense** : CA, PV, résistances
- **Utilité** : Capacités utilitaires
- **Ressources** : PA, PM, Wakfu

**Valeurs** :
- **Maximum** : Valeur maximum possible
- **Minimum** : Valeur minimum possible
- **Moyenne** : Valeur moyenne

**Évaluation** :
- **Comparaison** : Comparer les valeurs entre classes
- **Équilibre** : Vérifier l'équilibre
- **Ajustements** : Ajuster si nécessaire

### Exemple de comparaison

**Iop (Dégâts)** :
- **Dégâts** : Maximum élevé, minimum modéré, moyenne élevée
- **Défense** : Maximum modéré, minimum faible, moyenne modérée
- **Utilité** : Maximum faible, minimum faible, moyenne faible
- **Ressources** : PA élevés, PM modérés, Wakfu modéré

**Eniripsa (Soin)** :
- **Dégâts** : Maximum modéré, minimum faible, moyenne modérée
- **Défense** : Maximum modéré, minimum modéré, moyenne modérée
- **Utilité** : Maximum élevé, minimum modéré, moyenne élevée
- **Ressources** : PA modérés, PM modérés, Wakfu élevé

**Équilibre** :
- **Iop** : Fort en dégâts, faible en utilité
- **Eniripsa** : Fort en utilité, modéré en dégâts
- **Conclusion** : Équilibre approprié, rôles complémentaires
