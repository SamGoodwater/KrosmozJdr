## 5.2.5.1. XP et niveaux

La progression doit être équilibrée pour maintenir l'engagement.

### Tableau de progression

**Niveau 1-5** :
- **XP requis** : 0-1000 XP
- **Progression** : Rapide
- **Objectif** : Apprentissage des bases

**Niveau 6-10** :
- **XP requis** : 1000-5000 XP
- **Progression** : Modérée
- **Objectif** : Développement des capacités

**Niveau 11-15** :
- **XP requis** : 5000-15000 XP
- **Progression** : Lente
- **Objectif** : Maîtrise avancée

**Niveau 16-20** :
- **XP requis** : 15000-50000+ XP
- **Progression** : Très lente
- **Objectif** : Maîtrise légendaire

### Seuils d'avancement

**Rencontres** :
- **Faible** : 50-100 XP
- **Moyenne** : 100-300 XP
- **Puissante** : 300-800 XP
- **Boss** : 800-2000+ XP

**Quêtes** :
- **Simple** : 100-200 XP
- **Modérée** : 200-500 XP
- **Complexe** : 500-1000 XP
- **Épique** : 1000-3000+ XP

**Objectifs** :
- **Court terme** : 1-2 sessions
- **Moyen terme** : 3-5 sessions
- **Long terme** : 5-10+ sessions

---

## 5.2.5.2. Récompenses

Les récompenses doivent être équilibrées selon le type de rencontre.

### Ratio XP/Kamas/ressources

**Rencontres faibles** :
- **XP** : 50-100
- **Kamas** : 20-50
- **Ressources** : 1-2 communes

**Rencontres moyennes** :
- **XP** : 100-300
- **Kamas** : 50-150
- **Ressources** : 1-3 (communes à rares)

**Rencontres puissantes** :
- **XP** : 300-800
- **Kamas** : 150-400
- **Ressources** : 2-4 (rares à épiques)

**Boss** :
- **XP** : 800-2000+
- **Kamas** : 400-1000+
- **Ressources** : 3-5 (épiques à légendaires)

### Par niveau

**Niveau 1-5** :
- **XP** : 50-200 par rencontre
- **Kamas** : 30-100 par rencontre
- **Ressources** : 1-2 par rencontre

**Niveau 6-10** :
- **XP** : 200-500 par rencontre
- **Kamas** : 100-250 par rencontre
- **Ressources** : 1-3 par rencontre

**Niveau 11-15** :
- **XP** : 500-1200 par rencontre
- **Kamas** : 250-600 par rencontre
- **Ressources** : 2-4 par rencontre

**Niveau 16-20** :
- **XP** : 1200-3000+ par rencontre
- **Kamas** : 600-1500+ par rencontre
- **Ressources** : 3-5 par rencontre
