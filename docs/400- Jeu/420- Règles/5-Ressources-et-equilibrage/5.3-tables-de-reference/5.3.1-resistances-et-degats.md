# 5.3.1. Résistances et dégâts

**Description** : Tables indiquant les résistances types (Feu, Eau, etc.) et les barèmes de dégâts/soins attendus par niveau.

> **Règles rapides - Résistances et dégâts**
> - **Barèmes dégâts** : Niveau 1-3 (5-10/tour, 1d6+mod), 4-7 (10-20/tour, 2d6+mod), 8-11 (20-30/tour, 3d6+mod), 12-15 (30-40/tour, 4d6+mod), 16-20 (40-50+/tour, 5d6+mod)
> - **Barèmes soins** : Niveau 1-3 (1d4+mod, 5-10/tour), 4-7 (2d4+mod, 10-20/tour), 8-11 (3d4+mod, 20-30/tour), 12-15 (4d4+mod, 30-40/tour), 16-20 (5d4+mod, 40-50+/tour)
> - **Résistances** : Pourcentages recommandés par type (Feu, Eau, Terre, Air, Neutre)
> - **Coups critiques** : Multiplicateurs (x2), bonus additionnels

## Contenu

- **5.3.1.1. Barèmes dégâts/soins** : Moyennes par niveau.
- **5.3.1.2. Résistances** : Pourcentages recommandés par type.
- **5.3.1.3. Coups critiques** : Multiplicateurs, bonus additionnels.
- **5.3.1.4. Répartition** : Suggestions pour MJ selon la difficulté.

## 5.3.1.2. Résistances

### Pourcentages recommandés par type

**Résistances élémentaires** :

| Élément | Résistance faible | Résistance moyenne | Résistance forte |
|---------|-------------------|---------------------|------------------|
| **Feu** | 10-20% | 30-50% | 60-80% |
| **Eau** | 10-20% | 30-50% | 60-80% |
| **Terre** | 10-20% | 30-50% | 60-80% |
| **Air** | 10-20% | 30-50% | 60-80% |
| **Neutre** | 10-20% | 30-50% | 60-80% |

**Résistances physiques** :

| Type | Résistance faible | Résistance moyenne | Résistance forte |
|------|-------------------|---------------------|------------------|
| **Physique** | 5-10% | 15-25% | 30-50% |
| **Magique** | 10-20% | 30-50% | 60-80% |

### Vulnérabilités

**Vulnérabilités élémentaires** :
- **Feu** : Vulnérable à l'Eau (-50% résistance)
- **Eau** : Vulnérable à la Terre (-50% résistance)
- **Terre** : Vulnérable à l'Air (-50% résistance)
- **Air** : Vulnérable au Feu (-50% résistance)
- **Neutre** : Pas de vulnérabilité spécifique

## 5.3.1.3. Coups critiques

### Multiplicateurs

**Multiplicateurs de base** :
- **Arme normale** : ×2 dégâts
- **Arme critique** : ×3 dégâts
- **Sort critique** : ×1.5 dégâts

### Bonus additionnels

**Effets critiques** :
- **Dégâts supplémentaires** : +1d6 dégâts
- **États** : Application d'états (brûlure, gel, etc.)
- **Effets spéciaux** : Effets uniques selon l'arme/sort

### Probabilités

**Chances de critique** :
- **Base** : 5% (20 naturel)
- **Avec bonus** : 10-15% (selon équipement)
- **Avec capacités** : 15-20% (selon capacités)

## 5.3.1.4. Répartition

### Suggestions pour MJ selon la difficulté

**Rencontre facile** :
- **Dégâts** : -20% par rapport aux moyennes
- **PV** : -20% par rapport aux moyennes
- **Résistances** : Résistances faibles
- **Objectif** : Permettre aux joueurs de gagner facilement

**Rencontre normale** :
- **Dégâts** : Valeurs moyennes
- **PV** : Valeurs moyennes
- **Résistances** : Résistances moyennes
- **Objectif** : Défi équilibré

**Rencontre difficile** :
- **Dégâts** : +20% par rapport aux moyennes
- **PV** : +20% par rapport aux moyennes
- **Résistances** : Résistances fortes
- **Objectif** : Défi significatif

**Rencontre très difficile** :
- **Dégâts** : +50% par rapport aux moyennes
- **PV** : +50% par rapport aux moyennes
- **Résistances** : Résistances très fortes
- **Objectif** : Défi extrême

**Boss** :
- **Dégâts** : +100% par rapport aux moyennes
- **PV** : +200-300% par rapport aux moyennes
- **Résistances** : Résistances multiples et fortes
- **Objectif** : Combat épique et mémorable
