# 5.3.3. Météo et événements

**Description** : Tables pour déterminer rapidement la météo, les phénomènes magiques, les complications environnementales.

> **Règles rapides - Météo et événements**
> - **Météo standard** : Soleil (1-8), Nuages (9-12), Pluie légère (13-15, -1 Perception), Pluie forte (16-17, -2 Perception), Neige (18, -2 Perception), Tempête (19, -3 Perception), Anomalie (20)
> - **Phénomènes wakfu** : Tempêtes de Wakfu, anomalies temporelles (effets magiques)
> - **Complications** : Glissades, éboulements, pannes de Zaap (événements environnementaux)
> - **Effets** : Impact sur jets (modificateurs), déplacement (terrain difficile), sorts (perturbations)

## Contenu

- **5.3.3.1. Météo standard** : Soleil, pluie, neige, anomalies.
- **5.3.3.2. Phénomènes wakfu** : Tempêtes de Wakfu, anomalies temporelles.
- **5.3.3.3. Complications** : Glissades, éboulements, pannes de Zaap.
- **5.3.3.4. Effets en jeu** : Impact sur jets, déplacement, sorts.

## 5.3.3.1. Météo standard

### Table de météo (1d20)

| 1d20 | Météo | Effets |
|------|-------|--------|
| 1-8 | **Soleil** | Conditions normales |
| 9-12 | **Nuages** | Conditions normales, légère réduction visibilité |
| 13-15 | **Pluie légère** | -1 aux tests de Perception, terrain mouillé |
| 16-17 | **Pluie forte** | -2 aux tests de Perception, terrain difficile |
| 18 | **Neige** | -2 aux tests de Perception, terrain glissant, froid |
| 19 | **Tempête** | -3 aux tests de Perception, terrain très difficile, vent fort |
| 20 | **Anomalie** | Phénomène magique ou météo extrême |

### Variations saisonnières

**Printemps** :
- **Pluie** : +2 à la table
- **Soleil** : Probabilité normale
- **Neige** : -5 à la table (rare)

**Été** :
- **Soleil** : +3 à la table
- **Pluie** : Probabilité normale
- **Neige** : -10 à la table (très rare)

**Automne** :
- **Nuages** : +2 à la table
- **Pluie** : +1 à la table
- **Soleil** : Probabilité normale

**Hiver** :
- **Neige** : +5 à la table
- **Froid** : Conditions de froid fréquentes
- **Soleil** : -3 à la table (rare)

## 5.3.3.2. Phénomènes wakfu

### Table de phénomènes wakfu (1d20)

| 1d20 | Phénomène | Effets |
|------|-----------|--------|
| 1-15 | **Aucun** | Pas de phénomène magique |
| 16-17 | **Tempête de Wakfu** | Bonus aux sorts, malus aux créatures non magiques |
| 18 | **Anomalie temporelle** | Ralentissement ou accélération du temps |
| 19 | **Portail instable** | Portail magique apparaît temporairement |
| 20 | **Événement majeur** | Phénomène magique majeur (à déterminer par le MJ) |

### Tempête de Wakfu

**Effets** :
- **Sorts** : +1d4 aux jets de sorts
- **Créatures magiques** : +1 aux tests
- **Créatures non magiques** : -1 aux tests
- **Durée** : 1d4 heures

**Rareté** : Rare, mais peut survenir dans les zones magiques

### Anomalie temporelle

**Effets** :
- **Ralentissement** : Vitesse réduite de moitié
- **Accélération** : Vitesse doublée
- **Durée** : 1d6 rounds
- **Zone** : Zone limitée

**Rareté** : Très rare, zones temporelles instables

### Portail instable

**Effets** :
- **Apparition** : Portail apparaît temporairement
- **Destination** : Destination aléatoire ou déterminée
- **Durée** : 1d4 rounds
- **Risque** : Risque de fermeture prématurée

**Rareté** : Rare, zones magiques instables

## 5.3.3.3. Complications

### Table de complications (1d20)

| 1d20 | Complication | Effets |
|------|-------------|--------|
| 1-12 | **Aucune** | Pas de complication |
| 13-14 | **Glissade** | Terrain glissant, -2 aux tests d'Acrobaties |
| 15-16 | **Éboulement** | Blocage de passage, dégâts possibles |
| 17 | **Panne de Zaap** | Zaap temporairement indisponible |
| 18 | **Créature surprise** | Créature apparaît inopinément |
| 19 | **Piège naturel** | Piège environnemental se déclenche |
| 20 | **Complication majeure** | Plusieurs complications ou événement majeur |

### Glissades

**Conditions** :
- **Pluie** : Terrain mouillé, risque de glissade
- **Neige** : Terrain glissant, risque élevé
- **Glace** : Terrain très glissant, risque très élevé

**Effets** :
- **Tests** : -2 aux tests d'Acrobaties
- **Déplacement** : Vitesse réduite
- **Chutes** : Risque de chute et de dégâts

### Éboulements

**Conditions** :
- **Zones montagneuses** : Risque élevé
- **Tremblements** : Risque très élevé
- **Explosions** : Peuvent déclencher des éboulements

**Effets** :
- **Blocage** : Passage bloqué
- **Dégâts** : Dégâts de chute de pierres
- **Déblaiement** : Nécessite du temps pour déblayer

### Pannes de Zaap

**Conditions** :
- **Zones instables** : Risque élevé
- **Perturbations magiques** : Risque très élevé
- **Maintenance** : Maintenance programmée

**Effets** :
- **Indisponibilité** : Zaap temporairement indisponible
- **Durée** : 1d4 heures ou jusqu'à réparation
- **Alternatives** : Nécessite des moyens de transport alternatifs

## 5.3.3.4. Effets en jeu

### Impact sur jets

**Perception** :
- **Pluie légère** : -1 aux tests
- **Pluie forte** : -2 aux tests
- **Tempête** : -3 aux tests
- **Neige** : -2 aux tests

**Acrobaties** :
- **Terrain mouillé** : -1 aux tests
- **Terrain glissant** : -2 aux tests
- **Glace** : -3 aux tests

**Survie** :
- **Froid** : Tests de Survie pour résister au froid
- **Chaleur** : Tests de Survie pour résister à la chaleur
- **Tempête** : Tests de Survie pour se protéger

### Impact sur déplacement

**Vitesse** :
- **Terrain difficile** : Vitesse réduite de moitié
- **Terrain très difficile** : Vitesse réduite au quart
- **Glace** : Vitesse réduite, risque de chute

**PM** :
- **Coût supplémentaire** : +1 PM par case sur terrain difficile
- **Coût très élevé** : +2 PM par case sur terrain très difficile

### Impact sur sorts

**Tempête de Wakfu** :
- **Bonus** : +1d4 aux jets de sorts
- **Créatures magiques** : +1 aux tests
- **Créatures non magiques** : -1 aux tests

**Perturbations magiques** :
- **Malus** : -1 aux jets de sorts
- **Échecs** : Risque d'échec accru
- **Effets** : Effets réduits
