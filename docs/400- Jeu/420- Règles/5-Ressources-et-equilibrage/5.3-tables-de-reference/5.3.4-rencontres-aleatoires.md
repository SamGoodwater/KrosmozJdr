# 5.3.4. Rencontres aléatoires

**Description** : Listes de rencontres (combat, social, exploration) avec modificateurs selon la région et le niveau.

> **Règles rapides - Rencontres aléatoires**
> - **Combat** : Créatures sauvages (1-5), Bandits (6-10), Créatures magiques (11-13), Gardes/Miliciens (14-16), Créatures puissantes (17-18), Boss (19), Spéciale (20)
> - **Sociales** : Marchands itinérants, événements publics, PNJ variés
> - **Exploration** : Ruines, trésors, anomalies (découvertes)
> - **Modulateurs** : Ajustements selon heure, météo, réputation (contextuels)

## Contenu

- **5.3.4.1. Rencontres de combat** : Mobs, chasseurs de primes, milices.
- **5.3.4.2. Rencontres sociales** : Marchands itinérants, événements publics.
- **5.3.4.3. Rencontres d'exploration** : Ruines, trésors, anomalies.
- **5.3.4.4. Modulateurs** : Ajustements selon heure, météo, réputation.

## 5.3.4.1. Rencontres de combat

### Table de rencontres de combat (1d20)

| 1d20 | Rencontre | Niveau | Détails |
|------|----------|-------|---------|
| 1-5 | **Créatures sauvages** | Variable | Bouftous, Wabbits, Tofus, etc. |
| 6-10 | **Bandits** | Variable | Hors-la-loi, voleurs, pillards |
| 11-13 | **Créatures magiques** | Variable | Créatures élémentaires, invocations |
| 14-16 | **Gardes/Miliciens** | Variable | Forces de l'ordre, milices |
| 17-18 | **Créatures puissantes** | Élevé | Craqueleurs, créatures rares |
| 19 | **Boss/Créature unique** | Très élevé | Boss, créatures légendaires |
| 20 | **Rencontre spéciale** | Variable | Événement unique, rencontre personnalisée |

### Mobs

**Types** :
- **Créatures communes** : Bouftous, Wabbits, Tofus
- **Créatures rares** : Créatures spéciales, créatures adaptées
- **Créatures épiques** : Créatures puissantes, créatures uniques

**Niveau** : Adapté au niveau du groupe

### Chasseurs de primes

**Types** :
- **Solitaires** : Chasseurs de primes individuels
- **Groupes** : Groupes de chasseurs de primes
- **Organisations** : Organisations de chasseurs de primes

**Niveau** : Généralement niveau du groupe +1 à +3

### Milices

**Types** :
- **Gardiens** : Gardiens de villes, gardes
- **Soldats** : Soldats réguliers, milices
- **Forces spéciales** : Forces spéciales, élites

**Niveau** : Généralement niveau du groupe ±2

## 5.3.4.4. Modulateurs

### Ajustements selon heure

**Jour** :
- **Rencontres sociales** : +2 à la table
- **Rencontres de combat** : Probabilité normale
- **Rencontres d'exploration** : Probabilité normale

**Nuit** :
- **Rencontres sociales** : -3 à la table
- **Rencontres de combat** : +2 à la table
- **Rencontres d'exploration** : Probabilité normale

**Aube/Crépuscule** :
- **Rencontres sociales** : Probabilité normale
- **Rencontres de combat** : Probabilité normale
- **Rencontres d'exploration** : Probabilité normale

### Ajustements selon météo

**Conditions normales** :
- **Pas de modificateur** : Probabilités normales

**Pluie** :
- **Rencontres sociales** : -2 à la table
- **Rencontres de combat** : Probabilité normale
- **Rencontres d'exploration** : +1 à la table (obstacles)

**Tempête** :
- **Rencontres sociales** : -5 à la table
- **Rencontres de combat** : +1 à la table
- **Rencontres d'exploration** : +2 à la table (complications)

**Neige** :
- **Rencontres sociales** : -3 à la table
- **Rencontres de combat** : Probabilité normale
- **Rencontres d'exploration** : +1 à la table (obstacles)

### Ajustements selon réputation

**Réputation positive** :
- **Rencontres sociales** : +2 à la table
- **Rencontres de combat** : -2 à la table (moins d'ennemis)
- **Rencontres d'exploration** : Probabilité normale

**Réputation négative** :
- **Rencontres sociales** : -2 à la table
- **Rencontres de combat** : +2 à la table (plus d'ennemis)
- **Rencontres d'exploration** : Probabilité normale

**Réputation neutre** :
- **Pas de modificateur** : Probabilités normales

### Ajustements selon région

**Régions sûres** :
- **Rencontres sociales** : +3 à la table
- **Rencontres de combat** : -3 à la table
- **Rencontres d'exploration** : Probabilité normale

**Régions dangereuses** :
- **Rencontres sociales** : -3 à la table
- **Rencontres de combat** : +3 à la table
- **Rencontres d'exploration** : +2 à la table

**Régions légendaires** :
- **Rencontres sociales** : -5 à la table
- **Rencontres de combat** : +5 à la table
- **Rencontres d'exploration** : +3 à la table
