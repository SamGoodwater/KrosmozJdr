# 2.4.4. Capacités

**Description** : Présente les capacités, qui sont des avantages passifs ou contextuels, différents des aptitudes. Elles peuvent être fusionnées avec les aptitudes pour simplifier les règles.

> **Règles rapides - Capacités**
> - **Avantages passifs/contextuels** : Toujours actifs ou se déclenchent automatiquement
> - **Gratuites** : Ne coûtent généralement pas de ressources
> - **Progression** : Jusqu'à 6 capacités au niveau 20
> - **Types** : Passives (toujours actives), contextuelles (situations spécifiques), réactives (déclenchement automatique)
> - **Choix** : À chaque niveau de spécialisation, aptitude OU capacité

## 2.4.4.1. Concept

Les **capacités** sont des avantages **passifs ou contextuels** qui diffèrent des aptitudes.

### Distinction avec les aptitudes

**Aptitudes** :
- **Actives** : Vous devez les utiliser consciemment
- **Coûtent des ressources** : Réserve de Wakfu, PA, temps
- **Effets immédiats** : Produisent un effet quand utilisées

**Capacités** :
- **Passives** : Toujours actives ou se déclenchent automatiquement
- **Gratuites** : Ne coûtent généralement pas de ressources
- **Effets constants** : Produisent des effets en permanence ou dans certaines conditions

### Types de capacités

#### Capacités passives

Ces capacités sont **toujours actives** :
- **Bonus permanents** : +X à une caractéristique
- **Résistances** : Résistance à certains effets
- **Améliorations** : Amélioration de certaines actions

#### Capacités contextuelles

Ces capacités sont actives **dans certaines situations** :
- **En ville** : Bonus dans les environnements urbains
- **En nature** : Bonus dans les environnements naturels
- **En combat** : Bonus spécifiques au combat
- **Hors combat** : Bonus spécifiques hors combat

#### Capacités réactives

Ces capacités se **déclenchent automatiquement** dans certaines conditions :
- **Quand blessé** : Se déclenchent quand vous subissez des dégâts
- **Quand aidé** : Se déclenchent quand vous aidez quelqu'un
- **Quand menacé** : Se déclenchent face à un danger

---

## 2.4.4.2. Usage

### Quand les activer

**Capacités passives** :
- **Toujours actives** : Pas besoin de les activer
- **Automatiques** : Fonctionnent en permanence

**Capacités contextuelles** :
- **Actives dans certaines conditions** : Se déclenchent quand les conditions sont remplies
- **Détermination par le MJ** : Le MJ détermine si les conditions sont remplies

**Capacités réactives** :
- **Déclenchement automatique** : Se déclenchent quand les conditions sont remplies
- **Pas de choix** : Vous ne choisissez pas de les activer

### Impact narratif

Les capacités peuvent avoir un **impact narratif** :
- **Définissent votre personnage** : Reflètent votre spécialisation
- **Créent des opportunités** : Ouvrent de nouvelles possibilités
- **Enrichissent le roleplay** : Ajoutent de la profondeur à votre personnage

### Gestion

Les capacités sont généralement **faciles à gérer** :
- **Pas de ressources** : Ne coûtent généralement rien
- **Automatiques** : Fonctionnent sans intervention
- **Claires** : Effets bien définis

## 2.4.4.3. Exemples

Voici des exemples de capacités pour chaque spécialisation :

### Érudit

- **Mémoire parfaite** : Vous vous souvenez de tout ce que vous avez lu ou entendu
- **Connaissance approfondie** : Bonus aux tests de connaissances
- **Analyse rapide** : Analysez les situations plus rapidement

### Milicien

- **Discipline de fer** : Résistance aux effets de peur et de charme
- **Formation militaire** : Bonus aux tests de tactique et de commandement
- **Endurance** : Résistance à la fatigue

### Dévot

- **Aura de protection** : Les alliés proches bénéficient d'une légère protection
- **Résistance aux sorts** : Résistance aux sorts maléfiques
- **Guidance divine** : Bonus aux tests de Sagesse

### Artiste

- **Charisme naturel** : Bonus aux tests sociaux
- **Inspiration des autres** : Vos performances inspirent les autres
- **Créativité** : Bonus aux tests de création

### Négociant

- **Réseau de contacts** : Vous connaissez des personnes utiles dans chaque ville
- **Réduction des prix** : Obtenez des réductions lors des achats
- **Évaluation rapide** : Identifiez rapidement la valeur des objets

### Explorateur

- **Orientation naturelle** : Vous ne vous perdez jamais
- **Résistance aux dangers** : Résistance aux effets de terrain
- **Détection améliorée** : Bonus aux tests de Perception

### Sylvain

- **Affinité avec la nature** : Les animaux vous font confiance
- **Connaissance de la nature** : Bonus aux tests liés à la nature
- **Résistance naturelle** : Résistance aux poisons et maladies naturelles

### Artisan

- **Fabrication améliorée** : Créez des objets de meilleure qualité
- **Réparation rapide** : Réparez les objets plus rapidement
- **Identification des matériaux** : Identifiez les matériaux et leur qualité

### Voleur

- **Furtivité améliorée** : Bonus aux tests de Discrétion
- **Détection de trésors** : Sens spécial pour trouver des objets cachés
- **Instinct de survie** : Bonus aux tests d'esquive et de fuite

## 2.4.4.4. Simplification

### Fusion possible avec les aptitudes

Il est possible de **fusionner les capacités avec les aptitudes** pour simplifier les règles.

### Avantages de la fusion

- **Simplicité** : Moins de concepts à gérer
- **Clarté** : Plus facile à comprendre
- **Flexibilité** : Les aptitudes peuvent avoir des effets passifs

### Inconvénients de la fusion

- **Perte de distinction** : Moins de variété dans les mécaniques
- **Complexité des aptitudes** : Les aptitudes deviennent plus complexes

### Recommandation

**Option 1 : Garder la distinction**
- **Avantages** : Plus de variété, mécaniques claires
- **Inconvénients** : Plus de concepts à gérer

**Option 2 : Fusionner**
- **Avantages** : Plus simple, moins de concepts
- **Inconvénients** : Moins de variété

**Note** : Cette décision peut être prise par le MJ selon les préférences du groupe. Les deux approches sont valides.

## Conseils d'utilisation

### Pour les joueurs

1. **Comprenez vos capacités** : Sachez quand elles sont actives
2. **Utilisez-les** : N'oubliez pas vos capacités passives
3. **Créez des opportunités** : Utilisez vos capacités pour créer des moments intéressants

### Pour le MJ

1. **Rappelez les capacités** : Aidez les joueurs à se souvenir de leurs capacités
2. **Créez des situations** : Créez des situations où les capacités sont utiles
3. **Équilibrez** : Assurez-vous que toutes les capacités sont utiles
