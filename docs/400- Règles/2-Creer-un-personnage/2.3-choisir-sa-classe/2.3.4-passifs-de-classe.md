# 2.3.4. Passifs de classe

**Description** : Présente les passifs uniques de chaque classe, leurs mécaniques spécifiques et leurs impacts sur le gameplay.

> **Règles rapides - Passifs de classe**
> - **Unique** : Chaque classe a un passif unique qui définit son gameplay
> - **Actif niveau 1** : Disponible dès le niveau 1
> - **Gratuit** : Ne coûte généralement pas de PA (sauf exceptions)
> - **Permanent** : Actif tant que les conditions sont remplies
> - **Équilibré** : A des limites pour maintenir l'équilibre

## 2.3.4.1. Concept

Le **passif de classe** est une capacité unique et permanente qui définit l'identité et le gameplay de chaque classe.

### Rôle du passif

Le passif :
- **Définit l'identité** : C'est ce qui rend chaque classe unique
- **Influence le gameplay** : Change la façon dont vous jouez votre classe
- **Crée des synergies** : S'associe avec vos sorts et votre équipement
- **Équilibre les classes** : Compense les faiblesses ou renforce les forces

### Inspiration Dofus

Les passifs sont directement inspirés des passifs du jeu vidéo Dofus, mais **enrichis** pour le format JDR :
- Mécaniques adaptées pour le jeu de rôle
- Interactions avec le système PA/PM/PO
- Synergies avec les spécialisations et l'équipement

### Caractéristiques communes

Tous les passifs :
- Sont **actifs dès le niveau 1**
- **Ne coûtent pas de PA** (sauf exceptions spécifiques)
- Sont **permanents** (tant que les conditions sont remplies)
- Ont des **limites** pour l'équilibrage

## 2.3.4.2. Exemples

Voici quelques exemples de passifs pour illustrer leur diversité :

### Osamodas : Invocation permanente

**Description** : L'Osamodas peut conserver une invocation permanente grâce à sa dévotion. Il doit choisir une voie élémentaire qui détermine le type d'invocations utilisables de cette manière.

**Mécanique** :
- **1 invocation permanente** (en plus des invocations normales)
- **Choix de voie** : Détermine le type d'invocations disponibles
- **Concentration** : Nécessite de la concentration pour maintenir

**Impact** : L'Osamodas peut toujours avoir une créature à ses côtés, créant un avantage tactique constant.

### Sram : Réseaux de pièges

**Description** : Le Sram peut poser des pièges qui restent actifs sur le terrain. Le nombre maximum de pièges dépend du niveau.

**Mécanique** :
- **Limite de pièges** : 2-7 selon le niveau (voir tableau section 3.3)
- **Pièges cachés** : Nécessitent un test de Perception pour être détectés
- **Dégâts indirects** : Se déclenchent quand une créature marche dessus

**Impact** : Le Sram contrôle le terrain et peut préparer des embuscades.

### Féca : Glyphes protecteurs

**Description** : Le Féca peut créer des glyphes magiques visibles qui offrent des bonus ou des malus aux créatures qui passent dessus.

**Mécanique** :
- **Glyphes visibles** : Magiques, ont une aura de Wakfu
- **Limite de glyphes** : 2-7 selon le niveau
- **Effets cumulatifs** : Les glyphes peuvent se superposer
- **Déclenchement** : Au début du tour (combat) ou à l'arrêt (hors combat)

**Impact** : Le Féca crée des zones de contrôle et de protection pour son équipe.

### Iop : Fureur

**Description** : L'Iop gagne en puissance à mesure qu'il combat. Plus il fait de dégâts, plus il devient puissant.

**Mécanique** :
- **Accumulation** : Gagne des bonus en fonction des dégâts infligés
- **Décroissance** : Perd progressivement les bonus si aucun dégât n'est fait
- **Limite** : Bonus maximum selon le niveau

**Impact** : L'Iop devient de plus en plus dangereux au fil du combat.

### Eniripsa : Soins améliorés

**Description** : Les soins de l'Eniripsa sont plus efficaces et peuvent avoir des effets supplémentaires.

**Mécanique** :
- **Bonus de soin** : Soins augmentés selon le niveau
- **Effets supplémentaires** : Peut ajouter des bonus temporaires en soignant
- **Synergie** : Les soins peuvent déclencher d'autres effets

**Impact** : L'Eniripsa est le meilleur soigneur, capable de maintenir son équipe en vie.

### Xélor : Manipulation du temps

**Description** : Le Xélor peut manipuler l'ordre d'initiative et les ressources temporelles.

**Mécanique** :
- **Vol de PA/PM** : Peut retirer des PA/PM aux ennemis
- **Don de PA/PM** : Peut donner des PA/PM aux alliés
- **Manipulation d'initiative** : Peut modifier l'ordre d'action

**Impact** : Le Xélor contrôle le rythme du combat et désorganise l'ennemi.

**Note** : Pour la liste complète des passifs de toutes les classes, consultez la **Bibliothèque → Les classes** sur le site web.

## 2.3.4.3. Limites

Tous les passifs ont des limites pour maintenir l'équilibrage du jeu.

### Limites communes

- **Ressources** : Utilisation limitée par tour/jour
- **Nombre** : Limite du nombre d'effets actifs simultanément
- **Portée** : Limite de portée ou de zone d'effet
- **Conditions** : Nécessitent certaines conditions pour être activés

### Exemples de limites

**Osamodas** :
- **1 seule invocation permanente** (en plus des normales)
- **Choix de voie** : Limite le type d'invocations
- **Concentration** : Difficile à maintenir hors combat longtemps

**Sram** :
- **2-7 pièges maximum** selon le niveau
- **Détection** : Peuvent être détectés et désamorcés
- **Dégâts indirects** : Moins puissants que les dégâts directs

**Féca** :
- **2-7 glyphes maximum** selon le niveau
- **Visibles** : Les ennemis peuvent les éviter
- **Déclenchement** : Conditions spécifiques pour se déclencher

### Équilibrage

Les passifs sont équilibrés pour :
- **Ne pas être trop puissants** : Limites claires
- **Rester intéressants** : Utiles sans être obligatoires
- **Créer des choix** : Différentes façons de les utiliser
- **S'adapter au niveau** : Évoluent avec la progression

## 2.3.4.4. Impact gameplay

Le passif influence profondément la façon dont vous jouez votre classe.

### Synergies avec les voies

Votre passif peut s'associer particulièrement bien avec certaines voies :

**Exemples** :
- **Osamodas voie Feu** : Invoque des créatures offensives
- **Sram voie Terre** : Pièges de zone plus puissants
- **Féca voie Eau** : Glyphes de soin et protection

### Synergies avec les spécialisations

Votre spécialisation peut renforcer votre passif :

**Exemples** :
- **Spécialisation avec maîtrise Perception** : Détecte mieux les pièges (Sram)
- **Spécialisation avec maîtrise Nature** : Meilleure connexion avec les invocations (Osamodas)
- **Spécialisation avec expertise** : Passif plus efficace

### Rôle dans l'équipe

Votre passif définit souvent votre rôle principal dans l'équipe :

- **Osamodas** : Contrôle de créatures, présence sur le terrain
- **Sram** : Contrôle du terrain, embuscades
- **Féca** : Protection de l'équipe, zones de contrôle
- **Iop** : DPS croissant, menace principale
- **Eniripsa** : Soigneur principal, support
- **Xélor** : Contrôle du rythme, désorganisation

### Stratégies avancées

Les joueurs expérimentés peuvent :
- **Combiner le passif avec l'équipement** : Créer des synergies puissantes
- **Utiliser le passif de manière créative** : Trouver des usages inattendus
- **Adapter le passif à la situation** : Changer de stratégie selon le contexte

## Conseils d'utilisation

### Pour les débutants

1. **Lisez bien votre passif** : Comprenez exactement ce qu'il fait
2. **Utilisez-le régulièrement** : C'est une partie importante de votre classe
3. **Demandez de l'aide** : Votre MJ peut vous expliquer les subtilités

### Pour les joueurs expérimentés

1. **Optimisez votre passif** : Trouvez les meilleures façons de l'utiliser
2. **Créez des synergies** : Combinez-le avec vos sorts et votre équipement
3. **Expérimentez** : Testez différentes approches

### Erreurs courantes

- **Oublier son passif** : C'est une capacité importante, utilisez-la !
- **Surutiliser son passif** : Respectez les limites pour ne pas épuiser vos ressources
- **Ignorer les limites** : Les limites existent pour l'équilibrage, respectez-les
