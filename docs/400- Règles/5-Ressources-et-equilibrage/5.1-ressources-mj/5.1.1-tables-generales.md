# 5.1.1. Tables générales

**Description** : Réunir les tables essentielles (difficultés, seuils de dégâts, oppositions) pour accélérer les décisions du MJ.

> **Règles rapides**
> - Basez vos DD sur 15 puis ajustez ±5 selon l'avantage ou le désavantage
> - Dégâts attendus par tour : +10 tous les 4 niveaux
> - PA/PM des ennemis : alignés sur les personnages de niveau équivalent
> - Utilisez les tableaux ci-dessous pour gagner du temps en jeu

## Contenu

- **5.1.1.1. Difficultés standards** : Barèmes pour tâches simples, moyennes, difficiles.
- **5.1.1.2. Seuils de dégâts** : Dégâts moyens par niveau, conseils d'ajustement.
- **5.1.1.3. Oppositions** : Matrices pour tests opposés courants.
- **5.1.1.4. États rapides** : Résumé des effets fréquents (stun, entrave, etc.).

## 5.1.1.1. Difficultés standards

Les difficultés standards permettent au MJ d'évaluer rapidement la difficulté d'une action.

### Barème de base

| DD | Difficulté | Exemple |
|----|------------|---------|
| **5** | Très facile | Ouvrir une porte non verrouillée, soulever un objet léger |
| **10** | Facile | Escalader une corde, convaincre un ami, crocheter une serrure simple |
| **15** | Modérée | Escalader une paroi glissante, persuader un inconnu, crocheter une serrure complexe |
| **20** | Difficile | Escalader une paroi verticale, convaincre un ennemi, crocheter une serrure très complexe |
| **25** | Très difficile | Escalader une paroi sans prise, convaincre un roi, crocheter une serrure légendaire |
| **30** | Presque impossible | Escalader une paroi lisse et verticale, convaincre un dieu, crocheter une serrure magique |

### Ajustements rapides

**Facteurs favorables** (-2 à -5 DD) :
- **Outils appropriés** : -2 à -3 DD
- **Aide d'un allié** : -2 à -3 DD
- **Conditions favorables** : -1 à -2 DD
- **Préparation** : -1 à -2 DD

**Facteurs défavorables** (+2 à +5 DD) :
- **Manque d'outils** : +2 à +3 DD
- **Conditions défavorables** : +1 à +2 DD
- **Urgence** : +1 à +2 DD
- **Obstacles** : +2 à +3 DD

### Par niveau

**Personnages de bas niveau (1-5)** :
- **Actions simples** : DD 10
- **Actions normales** : DD 15
- **Actions difficiles** : DD 20

**Personnages de niveau moyen (6-10)** :
- **Actions simples** : DD 15
- **Actions normales** : DD 20
- **Actions difficiles** : DD 25

**Personnages de haut niveau (11-20)** :
- **Actions simples** : DD 20
- **Actions normales** : DD 25
- **Actions difficiles** : DD 30

> **Exemple MJ**
> - Le groupe veut convaincre un gouverneur (importance élevée) → DD 20.
> - Ils disposent de preuves irréfutables → -2 au DD.
> - Ils sont pressés par le temps → +2 au DD.
> - **DD final** : 20 (les modificateurs s'annulent).

## 5.1.1.2. Seuils de dégâts

Les seuils de dégâts aident à équilibrer les combats selon le niveau des personnages.

### Dégâts moyens par niveau

| Niveau | Dégâts moyens par tour | Dégâts moyens par sort |
|--------|------------------------|------------------------|
| **1-3** | 5-10 | 1d6+mod |
| **4-7** | 10-20 | 2d6+mod |
| **8-11** | 20-30 | 3d6+mod |
| **12-15** | 30-40 | 4d6+mod |
| **16-20** | 40-50+ | 5d6+mod |

### Points de vie moyens par niveau

| Niveau | PV moyens | PV boss |
|--------|-----------|---------|
| **1-3** | 20-40 | 50-80 |
| **4-7** | 40-80 | 80-150 |
| **8-11** | 80-120 | 150-250 |
| **12-15** | 120-180 | 250-400 |
| **16-20** | 180-250+ | 400-600+ |

### Conseils d'ajustement

**Groupe faible** :
- **Réduire** : Réduire les dégâts de 20-30%
- **PV** : Réduire les PV de 20-30%
- **Simplifier** : Simplifier les capacités spéciales

**Groupe puissant** :
- **Augmenter** : Augmenter les dégâts de 20-30%
- **PV** : Augmenter les PV de 20-30%
- **Complexifier** : Ajouter des capacités spéciales

**Groupe équilibré** :
- **Standard** : Utiliser les valeurs de base
- **Varier** : Varier les types d'ennemis
- **Adapter** : Adapter selon les forces et faiblesses

## 5.1.1.3. Oppositions

Les tests opposés sont fréquents dans le jeu.

### Matrices courantes

**Discrétion vs Perception** :
- **Usage** : Se cacher, se faufiler
- **Test** : Discrétion (Agilité) vs Perception (Sagesse)
- **Résultat** : Le résultat le plus élevé l'emporte

**Persuasion vs Perspicacité** :
- **Usage** : Convaincre, mentir
- **Test** : Persuasion (Chance) vs Perspicacité (Sagesse)
- **Résultat** : Le résultat le plus élevé l'emporte

**Athlétisme vs Athlétisme** :
- **Usage** : Pousser, tirer, lutter
- **Test** : Athlétisme (Force) vs Athlétisme (Force)
- **Résultat** : Le résultat le plus élevé l'emporte

**Intimidation vs Intimidation** :
- **Usage** : Menacer, imposer
- **Test** : Intimidation (Force/Chance) vs Intimidation (Force/Chance)
- **Résultat** : Le résultat le plus élevé l'emporte

### Gestion des égalités

**Règle** : Si les résultats sont identiques, la situation reste inchangée.

**Exemples** :
- **Discrétion vs Perception** : Si égalité, le personnage caché n'est pas détecté mais ne progresse pas
- **Persuasion vs Perspicacité** : Si égalité, la négociation reste bloquée
- **Athlétisme vs Athlétisme** : Si égalité, aucun des deux ne progresse

## 5.1.1.4. États rapides

Résumé des effets fréquents pour référence rapide.

### États positifs

**Bénédiction** :
- **Effet** : +1d4 aux jets d'attaque et de sauvegarde
- **Durée** : 1 minute
- **Source** : Sorts, potions

**Protection** :
- **Effet** : Réduction des dégâts
- **Durée** : Variable
- **Source** : Sorts, équipements

**Hâte** :
- **Effet** : +2 PM, action bonus
- **Durée** : 1 minute
- **Source** : Sorts, potions

### États négatifs

**Stun** :
- **Effet** : Ne peut pas agir, désavantage à la CA
- **Durée** : 1 tour ou jusqu'à sauvegarde
- **Source** : Sorts, capacités

**Entrave** :
- **Effet** : Vitesse réduite à 0, désavantage à l'esquive
- **Durée** : Variable
- **Source** : Sorts, capacités, pièges

**Ralentissement** :
- **Effet** : Vitesse réduite de moitié, -2 PM
- **Durée** : Variable
- **Source** : Sorts, capacités

**Empoisonnement** :
- **Effet** : Dégâts de poison chaque tour, désavantage aux tests
- **Durée** : Variable
- **Source** : Poisons, sorts

**Brûlure** :
- **Effet** : Dégâts de feu chaque tour
- **Durée** : Variable
- **Source** : Feu, sorts de feu

**Gel** :
- **Effet** : Vitesse réduite, dégâts de froid
- **Durée** : Variable
- **Source** : Froid, sorts de glace

### États neutres

**Pesanteur** :
- **Effet** : Ne peut pas être déplacé
- **Durée** : Variable
- **Source** : Sorts, capacités
- **Note** : Peut être bénéfique ou néfaste selon le contexte

**Invisibilité** :
- **Effet** : Invisible, avantage aux attaques
- **Durée** : Variable
- **Source** : Sorts, capacités, équipements

---

### À retenir

- Par défaut, utilisez **DD 15** puis ajustez par paliers de 5.
- Les dégâts moyens augmentent d'environ 10 tous les 4 niveaux.
- Ajustez la difficulté en agissant sur **les dégâts, les PV et la complexité** plutôt que sur un seul paramètre.
- Notez vos propres variantes (ex : table maison des états) pour accélérer vos parties.
