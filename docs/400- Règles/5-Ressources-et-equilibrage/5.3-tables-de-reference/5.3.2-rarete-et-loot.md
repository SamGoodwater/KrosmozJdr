# 5.3.2. Rareté et loot

**Description** : Tables de rareté des objets/ressources, probabilités de loot, adaptations selon la zone.

> **Règles rapides - Rareté et loot**
> - **Rareté** : Commun (60-70%, 10-50 kamas), Invo (15-25%, 50-200 kamas), Rare (10-15%, 200-1000 kamas), Épique (3-5%, 1000-5000 kamas), Légendaire (0.5-1%, 5000+ kamas)
> - **Probabilités** : Jet de loot (1d100), modificateurs de groupe (bonus/malus)
> - **Zones** : Ajustements selon région, donjon, boss (modificateurs contextuels)
> - **Panoplies** : Chances d'obtenir des pièces manquantes (probabilités ajustées)

## Contenu

- **5.3.2.1. Rareté** : Commun, invo, rare, épique, légendaire.
- **5.3.2.2. Probabilités** : Jet de loot, modificateurs de groupe.
- **5.3.2.3. Zones** : Ajustements selon région, donjon, boss.
- **5.3.2.4. Bonus de panoplie** : Chances d'obtenir des pièces manquantes.

---

## 5.3.2.1. Rareté

### Échelle de rareté

**Commun** :
- **Probabilité** : 60-70% des objets trouvés
- **Valeur** : 10-50 kamas
- **Usage** : Utilisé dans de nombreuses recettes de base

**Invo (Invocation)** :
- **Probabilité** : 15-25% des objets trouvés
- **Valeur** : 50-200 kamas
- **Usage** : Utilisé dans des recettes avancées

**Rare** :
- **Probabilité** : 10-15% des objets trouvés
- **Valeur** : 200-1000 kamas
- **Usage** : Utilisé dans des recettes épiques

**Épique** :
- **Probabilité** : 3-5% des objets trouvés
- **Valeur** : 1000-5000 kamas
- **Usage** : Utilisé dans des recettes légendaires

**Légendaire** :
- **Probabilité** : 1-2% des objets trouvés
- **Valeur** : 5000+ kamas, parfois inestimables
- **Usage** : Utilisé dans des recettes légendaires

## 5.3.2.2. Probabilités

### Jet de loot

**Table de base** (1d100) :

| 1d100 | Rareté |
|-------|--------|
| 1-60 | Commun |
| 61-80 | Invo |
| 81-90 | Rare |
| 91-97 | Épique |
| 98-100 | Légendaire |

### Modificateurs de groupe

**Taille du groupe** :
- **1 joueur** : Pas de modificateur
- **2-3 joueurs** : +5 à la table
- **4-5 joueurs** : +10 à la table
- **6+ joueurs** : +15 à la table

**Niveau du groupe** :
- **Niveau faible** : -10 à la table
- **Niveau moyen** : Pas de modificateur
- **Niveau élevé** : +10 à la table

**Difficulté de la rencontre** :
- **Facile** : -10 à la table
- **Normale** : Pas de modificateur
- **Difficile** : +10 à la table
- **Très difficile** : +20 à la table

---

## 5.3.2.3. Zones

### Ajustements selon région

**Régions communes** :
- **Modificateur** : Pas de modificateur
- **Rareté** : Rareté standard
- **Exemples** : Amakna, Astrub

**Régions dangereuses** :
- **Modificateur** : +10 à la table
- **Rareté** : Rareté améliorée
- **Exemples** : Donjons, zones hostiles

**Régions légendaires** :
- **Modificateur** : +20 à la table
- **Rareté** : Rareté très améliorée
- **Exemples** : Zones épiques, sanctuaires

### Ajustements selon donjon

**Donjons simples** :
- **Modificateur** : +5 à la table
- **Rareté** : Rareté légèrement améliorée

**Donjons moyens** :
- **Modificateur** : +10 à la table
- **Rareté** : Rareté améliorée

**Donjons complexes** :
- **Modificateur** : +15 à la table
- **Rareté** : Rareté très améliorée

**Donjons épiques** :
- **Modificateur** : +25 à la table
- **Rareté** : Rareté exceptionnelle

### Ajustements selon boss

**Boss normaux** :
- **Modificateur** : +20 à la table
- **Rareté** : Rareté très améliorée
- **Garantie** : Au moins un objet rare

**Boss épiques** :
- **Modificateur** : +30 à la table
- **Rareté** : Rareté exceptionnelle
- **Garantie** : Au moins un objet épique

**Boss légendaires** :
- **Modificateur** : +40 à la table
- **Rareté** : Rareté légendaire
- **Garantie** : Au moins un objet légendaire

## 5.3.2.4. Bonus de panoplie

### Chances d'obtenir des pièces manquantes

**Table de base** (1d100) :

| 1d100 | Résultat |
|-------|----------|
| 1-50 | Pas de pièce de panoplie |
| 51-75 | Pièce commune de panoplie |
| 76-90 | Pièce rare de panoplie |
| 91-98 | Pièce épique de panoplie |
| 99-100 | Pièce légendaire de panoplie |

### Modificateurs

**Pièces déjà possédées** :
- **0 pièce** : Pas de modificateur
- **1-2 pièces** : +10 à la table
- **3-4 pièces** : +20 à la table
- **5+ pièces** : +30 à la table

**Niveau de la panoplie** :
- **Niveau faible** : +10 à la table
- **Niveau moyen** : Pas de modificateur
- **Niveau élevé** : -10 à la table

**Type de rencontre** :
- **Rencontre normale** : Pas de modificateur
- **Boss** : +20 à la table
- **Boss épique** : +30 à la table

### Gestion spéciale

**Quêtes de panoplie** :
- **Fonction** : Quêtes spécifiques pour obtenir des pièces
- **Récompense** : Pièce garantie à la fin de la quête
- **Rareté** : Rareté selon la difficulté de la quête

**Échanges** :
- **Fonction** : Échanger des pièces avec d'autres joueurs
- **Avantage** : Permet de compléter les panoplies
- **Économie** : Crée une économie d'échange
