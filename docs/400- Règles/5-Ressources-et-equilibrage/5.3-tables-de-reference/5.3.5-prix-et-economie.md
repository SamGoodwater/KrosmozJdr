# 5.3.5. Prix et économie

**Description** : Barèmes de prix (équipements, services), variations régionales, inflation, troc.

> **Règles rapides - Prix et économie**
> - **Prix standard** : Armes (+1 = 100 kamas, +2 = 200 kamas, etc.), Armures, Accessoires, Consommables
> - **Services** : Soins (50-200 kamas), téléportations (10-50 kamas), informations (variable)
> - **Variations régionales** : Offres/demandes par zone (modificateurs de prix)
> - **Troc** : Règles pour échanges sans Kamas (valeur relative des objets)

## Contenu

- **5.3.5.1. Prix standard** : Liste indicative par catégorie d'objet.
- **5.3.5.2. Services** : Soins, téléportations, informations.
- **5.3.5.3. Variations régionales** : Offres/demandes par zone.

## 5.3.5.1. Prix standard

### Armes

**Bonus de touche** :
- **+1** : 100 kamas par unité
- **+2** : 200 kamas par unité
- **+3** : 300 kamas par unité
- **+4** : 400 kamas par unité
- **+5** : 500 kamas par unité

**Bonus de dégâts** :
- **Dégâts fixes** : 700 kamas par unité
- **Dégâts multiples** : 900 kamas par unité

**Éléments** :
- **Neutre** : Pas de coût supplémentaire
- **Terre, Feu, Air, Eau** : Pas de coût supplémentaire

**Effets bonus** :
- **Avantage au toucher** : +1000 kamas
- **Autres effets** : Variable selon l'effet

### Équipements

**Caractéristiques** (par unité) :

| Caractéristique | Prix |
|-----------------|------|
| **Vitalité** | 1000-3000 kamas |
| **Force** | 1000 kamas |
| **Intelligence** | 1000 kamas |
| **Sagesse** | 1000 kamas |
| **Agilité** | 1000 kamas |
| **Chance** | 1000 kamas |
| **PA** | 500 kamas |
| **PM** | 600 kamas |
| **PO** | 800 kamas |
| **Nombre d'invocations** | 600 kamas |
| **Bonus d'initiative** | 400 kamas |
| **Esquive** | 70 kamas |
| **Recharge de magie** | 300 kamas |
| **Tacle/Fuite** | 1000 kamas |

**CA et résistances** (par unité) :

| Caractéristique | Prix |
|-----------------|------|
| **CA** | 400 kamas |
| **Résistance fixe** | 700 kamas |

**Compétences** (par unité) : 300 kamas

### Consommables

**Règle générale** : Minimum 10 × niveau du consommable

**Potions de soins** :
- **Niveau 1** : 150 kamas
- **Niveau 5** : 500 kamas
- **Niveau 10** : 1000 kamas

**Parchemins d'apprentissage** :
- **Règle** : 2000 × niveau du sort
- **Exemple** : Sort niveau 13 = 26 000 kamas

**Autres consommables** : Variable selon l'effet et la rareté

## 5.3.5.3. Variations régionales

### Grandes villes

**Caractéristiques** :
- **Concurrence** : Grande concurrence entre vendeurs
- **Prix** : Prix généralement plus bas
- **Disponibilité** : Grande disponibilité d'objets

**Modificateurs** :
- **Achat** : -10% à -20% sur les prix
- **Vente** : -10% à -20% sur les prix de vente
- **Services** : Prix standards

### Petites villes

**Caractéristiques** :
- **Concurrence** : Concurrence limitée
- **Prix** : Prix généralement plus élevés
- **Disponibilité** : Disponibilité limitée

**Modificateurs** :
- **Achat** : +10% à +20% sur les prix
- **Vente** : +10% à +20% sur les prix de vente
- **Services** : Prix standards à légèrement élevés

### Régions reculées

**Caractéristiques** :
- **Concurrence** : Pas de concurrence
- **Prix** : Prix très variables
- **Disponibilité** : Disponibilité très limitée

**Modificateurs** :
- **Achat** : +20% à +50% sur les prix
- **Vente** : -20% à -50% sur les prix de vente
- **Services** : Prix élevés ou indisponibles

### Zones de guerre

**Caractéristiques** :
- **Demande** : Demande élevée pour les armes
- **Prix** : Prix élevés pour les armes
- **Disponibilité** : Disponibilité limitée

**Modificateurs** :
- **Armes** : +50% à +100% sur les prix
- **Armures** : +20% à +50% sur les prix
- **Autres** : Prix variables

## 5.3.5.4. Troc

### Règles de base

**Définition** : Échange d'objets sans utiliser de Kamas.

**Utilisation** :
- **Régions reculées** : Plus fréquent dans les régions reculées
- **Objets rares** : Utile pour les objets rares
- **Relations** : Crée des relations avec les PNJ

### Évaluation

**Valeur des objets** :
- **Valeur de base** : Utiliser les prix standards
- **Valeur relative** : Adapter selon le contexte
- **Valeur sentimentale** : Prendre en compte la valeur sentimentale

**Négociation** :
- **Tests** : Tests de Persuasion pour négocier
- **Relations** : Relations avec le PNJ affectent la négociation
- **Contexte** : Contexte affecte la négociation

### Exemples

**Échange simple** :
- **Objet A** : Valeur 500 kamas
- **Objet B** : Valeur 500 kamas
- **Échange** : Échange direct

**Échange complexe** :
- **Objet A** : Valeur 1000 kamas
- **Objet B** : Valeur 500 kamas
- **Compensation** : Compensation nécessaire (objets supplémentaires ou Kamas)

**Échange avec relations** :
- **Relations positives** : Meilleur échange possible
- **Relations négatives** : Échange désavantageux
- **Relations neutres** : Échange standard
