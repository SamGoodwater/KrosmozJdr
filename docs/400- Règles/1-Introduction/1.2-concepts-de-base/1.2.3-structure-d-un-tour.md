# 1.2.3. Structure d'un tour

**Description** : Résumer le déroulement d'un tour (hors combat et en combat), types d'actions disponibles, réactions.

> **Règles rapides - Structure d'un tour**
>
> - **Hors combat** : Actions libres, tests si nécessaire, sorts consomment Wakfu
> - **En combat** : Début (régénération PA/PM) → Actions (PA/PM) → Fin (effets persistants)
> - **Actions** : Standard (3-4 PA), Forte (5 PA), Bonus (2-3 PA)
> - **Ordre** : Déterminé par l'initiative, reste le même pour tous les rounds

## Contenu

- **1.2.3.1. Tour hors combat** : Actions simples, exploration, interactions.
- **1.2.3.2. Tour de combat** : Séquence standard (début, actions, fin).
- **1.2.3.3. Actions disponibles** : Standard, forte, bonus, réactions.
- **1.2.3.4. Gestion du temps** : Tours vs rounds, transitions.

---

## 1.2.3.1. Tour hors combat

Hors combat, les personnages peuvent effectuer **n'importe quelle action** que le MJ juge faisable.

### Actions simples

Certaines actions sont **triviales** et ne nécessitent pas de test :

- Ouvrir une porte normale
- Parler
- Observer passivement
- Se déplacer normalement

### Actions nécessitant un test

D'autres actions nécessitent un **test de caractéristique** :

- Escalader une paroi
- Convaincre un PNJ
- Détecter un piège
- Se cacher

Le MJ détermine la compétence requise et la difficulté (DD).

### Utilisation de sorts hors combat

Les personnages peuvent utiliser des sorts hors combat :

- **Spécificités** : Certains sorts ont des règles spéciales hors combat
- **Réserve de Wakfu** : Les sorts hors combat consomment la réserve de Wakfu

### Exploration et interactions

Hors combat, les personnages peuvent :

- Explorer des lieux
- Interagir avec l'environnement
- Parler avec des PNJ
- Préparer des stratégies

---

## 1.2.3.2. Tour de combat

Un tour de combat suit une **séquence standard** :

### Début de tour

1. **Régénération** : PA et PM reviennent à leur maximum (sauf si un sort l'empêche)
2. **Effets persistants** : Les effets en cours continuent (états, sorts, etc.)
3. **Sauvegardes** : Tests de sauvegarde si nécessaire

### Actions du tour

Le personnage peut utiliser ses **PA et PM** pour :

- Se déplacer (PM)
- Lancer des sorts (PA)
- Attaquer (PA)
- Utiliser des aptitudes (PA)
- Effectuer d'autres actions (PA)

### Fin de tour

1. **Résolution des effets** : Les effets de fin de tour se déclenchent
2. **Passage au suivant** : Le tour suivant dans l'ordre d'initiative commence

### Ordre d'initiative

Les tours se succèdent selon l'**ordre d'initiative** déterminé au début du combat. Cet ordre reste le même pour tous les rounds.

---

## 1.2.3.3. Actions disponibles

### Actions standard

Les actions standard coûtent des **PA** :

- **Lancer un sort** : Coût variable selon le sort
- **Attaquer avec une arme** : Coût variable selon l'arme
- **Aider** : 2 PA - Donne un avantage à un allié
- **Chercher** : Tous les PA - Recherche approfondie
- **Esquiver** : 3 PA + 2 PM - Réduit les chances d'être touché
- **Se cacher** : 5 PA - Tente de se cacher
- **Utiliser un objet** : Coût variable selon l'objet

### Actions fortes

Les actions fortes coûtent **beaucoup de PA** mais ont des effets puissants :

- Certains sorts de haut niveau
- Capacités spéciales des classes
- Actions tactiques complexes

### Actions bonus

Les actions bonus sont **gratuites** ou coûtent très peu :

- Certaines aptitudes de classe
- Utilisation d'objets spécifiques
- Réactions conditionnelles

### Réactions

Les réactions peuvent être utilisées **en dehors de votre tour** :

- **Tacle** : Empêcher un ennemi de fuir
- **Fuite** : Quitter un corps-à-corps
- **Contre-sorts** : Réagir à un sort ennemi avec un sort

Les réactions sont limitées à **une par round et coute des PA de la réserve de Wakfu**.

---

## 1.2.3.4. Gestion du temps

### Tours vs rounds

- **Round** : Unité de temps de 10 secondes
- **Tour** : Action d'un personnage pendant un round
- **Combat** : Série de rounds jusqu'à la fin du combat

### Transitions

**Hors combat → Combat** :

- Détermination de la surprise
- Calcul de l'initiative
- Passage aux rounds de 10 secondes

**Combat → Hors combat** :

- Retour à la gestion du temps normale (minutes, heures, jours)
- Récupération des ressources selon les règles hors combat

### Échelle de temps

- **Combat** : 10 secondes par round
- **Exploration (donjon)** : Minutes
- **Voyage (ville/nature)** : Heures
- **Longs voyages** : Jours

Le MJ adapte l'échelle selon la situation.

---

### À retenir

- **Hors combat** : Actions libres, tests si nécessaire, sorts consomment la réserve de Wakfu
- **En combat** : Séquence standard (début → actions → fin), PA/PM régénèrent chaque tour
- **Actions** : Standard (3-4 PA), Forte (5 PA), Bonus (2-3 PA)
- **Ordre** : Déterminé par l'initiative au début du combat, reste le même pour tous les rounds
- **Temps** : Combat = 10 secondes/round, Exploration = minutes/heures/jours
