## 1.1.4.1. Site officiel

Le site officiel **krosmoz-jdr.fr** est actuellement en cours de création et proposera :

### Fonctionnalités prévues

- **Règles en ligne** : Accès à toutes les règles du jeu
- **Base de données** : 
  - Liste complète des sorts de toutes les classes
  - Catalogue des équipements et panoplies
  - Bestiaire des créatures
  - Liste des ressources et consommables
- **Fiches de personnage interactives** : Création et gestion de personnages en ligne
- **Générateurs** : 
  - Générateur de personnages
  - Générateur de rencontres
  - Générateur de trésors
- **Outils pour MJ** : 
  - Calculateur d'équilibrage
  - Créateur de PNJ et monstres
  - Gestionnaire de campagnes

### Accès

- **URL** : <https://krosmoz-jdr.fr>
- **Statut** : En cours de développement
- **Mises à jour** : Le site sera mis à jour régulièrement avec le nouveau contenu

## 1.1.4.2. Fiches et outils

### Fiches de personnage

Deux formats de fiches de personnage sont disponibles :

- **Format PDF** : Fiche imprimable pour jouer en présentiel
- **Format PPTX** : Fiche modifiable (PowerPoint) pour personnalisation

Les fiches incluent :

- Toutes les caractéristiques principales et secondaires
- Espaces pour les sorts, aptitudes et capacités
- Section équipement et panoplies
- Espace pour l'historique et les notes

### Générateurs

Des générateurs sont disponibles ou en développement :

- **Générateur de personnages** : Création rapide de personnages de niveau 1
- **Générateur de rencontres** : Création de combats équilibrés
- **Générateur de trésors** : Distribution de butin selon le niveau

### Outils complémentaires

- **Calculateurs** : Outils pour calculer les modificateurs, les DD, etc.
- **Tables de référence** : Tables rapides pour le MJ
- **Aides de jeu** : Résumés des règles essentielles

## 1.1.4.3. Communauté

KrosmozJDR bénéficie d'une **communauté active** et ouverte.

### Discord

Le serveur Discord est le lieu principal d'échange :

- **Discussions** : Échanges sur les règles, les parties, les stratégies
- **Partage** : Partage de scénarios, de personnages, d'expériences de jeu
- **Annonces** : Nouvelles versions, mises à jour, événements

**Lien** : [#XVu4VWFskj](https://discord.gg/XVu4VWFskj)

### GitHub

Le projet est open source sur GitHub :

- **Code source** : Accès au code du site et des outils
- **Issues** : Signalement de bugs et suggestions d'amélioration
- **Contributions** : Possibilité de contribuer au développement
- **Versions** : Suivi des versions et changelog

**Lien** : [SamGoodwater/KrosmozJdr](https://github.com/SamGoodwater/KrosmozJdr)

### Contributions ouvertes

Le projet encourage les contributions :

- **Règles** : Suggestions d'amélioration des règles
- **Contenu** : Création de scénarios, créatures, équipements
- **Traduction** : Aide à la traduction dans d'autres langues
- **Tests** : Retours sur les parties de test

### Ancien projet

**DofusJDR** est l'ancien nom du projet, toujours accessible mais plus maintenu. KrosmozJDR est l'évolution et la continuation de ce projet.

**Site DofusJDR** : [jdr.iota21.fr](http://jdr.iota21.fr) (archivé)
