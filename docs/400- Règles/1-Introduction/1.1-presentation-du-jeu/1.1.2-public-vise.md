**Description** : Identifier les profils de joueurs (débutants, fans de Dofus, MJ expérimentés) et préciser les niveaux de difficulté.

## Contenu

- **1.1.2.1. Débutants** : Aides d'initiation, modules simplifiés.

## 1.1.2.1. Débutants

KrosmozJDR est conçu pour être **accessible aux débutants** en jeu de rôle. Si vous découvrez les jeux de rôle sur table, voici ce qui vous aidera :

### Aides d'initiation

- **Règles progressives** : Les concepts sont introduits étape par étape, des bases aux mécaniques avancées
- **Exemples concrets** : Chaque règle est illustrée par des exemples de jeu
- **Fiches de personnage claires** : Les fiches sont conçues pour être faciles à comprendre et à remplir
- **Glossaire** : Les termes techniques sont expliqués au fur et à mesure

### Modules simplifiés

Pour vos premières parties, vous pouvez :

- **Ignorer certaines règles** : Commencez avec les règles de base (combat, caractéristiques) et ajoutez les règles avancées progressivement
- **Utiliser des scénarios d'introduction** : Des scénarios pré-construits sont disponibles pour faciliter la prise en main
- **Simplifier les combats** : Réduire le nombre d'adversaires ou leur complexité pour se concentrer sur l'apprentissage

### Ressources pour débutants

- **Guide de création de personnage** : Section 2 détaillant chaque étape
- **Scénarios d'introduction** : Disponibles sur le site web et la communauté
- **Support communautaire** : Le Discord est là pour répondre à vos questions

## 1.1.2.2. Fans de Dofus

Si vous connaissez déjà **Dofus**, vous retrouverez de nombreux éléments familiers :

### Éléments familiers

- **Les 19 classes** : Toutes les classes du jeu vidéo sont présentes avec leurs mécaniques caractéristiques
- **Système PA/PM/PO** : Le combat tactique utilise le même système que Dofus
- **Les sorts** : Les sorts sont adaptés du jeu vidéo pour le format JDR
- **L'univers** : Le Monde des Douze, ses créatures, ses lieux et son histoire
- **Les équipements** : Panoplies, forgemagie, runes - tout le système d'équipement de Dofus

### Références dans l'univers

- **Lieux emblématiques** : Bonta, Brâkmar, Astrub, Amakna et bien d'autres
- **Créatures iconiques** : Bouftous, Wabbits, Chafeurs, et les maîtres de donjons
- **Métiers** : Tous les métiers de récolte et d'artisanat de Dofus
- **Histoire et lore** : L'histoire du Monde des Douze

### Adaptations pour le JDR

Certains éléments ont été adaptés pour fonctionner en jeu de rôle :

- **Progression** : Système de niveaux 1-20 au lieu des niveaux 1-200
- **Équilibrage** : Ajustements pour l'équilibre en JDR
- **Flexibilité narrative** : Plus de liberté pour créer des histoires originales

## 1.1.2.4. Niveau de difficulté

KrosmozJDR peut s'adapter à différents styles de jeu et niveaux de difficulté :

### Parties Casual

Pour des parties plus détendues et narratives :

- **Simplifier les combats** : Réduire la complexité tactique, se concentrer sur la narration
- **Ajuster les DD** : Réduire les difficultés pour favoriser la réussite
- **Privilégier le roleplay** : Plus de temps pour les interactions sociales et l'exploration
- **Règles optionnelles** : Ignorer certaines règles complexes si elles ralentissent le jeu

### Parties Tactiques

Pour des parties axées sur la stratégie et le combat :

- **Combats complexes** : Utiliser toutes les mécaniques (réactions, positionnement, synergies)
- **DD standards** : Respecter les difficultés recommandées
- **Ennemis intelligents** : Faire jouer les adversaires de manière stratégique
- **Ressources limitées** : Gérer strictement les PA, PM, PO et la réserve de Wakfu

### Équilibre recommandé

Pour la plupart des groupes, un équilibre entre :

- **40% Combat** : Combats tactiques et stratégiques
- **30% Exploration** : Découverte, investigation, pièges
- **30% Social/Roleplay** : Interactions, négociations, développement de personnage

Cet équilibre peut être ajusté selon les préférences du groupe et le style du MJ.
