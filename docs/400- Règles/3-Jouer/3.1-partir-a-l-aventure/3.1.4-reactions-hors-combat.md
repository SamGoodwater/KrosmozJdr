# 3.1.4. Réactions hors combat

**Description** : Gérer les réactions rapides en dehors des combats (surprises, embuscades, initiatives sociales, tests opposés) et les interactions dynamiques avec l'environnement.

> **Règles rapides - Réactions hors combat**
> - **Surprise** : Test opposé Discrétion/Furtivité vs Perception/Vigilance, créature surprise perd son tour initial
> - **Embuscades** : Conditions favorables (terrain, préparation), avantage pour l'attaquant
> - **Initiatives sociales** : Gestion des prises de parole, interruptions, escalades de conflits
> - **Tests opposés** : Résolution simultanée, compétences requises selon la situation
> - **Événements dynamiques** : Effondrements, chutes, réactions en chaîne nécessitant des décisions rapides

## Contenu

- **3.1.4.1. Surprises et embuscades** : Conditions, tests opposés, avantage/désavantage.
- **3.1.4.2. Initiatives sociales** : Gestion des prises de parole, interruptions, escalades de conflits.
- **3.1.4.3. Tests opposés** : Compétences requises, résolution simultanée.
- **3.1.4.4. Événements dynamiques** : Effondrements, chutes, réactions en chaîne nécessitant des décisions rapides.

## 3.1.4.1. Surprises et embuscades

Les surprises et embuscades peuvent survenir à tout moment, même hors combat.

### Conditions de surprise

Une créature est **surprise** si :
- Elle ne s'attend pas à une attaque ou à un danger
- Elle n'a pas détecté la menace
- Elle est distraite ou non préparée

**Effets** :
- La créature surprise est **incapable de se déplacer**
- Elle ne peut entreprendre ni **action** ni **réaction**
- Elle perd son tour initial

### Tests opposés

Pour déterminer si une créature est surprise, le MJ peut utiliser des **tests opposés** :

**Attaquant** :
- **Discrétion** : Test de Dextérité (Discrétion) pour se cacher
- **Furtivité** : Test de Dextérité (Furtivité) pour approcher sans être détecté

**Cible** :
- **Perception** : Test de Sagesse (Perception) pour remarquer la menace
- **Vigilance** : Test de Sagesse (Perception) passive pour détecter les dangers

**Résolution** :
- Si l'attaquant réussit son test et que la cible échoue, la cible est surprise
- Si la cible réussit son test, elle n'est pas surprise et peut réagir

### Avantage et désavantage

Certaines situations peuvent donner un **avantage** ou un **désavantage** :

**Avantage pour l'attaquant** :
- **Ténèbres** : Attaquer depuis les ténèbres
- **Distraction** : La cible est distraite
- **Position** : Position élevée ou cachée

**Désavantage pour l'attaquant** :
- **Lumière vive** : Attaquer en pleine lumière
- **Terrain découvert** : Pas de couverture
- **Bruit** : Faire du bruit en approchant

**Avantage pour la cible** :
- **Vigilance** : La cible est en alerte
- **Compagnons** : Des alliés peuvent alerter
- **Sorts de détection** : Sorts actifs de détection

**Désavantage pour la cible** :
- **Distraction** : La cible est occupée
- **Fatigue** : La cible est épuisée
- **Conditions** : Conditions défavorables (pluie, vent, etc.)

## 3.1.4.2. Initiatives sociales

Les interactions sociales peuvent nécessiter une gestion du temps et de l'ordre des actions.

### Gestion des prises de parole

Dans des situations sociales complexes, le MJ peut utiliser un système d'**initiative sociale** :

**Détermination** :
- **Initiative** : Test de Chance (Persuasion) ou Chance (Représentation)
- **Ordre** : Les personnages agissent dans l'ordre de leur initiative
- **Flexibilité** : Le MJ peut permettre des interruptions

### Interruptions

Les personnages peuvent **interrompre** les autres :

**Conditions** :
- **Réaction** : Utiliser une réaction pour interrompre
- **Compétence** : Test de Chance (Persuasion) ou Chance (Intimidation)
- **DD** : DD basé sur l'initiative de la cible

**Effets** :
- L'interruption peut changer le cours de la conversation
- Peut donner un avantage ou un désavantage aux tests suivants
- Peut créer de la tension ou de la comédie

### Escalades de conflits

Les interactions sociales peuvent **escalader** en conflits :

**Niveaux d'escalade** :
1. **Discussion** : Échange normal
2. **Débat** : Désaccord mais respectueux
3. **Confrontation** : Tension élevée, menaces implicites
4. **Conflit** : Menaces explicites, violence imminente
5. **Combat** : Passage au combat

**Gestion** :
- Le MJ suit le niveau d'escalade
- Certaines actions peuvent faire monter ou descendre le niveau
- Le passage au combat suit les règles normales de combat

## 3.1.4.3. Tests opposés

Les tests opposés sont utilisés quand deux créatures s'opposent directement.

### Compétences requises

Les tests opposés utilisent généralement :
- **Persuasion vs Perspicacité** : Convaincre quelqu'un
- **Intimidation vs Intimidation** : Intimider quelqu'un
- **Escamotage vs Perception** : Voler ou tricher
- **Discrétion vs Perception** : Se cacher
- **Athlétisme vs Athlétisme** : Lutte, poussée
- **Acrobaties vs Acrobaties** : Éviter une saisie

### Résolution simultanée

Les tests opposés sont résolus **simultanément** :
- Les deux créatures lancent leurs dés en même temps
- Le résultat le plus élevé gagne
- En cas d'égalité, la situation reste inchangée ou un nouveau test est nécessaire

**Exemple** :
- **Joueur A** : Test de Chance (Persuasion) = 18
- **Joueur B** : Test de Sagesse (Perspicacité) = 15
- **Résultat** : Joueur A réussit à convaincre Joueur B

### Modificateurs

Les tests opposés peuvent être modifiés par :
- **Avantage/Désavantage** : Selon les circonstances
- **Bonus de situation** : Selon le contexte
- **Compétences** : Maîtrise et expertise

## 3.1.4.4. Événements dynamiques

Certains événements nécessitent des réactions rapides et des décisions immédiates.

### Effondrements

Un effondrement peut survenir soudainement :
- **Détection** : Test de Sagesse (Perception) pour remarquer les signes
- **Réaction** : Test de Dextérité (Acrobaties) pour éviter
- **Conséquences** : Dégâts, immobilisation, piégage

**Gestion** :
- Le MJ peut donner un avertissement (test de Perception)
- Les personnages peuvent réagir avec des tests appropriés
- Les conséquences dépendent de la réussite ou de l'échec

### Chutes

Les chutes peuvent survenir de manière inattendue :
- **Détection** : Test de Sagesse (Perception) pour remarquer le danger
- **Réaction** : Test de Force (Athlétisme) pour s'accrocher
- **Conséquences** : Dégâts selon la hauteur (1d6 par tranche de 3 mètres, maximum 20d6)

**Gestion** :
- Le MJ détermine la hauteur de la chute
- Les personnages peuvent tenter de s'accrocher ou d'amortir la chute
- Les dégâts sont calculés selon les règles standard

### Réactions en chaîne

Certains événements peuvent déclencher des **réactions en chaîne** :

**Exemples** :
- **Déclencher un piège** → Alerte les gardes → Combat
- **Faire du bruit** → Attire des créatures → Confrontation
- **Casser quelque chose** → Effondrement → Piégage

**Gestion** :
- Le MJ suit la chaîne d'événements
- Les personnages peuvent réagir à chaque étape
- Les conséquences s'accumulent ou se résolvent selon les actions

### Décisions rapides

Dans des situations dynamiques, les personnages doivent prendre des **décisions rapides** :

**Gestion** :
- Le MJ peut limiter le temps de réflexion
- Les joueurs doivent décider rapidement
- Les conséquences reflètent la qualité de la décision

**Exemples** :
- **Pont qui s'effondre** : Décider rapidement de sauter ou de reculer
- **Porte qui se ferme** : Décider rapidement de passer ou de rester
- **Explosion imminente** : Décider rapidement de la meilleure action

## Conseils pour le MJ

### Créer de la tension

Utilisez les réactions hors combat pour :
- **Créer de la tension** : Surprises, embuscades, événements soudains
- **Maintenir l'engagement** : Garder les joueurs attentifs
- **Enrichir la narration** : Ajouter de la profondeur aux interactions

### Équilibrage

Assurez-vous que :
- **Les défis sont équitables** : Donnez des chances de réussir
- **Les conséquences sont appropriées** : Pas trop sévères, pas trop légères
- **Les récompenses sont justes** : Récompensez la prudence et la créativité

### Flexibilité

Soyez flexible :
- **Adaptez les règles** : Selon le contexte et le style de jeu
- **Permettez la créativité** : Laissez les joueurs trouver des solutions originales
- **Ajustez la difficulté** : Selon les capacités du groupe
