## 3.4.4.1. Avec les classes

Les aptitudes et capacités peuvent créer des synergies puissantes avec certaines classes.

### Bonus ciblés

Certaines aptitudes sont **ciblées** pour renforcer le rôle d'une classe :
- **Tank** : Aptitudes de protection, de défense
- **DPS** : Aptitudes de dégâts, d'offense
- **Support** : Aptitudes de soin, de buff
- **Utilité** : Aptitudes d'exploration, de résolution de problèmes

**Exemples** :
- **Milicien + Iop** : Formation tactique renforce le rôle de DPS
- **Dévot + Eniripsa** : Bénédiction renforce le rôle de soigneur
- **Voleur + Sram** : Furtivité renforce le rôle de roublard

### Aptitudes renforçant les rôles

Les aptitudes peuvent **renforcer** le rôle principal d'une classe :
- **Complément** : Ajoutent des capacités manquantes
- **Amélioration** : Améliorent les capacités existantes
- **Flexibilité** : Ajoutent de la flexibilité au rôle

**Exemples** :
- **Érudit + Xélor** : Analyse renforce le rôle de mage
- **Artisan + Forgelance** : Fabrication renforce le rôle de forgeron
- **Explorateur + Enutrof** : Détection de trésors renforce le rôle de chasseur de trésors

---

## 3.4.4.2. Avec les spécialisations

Les aptitudes et capacités s'harmonisent naturellement avec leur spécialisation d'origine.

### Combinaisons recommandées

Certaines combinaisons sont particulièrement efficaces :
- **Spécialisation + Classe correspondante** : Synergie naturelle
- **Spécialisation + Classe complémentaire** : Crée des builds intéressants
- **Spécialisation + Classe opposée** : Crée des builds uniques et surprenants

**Exemples** :
- **Érudit + Xélor** : Mage intellectuel avec aptitudes de recherche
- **Milicien + Iop** : Guerrier discipliné avec aptitudes tactiques
- **Artiste + Eniripsa** : Soigneur charismatique avec aptitudes d'inspiration

### Redondances à éviter

Certaines combinaisons peuvent créer des **redondances** :
- **Aptitudes similaires** : Si votre classe a déjà une capacité similaire
- **Effets identiques** : Si l'aptitude fait la même chose qu'un sort de classe
- **Sur-spécialisation** : Trop se concentrer sur un seul aspect

**Conseil** : Variez vos aptitudes pour avoir une palette complète d'options.

---

## 3.4.4.3. Avec les équipements

Certains équipements peuvent améliorer les aptitudes et capacités.

### Objets améliorant les aptitudes

Certains équipements peuvent :
- **Réduire les coûts** : Réduisent le coût en PA ou réserve de Wakfu
- **Augmenter les effets** : Augmentent la puissance des aptitudes
- **Ajouter des effets** : Ajoutent des effets supplémentaires

**Exemples** :
- **Outils spécialisés** : Outils qui améliorent les aptitudes d'artisanat
- **Livres** : Livres qui améliorent les aptitudes d'érudit
- **Instruments** : Instruments qui améliorent les aptitudes d'artiste

### Objets améliorant les capacités

Certains équipements peuvent :
- **Renforcer les bonus** : Augmentent les bonus des capacités passives
- **Étendre les conditions** : Étendent les conditions d'activation
- **Ajouter des capacités** : Confèrent de nouvelles capacités

**Exemples** :
- **Amulettes** : Amulettes qui renforcent les capacités
- **Anneaux** : Anneaux qui étendent les capacités
- **Artefacts** : Artefacts qui confèrent des capacités spéciales

## 3.4.4.4. Prévention des abus

Il est important d'éviter les abus et les déséquilibres.

### Règles anti-stack

**Même type de bonus** : Les bonus du même type ne se cumulent généralement pas :
- **Meilleur bonus** : On prend le meilleur bonus
- **Exceptions** : Si explicitement indiqué, certains bonus peuvent se cumuler

**Exemple** : Si vous avez +2 Force d'une capacité et +1 Force d'un équipement, vous avez +2 Force (pas +3).

### Limites de puissance

**Plafonds** : Certaines capacités ont des **plafonds** :
- **Maximum de bonus** : Limite au bonus maximum possible
- **Limites par niveau** : Certaines capacités sont limitées par niveau
- **Limites par source** : Certaines capacités ne peuvent pas être combinées avec certaines sources

### Arbitrage MJ

Le MJ a le dernier mot sur :
- **Cumul** : Si des bonus se cumulent ou non
- **Équilibrage** : Si une combinaison est trop puissante
- **Exceptions** : Si des exceptions sont justifiées

**Conseil** : Discutez avec votre MJ si vous avez des questions sur les interactions.

## Exemples de synergies

### Synergie 1 : Érudit + Xélor

**Aptitudes** : Analyse, Recherche approfondie, Décryptage

**Synergie** :
- **Analyse** : Permet d'analyser les sorts ennemis avant de les contrer
- **Recherche approfondie** : Permet de trouver des informations sur les sorts
- **Décryptage** : Permet de comprendre les sorts complexes

**Résultat** : Mage très compétent en analyse et contre-sorts.

### Synergie 2 : Milicien + Iop

**Aptitudes** : Formation tactique, Commandement, Discipline

**Synergie** :
- **Formation tactique** : Donne des bonus tactiques aux alliés
- **Commandement** : Inspire et dirige en combat
- **Discipline** : Résiste aux effets mentaux

**Résultat** : Guerrier leader très efficace en combat.

### Synergie 3 : Artiste + Eniripsa

**Aptitudes** : Performance, Inspiration, Création artistique

**Synergie** :
- **Performance** : Donne des performances qui inspirent
- **Inspiration** : Inspire les alliés avec l'art
- **Création artistique** : Crée des œuvres qui soignent

**Résultat** : Soigneur charismatique qui inspire et soigne.

---

## Conseils pour éviter les abus

### Pour les joueurs

- **Variez** : Ne choisissez pas que des aptitudes/capacités de combat
- **Équilibrez** : Équilibrez entre aptitudes actives et capacités passives
- **Communiquez** : Discutez avec le MJ de vos choix

### Pour le MJ

- **Surveillez** : Surveillez les combinaisons trop puissantes
- **Ajustez** : Ajustez les règles si nécessaire
- **Arbitrez** : Arbitrez les conflits de règles avec équité
