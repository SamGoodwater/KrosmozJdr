# 3.3.3. Apprentissage et réapprentissage

**Description** : Expliquer comment apprendre un sort (parchemins, maître, temple), combien de temps cela prend, et comment permuter les sorts connus.

> **Règles rapides**
> - Temps d'étude : `2 jours × niveau du sort` (÷4 avec un maître / environnement adapté)
> - Parchemin consommé une fois le sort appris
> - Limite de sorts connus : oubliez un sort avant d'en apprendre un nouveau si vous êtes à la limite
> - Réapprentissage / changement de variante : nécessite un rituel, du temps et parfois un coût

## Contenu

- **3.3.3.1. Sources d'apprentissage** : Parchemins, mentors, temples, artefacts.
- **3.3.3.2. Temps et coûts** : Durée d'étude, frais matériels, tests requis.
- **3.3.3.3. Réapprentissage** : Oublier un sort, réassigner des variantes.

## 3.3.3.1. Sources d'apprentissage

Il existe plusieurs façons d'apprendre un sort apprenable.

### Parchemins de sort

La méthode la plus courante est d'obtenir un **parchemin de sort**.

**Contenu** : Sur ce parchemin, il y a un descriptif de comment utiliser et maîtriser ce sort.

**Processus** : Le joueur/joueuse doit alors s'imprégner du wakfu émanant du parchemin et comprendre le fonctionnement du sort.

**Consommation** : Une fois le sort appris, le parchemin a perdu son wakfu et n'est donc plus utilisable.

### Mentors et maîtres

Un **maître** connaissant déjà le sort ou étant compétent en enseignement peut accélérer l'apprentissage.

**Avantage** : Le temps d'apprentissage est **divisé par 4** avec un maître.

**Conditions** : Le maître doit :
- Connaître déjà le sort, OU
- Être compétent en enseignement

### Temples et lieux sacrés

Certains temples ou lieux sacrés peuvent offrir des opportunités d'apprentissage :
- **Bibliothèques** : Collections de parchemins
- **Salles d'étude** : Environnements propices à l'apprentissage
- **Maîtres résidents** : Maîtres disponibles pour enseigner

### Artefacts et objets magiques

Certains artefacts ou objets magiques peuvent contenir la connaissance d'un sort :
- **Livres magiques** : Tomes contenant des sorts
- **Artefacts** : Objets imprégnés de magie
- **Reliques** : Objets sacrés avec des connaissances anciennes

## 3.3.3.2. Temps et coûts

### Temps d'apprentissage

Le temps d'apprentissage dépend du **niveau du sort** :
- **Base** : Pour maîtriser les subtilités du sort, il faudra un nombre de jours égal à **2 fois le niveau du sort**

**Exemples** :
- Sort niveau 1 : **2 jours**
- Sort niveau 5 : **10 jours**
- Sort niveau 10 : **20 jours**

### Avec un maître

À l'aide d'un maître connaissant déjà le sort ou étant compétent en enseignement, ce temps sera **divisé par 4**.

**Exemples** :
- Sort niveau 1 : **0,5 jour** (arrondi à 1 jour)
- Sort niveau 5 : **2,5 jours** (arrondi à 3 jours)
- Sort niveau 10 : **5 jours**

| Niveau du sort | Temps standard | Avec maître / lieu adapté |
|---------------:|:--------------:|:-------------------------:|
| 1              | 2 jours        | 0,5 jour (arrondi à 1)    |
| 5              | 10 jours       | 2,5 jours (arrondi à 3)   |
| 10             | 20 jours       | 5 jours                   |
| 15             | 30 jours       | 7,5 jours (arrondi à 8)   |
| 20             | 40 jours       | 10 jours                  |

### Frais matériels

L'apprentissage peut nécessiter :
- **Achat du parchemin** : Coût variable selon la rareté du sort
- **Frais d'enseignement** : Si vous utilisez un maître, il peut demander une rémunération
- **Matériaux** : Certains sorts peuvent nécessiter des matériaux pour l'apprentissage
- **Location de salle** : Si vous utilisez une salle d'étude

> **Exemple – Budget d'apprentissage**
> - Parchemin de *Nova aqueuse* (niveau 6) : 1 500 kamas.
> - Honoraires du maître : 300 kamas / jour (3 jours avec maître) = 900 kamas.
> - Matériel (encens, composants) : 200 kamas.
> - **Total** : 2 600 kamas pour apprendre le sort en 3 jours.

### Tests requis

Certains sorts peuvent nécessiter des **tests** pour être appris :
- **Intelligence (Arcanes)** : Comprendre la théorie magique
- **Sagesse (Religion)** : Pour les sorts divins
- **Intelligence (Investigation)** : Analyser le parchemin

**DD** : Le degré de difficulté dépend du niveau et de la complexité du sort.

> **Conseil MJ** : Utilisez `DD = 10 + niveau du sort ÷ 2` pour estimer rapidement la difficulté d'apprentissage.

## 3.3.3.3. Réapprentissage

### Oublier un sort

Dans certaines situations, vous pouvez vouloir **oublier un sort** pour en apprendre un autre :
- **Limite de sorts** : Vous avez atteint votre limite de sorts connus
- **Changement de stratégie** : Vous voulez changer votre approche
- **Nouveau sort préféré** : Vous préférez un autre sort

**Processus** : Le processus d'oubli peut nécessiter :
- **Temps** : Méditation, rituel, etc.
- **Coûts** : Potions, rituels, etc.
- **Tests** : Tests de caractéristiques appropriées

### Réassigner des variantes

Certains sorts ont des **variantes** (voir section 3.3.5). Vous pouvez changer de variante :
- **Temple** : Certains temples permettent de changer de variante
- **Méditation** : Temps de méditation et de réflexion
- **Coût** : Peut nécessiter des ressources ou des frais

**Limitations** : Le changement de variante peut être limité en fréquence (une fois par niveau, une fois par mois, etc.).

> **Exemple – Réaffecter un sort**
> - **Situation** : L'Huppermage a atteint sa limite de 12 sorts connus.
> - **Choix** : Il souhaite oublier *Bouclier prismatique* pour apprendre *Porte astrale*.
> - **Processus** : 1 jour de méditation + rituel coûtant 200 kamas.
> - **Résultat** : *Bouclier prismatique* est oublié, *Porte astrale* devient disponible immédiatement.

## 3.3.3.4. Assistance

### Maître

Un **maître** peut fournir plusieurs types d'assistance :

**Réduction du temps** : Le temps d'apprentissage est divisé par 4.

**Guidance** : Le maître peut :
- Expliquer les subtilités du sort
- Corriger les erreurs
- Fournir des conseils

**Tests** : Le maître peut donner un **avantage** sur les tests d'apprentissage.

### Spécialisation

Certaines **spécialisations** peuvent aider à l'apprentissage :
- **Érudit** : Bonus à l'apprentissage des sorts
- **Dévot** : Bonus pour les sorts divins
- **Autres** : Selon les capacités de la spécialisation

### Environnement

Certains **environnements** peuvent faciliter l'apprentissage :
- **Bibliothèques** : Bonus à l'apprentissage
- **Temples** : Bonus pour les sorts divins
- **Laboratoires** : Bonus pour les sorts magiques
- **Lieux sacrés** : Bonus selon le type de sort

---

## Conseils pour les joueurs

### Planifier l'apprentissage

- **Priorités** : Choisissez les sorts les plus utiles en premier
- **Temps** : Planifiez vos périodes d'apprentissage entre les aventures
- **Ressources** : Assurez-vous d'avoir les ressources nécessaires

### Utiliser les maîtres

- **Recherchez des maîtres** : Ils accélèrent considérablement l'apprentissage
- **Négociez** : Les frais peuvent être négociables
- **Relations** : Construisez des relations avec des maîtres

### Gérer les limites

- **Limite de sorts** : Vous ne pouvez pas apprendre un nombre illimité de sorts
- **Choix stratégiques** : Choisissez vos sorts avec soin
- **Réapprentissage** : N'hésitez pas à oublier des sorts moins utiles

---

### À retenir

- Temps d'étude = 2 jours × niveau (÷4 avec mentor ou lieu adapté).
- Les parchemins sont consommés une fois le sort appris : anticipez vos achats.
- Tenez un registre des sorts connus pour savoir lesquels sacrifier en cas de limite atteinte.
- Profitez des temples, bibliothèques et maîtres pour réduire temps, coût et DD.
