# 3.3.5. Variantes et personnalisation

**Description** : Décrire le système de variantes de sort, les restrictions de changement, et comment personnaliser ses sorts dans une classe donnée.

> **Règles rapides - Variantes et personnalisation**
> - **Choix initial** : Variantes disponibles au déblocage d'un sort (dégâts, effets, portée, coût)
> - **Changement** : Possible dans un temple de classe (2 heures par sort), coût possible
> - **Personnalisation** : Ajustements autorisés selon le MJ, limites d'équilibrage
> - **Variantes** : Chaque sort a généralement 2 variantes, choix unique au déblocage

## Contenu

- **3.3.5.1. Choix initial** : Variantes disponibles au déblocage d'un sort.
- **3.3.5.2. Changement de variante** : Conditions (temple, méditation, coût).
- **3.3.5.3. Personnalisation avancée** : Ajustements autorisés, limites d'équilibrage.
- **3.3.5.4. Exemples de variantes** : Cas concrets par classe.

## 3.3.5.1. Choix initial

Quand vous débloquez un sort, vous pouvez avoir le choix entre plusieurs variantes.

### Variantes disponibles

Certains sorts ont plusieurs **variantes** qui modifient leurs effets :
- **Variantes de dégâts** : Différents types ou quantités de dégâts
- **Variantes d'effets** : Différents effets secondaires
- **Variantes de portée** : Différentes portées ou zones
- **Variantes de coût** : Différents coûts en PA

### Moment du choix

Le choix de la variante se fait généralement :
- **Au déblocage** : Quand vous apprenez le sort pour la première fois
- **Lors de l'apprentissage** : Si vous apprenez un sort apprenable

### Limitations

**Choix unique** : Vous choisissez généralement une variante et vous vous y tenez.

**Changement possible** : Vous pouvez changer de variante, mais cela nécessite certaines conditions (voir ci-dessous).

---

## 3.3.5.2. Changement de variante

Vous pouvez changer la variante d'un sort que vous connaissez déjà, mais cela nécessite certaines conditions.

### Conditions

**Temple** : Certains temples permettent de changer de variante :
- **Temples de classe** : Temples dédiés à votre classe
- **Temples génériques** : Temples magiques généraux
- **Coût** : Peut nécessiter une donation ou un paiement

**Méditation** : Vous pouvez changer de variante par méditation :
- **Temps** : Nécessite du temps de méditation et de réflexion
- **Environnement** : Peut nécessiter un environnement calme et propice
- **Tests** : Peut nécessiter des tests de caractéristiques

**Coût** : Le changement peut avoir un coût :
- **Ressources** : Matériaux, potions, etc.
- **Temps** : Plusieurs jours de méditation
- **Argent** : Frais pour utiliser un temple ou un service

### Limitations de fréquence

Le changement de variante peut être limité :
- **Une fois par niveau** : Vous ne pouvez changer qu'une fois par niveau
- **Une fois par mois** : Limitation temporelle
- **Coût croissant** : Le coût augmente à chaque changement

---

## 3.3.5.3. Personnalisation avancée

Au-delà des variantes officielles, certains ajustements peuvent être autorisés.

### Ajustements autorisés

**Avec l'accord du MJ** : Certains ajustements mineurs peuvent être autorisés :
- **Flavor** : Changements esthétiques ou narratifs sans impact mécanique
- **Petits ajustements** : Modifications mineures qui ne déséquilibrent pas le jeu
- **Thématique** : Ajustements pour mieux correspondre à votre personnage

### Limites d'équilibrage

**Pas de déséquilibrage** : Les ajustements ne doivent pas :
- **Augmenter la puissance** : Sans compensation
- **Réduire les coûts** : Sans justification
- **Changer les mécaniques fondamentales** : Sans accord explicite

**Équilibrage** : Tous les ajustements doivent être équilibrés :
- **Compensations** : Si vous augmentez quelque chose, réduisez autre chose
- **Validation du MJ** : Le MJ doit valider tous les ajustements
- **Tests** : Les ajustements peuvent nécessiter des tests en jeu

## 3.3.5.4. Exemples de variantes

Voici des exemples de variantes pour différentes classes.

### Iop - Attaque Naturelle

**Variante 1 - Flamme** :
- Dégâts Feu
- Portée 1-4 PO
- Coût 3 PA

**Variante 2 - Éclair** :
- Dégâts Air
- Portée 1-6 PO
- Coût 4 PA

**Variante 3 - Explosion** :
- Dégâts Feu en zone
- Portée 1-3 PO
- Coût 5 PA

### Féca - Bouclier Féca

**Variante 1 - Protection** :
- Points de bouclier : 1d4 + Vitalité
- Portée 0-3 PO
- Coût 3 PA

**Variante 2 - Bouclier renforcé** :
- Points de bouclier : 1d6 + Vitalité
- Portée 0-2 PO
- Coût 4 PA

**Variante 3 - Bouclier partagé** :
- Points de bouclier : 1d4 + Vitalité
- Portée 0-5 PO (plusieurs cibles)
- Coût 5 PA

### Eniripsa - Soin

**Variante 1 - Soin rapide** :
- Soin : 1d6 + Sagesse
- Portée 1-4 PO
- Coût 3 PA

**Variante 2 - Soin puissant** :
- Soin : 2d4 + Sagesse
- Portée 1-3 PO
- Coût 5 PA

**Variante 3 - Soin de zone** :
- Soin : 1d4 + Sagesse
- Portée 0-2 PO (zone)
- Coût 4 PA

## Conseils pour les joueurs

### Choisir une variante

- **Style de jeu** : Choisissez une variante qui correspond à votre style de jeu
- **Rôle dans l'équipe** : Considérez votre rôle dans l'équipe
- **Synergies** : Pensez aux synergies avec vos autres sorts et capacités

### Changer de variante

- **Planifiez** : Ne changez pas trop souvent, planifiez vos changements
- **Coûts** : Tenez compte des coûts et limitations
- **Moment** : Choisissez le bon moment pour changer (entre les aventures)

### Personnalisation

- **Discutez avec le MJ** : Parlez de vos idées de personnalisation avec le MJ
- **Équilibrage** : Assurez-vous que vos ajustements sont équilibrés
- **Cohérence** : Gardez une cohérence avec votre personnage et l'univers
