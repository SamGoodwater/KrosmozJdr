# 3.5.4. Modificateurs et difficultés

**Description** : Décrire les DC (classes de difficulté), les modificateurs liés aux circonstances, outils, assistance ou obstacles.

> **Règles rapides - Modificateurs et difficultés**
> - **DC** : 5 (Très facile), 10 (Facile), 15 (Moyen), 20 (Difficile), 25 (Très difficile), 30 (Quasi impossible)
> - **Aide** : Un allié peut donner un avantage (2d20, meilleur résultat)
> - **Désavantage** : 2d20, moins bon résultat (conditions défavorables)
> - **Outils** : Bonus fournis selon l'outil, prérequis possibles
> - **Obstacles** : Fatigue, blessures, conditions défavorables augmentent la difficulté

## Contenu

- **3.5.4.1. Modificateurs contextuels** : Aide, désavantage, environnement.
- **3.5.4.2. Outils et équipements** : Bonus fournis, prérequis.
- **3.5.4.3. Obstacles** : Conditions qui augmentent la difficulté (fatigue, blessures).
- **3.5.4.4. Ajustements MJ** : Comment évaluer rapidement une DC.

---

## 3.5.4.1. Modificateurs contextuels

Les modificateurs contextuels représentent les circonstances qui affectent vos tests de compétence.

### Aide

L'**aide** d'un autre personnage peut améliorer vos chances de réussite.

**Règle de base** :
- **Un allié peut vous aider** : Un allié peut vous donner un avantage sur votre test
- **Condition** : L'allié doit être capable d'aider (avoir la compétence ou être capable d'aider)
- **Action** : L'allié doit utiliser son action pour vous aider

**Avantage** : Quand vous recevez de l'aide, vous lancez **2d20** et prenez le **meilleur résultat**.

**Limitations** :
- **Un seul allié** : Généralement, un seul allié peut vous aider à la fois
- **Capacité requise** : L'allié doit être capable d'aider (avoir la compétence ou être capable d'aider)
- **Distance** : L'allié doit être à proximité

**Exemples** :
- Un allié vous aide à escalader une paroi en vous tenant une corde
- Un allié vous aide à crocheter une serrure en vous éclairant
- Un allié vous aide à persuader quelqu'un en appuyant vos arguments

### Désavantage

Le **désavantage** représente des circonstances défavorables.

**Règle de base** :
- **Vous lancez 2d20** et prenez le **moins bon résultat**
- **Causes** : Conditions défavorables, obstacles, malus

**Causes courantes** :
- **Conditions défavorables** : Pluie, obscurité, bruit
- **Obstacles** : Surface glissante, espace étroit
- **Malus** : Blessures, fatigue, états négatifs

**Exemples** :
- Escalader une paroi sous la pluie (désavantage)
- Se faufiler dans un espace étroit (désavantage)
- Persuader quelqu'un quand vous êtes blessé (désavantage)

### Avantage

L'**avantage** représente des circonstances favorables.

**Règle de base** :
- **Vous lancez 2d20** et prenez le **meilleur résultat**
- **Causes** : Conditions favorables, outils, assistance

**Causes courantes** :
- **Conditions favorables** : Bonne visibilité, surface stable
- **Outils** : Avoir les bons outils
- **Assistance** : Aide d'un allié
- **Préparation** : Avoir préparé l'action

**Exemples** :
- Escalader une paroi avec des prises solides (avantage)
- Crocheter une serrure avec les bons outils (avantage)
- Persuader quelqu'un avec de bons arguments préparés (avantage)

### Environnement

L'**environnement** peut affecter vos tests de compétence.

**Facteurs environnementaux** :
- **Visibilité** : Obscurité, brouillard, lumière
- **Conditions météorologiques** : Pluie, vent, neige
- **Terrain** : Surface glissante, instable, difficile
- **Bruit** : Bruit ambiant, silence

**Exemples** :
- **Discrétion** : Obscurité donne avantage, lumière vive donne désavantage
- **Athlétisme** : Surface glissante donne désavantage, surface stable donne avantage
- **Perception** : Bruit ambiant donne désavantage, silence donne avantage

## 3.5.4.2. Outils et équipements

Les outils et équipements peuvent améliorer vos tests de compétence.

### Bonus fournis

Les outils et équipements peuvent :
- **Donner des bonus** : +1, +2, +3 aux tests
- **Donner la maîtrise** : Conférer la maîtrise d'une compétence
- **Donner l'avantage** : Conférer l'avantage sur les tests
- **Réduire la difficulté** : Réduire le DD du test

**Exemples** :
- **Outils de crochetage** : +2 aux tests d'Escamotage pour crocheter
- **Corde et grappin** : +2 aux tests d'Athlétisme pour escalader
- **Livre de connaissances** : +1 aux tests de connaissances spécifiques

### Prérequis

Certains outils et équipements ont des **prérequis** :
- **Maîtrise requise** : Vous devez maîtriser la compétence pour utiliser l'outil
- **Caractéristique minimale** : Vous devez avoir une caractéristique minimale
- **Formation** : Vous devez être formé à l'utilisation de l'outil

**Exemples** :
- **Outils de forge** : Nécessitent la maîtrise de la compétence Forge
- **Instruments de musique** : Nécessitent la maîtrise de la compétence Représentation
- **Outils d'alchimie** : Nécessitent la maîtrise de la compétence Alchimie

### Outils spécialisés

Certains outils sont **spécialisés** pour des compétences spécifiques :
- **Outils d'artisanat** : Pour les compétences d'artisanat
- **Outils d'exploration** : Pour les compétences d'exploration
- **Outils sociaux** : Pour les compétences sociales

**Exemples** :
- **Outils de forge** : Pour la compétence Forge
- **Boussole et cartes** : Pour la compétence Survie
- **Instruments de musique** : Pour la compétence Représentation

## 3.5.4.3. Obstacles

Les obstacles représentent les conditions qui augmentent la difficulté de vos tests.

### Fatigue

La **fatigue** peut affecter vos tests :
- **Fatigue légère** : -1 aux tests
- **Fatigue modérée** : -2 aux tests ou désavantage
- **Fatigue sévère** : -3 aux tests ou désavantage

**Causes** :
- **Manque de repos** : Ne pas avoir assez reposé
- **Effort prolongé** : Avoir fait beaucoup d'efforts
- **Conditions difficiles** : Conditions environnementales difficiles

### Blessures

Les **blessures** peuvent affecter vos tests :
- **Blessures légères** : -1 aux tests physiques
- **Blessures modérées** : -2 aux tests physiques ou désavantage
- **Blessures graves** : -3 aux tests physiques ou désavantage

**Types de blessures** :
- **Blessures physiques** : Affectent les compétences physiques
- **Blessures mentales** : Affectent les compétences mentales
- **Blessures générales** : Affectent toutes les compétences

### États négatifs

Les **états négatifs** peuvent affecter vos tests :
- **Empoisonnement** : -2 aux tests ou désavantage
- **Maladie** : -1 à -3 aux tests selon la gravité
- **Peur** : Désavantage aux tests sociaux
- **Confusion** : Désavantage aux tests mentaux

**Note** : Pour plus de détails, consultez la section **3.2.5. Traits et états**.

### Conditions défavorables

Les **conditions défavorables** peuvent affecter vos tests :
- **Distraction** : Désavantage aux tests nécessitant de la concentration
- **Pression** : Désavantage aux tests sociaux sous pression
- **Urgence** : Augmente le DD ou donne désavantage

## 3.5.4.4. Ajustements MJ

Le MJ doit évaluer rapidement la difficulté d'une action pour déterminer le DD approprié.

### Évaluation rapide

**Méthode simple** :
1. **Évaluez la complexité** : L'action est-elle simple, normale, difficile, très difficile ?
2. **Considérez les circonstances** : Y a-t-il des facteurs favorables ou défavorables ?
3. **Ajustez le DD** : Ajustez le DD selon les circonstances

**Barème de base** :
- **Action simple** : DD 10
- **Action normale** : DD 15
- **Action difficile** : DD 20
- **Action très difficile** : DD 25

### Facteurs à considérer

**Facteurs favorables** :
- **Outils appropriés** : Réduisez le DD de 2-3
- **Aide d'un allié** : Réduisez le DD de 2-3 ou donnez avantage
- **Conditions favorables** : Réduisez le DD de 1-2
- **Préparation** : Réduisez le DD de 1-2

**Facteurs défavorables** :
- **Manque d'outils** : Augmentez le DD de 2-3
- **Conditions défavorables** : Augmentez le DD de 1-2 ou donnez désavantage
- **Urgence** : Augmentez le DD de 1-2
- **Obstacles** : Augmentez le DD de 2-3

### Ajustement selon le niveau

**Personnages de bas niveau (1-5)** :
- **DD plus bas** : Utilisez des DD de 10-15 pour les actions normales
- **DD plus élevé** : Utilisez des DD de 20-25 pour les actions très difficiles

**Personnages de niveau moyen (6-10)** :
- **DD normal** : Utilisez des DD de 15-20 pour les actions normales
- **DD plus élevé** : Utilisez des DD de 25-30 pour les actions très difficiles

**Personnages de haut niveau (11-20)** :
- **DD plus élevé** : Utilisez des DD de 20-25 pour les actions normales
- **DD très élevé** : Utilisez des DD de 30+ pour les actions très difficiles

### Conseils pratiques

**Ne pas hésiter à ajuster** :
- **Selon le contexte** : Ajustez le DD selon le contexte narratif
- **Selon les joueurs** : Ajustez le DD selon les capacités des joueurs
- **Selon l'importance** : Ajustez le DD selon l'importance de l'action

**Communiquer** :
- **Expliquez le DD** : Expliquez pourquoi le DD est ce qu'il est
- **Donnez des indices** : Donnez des indices sur la difficulté avant le test
- **Soyez cohérent** : Soyez cohérent dans vos évaluations

## Exemples pratiques

### Exemple 1 : Escalade avec outils

**Scénario** : Un personnage tente d'escalader une paroi rocheuse avec une corde et un grappin.

**Évaluation** :
- **Action de base** : Escalader une paroi (DD 15)
- **Outils appropriés** : Corde et grappin (-2 DD)
- **DD final** : 13

**Modificateurs** :
- **Bonus d'outils** : +2 aux tests d'Athlétisme
- **Avantage** : Si la corde est bien fixée

### Exemple 2 : Persuasion sous pression

**Scénario** : Un personnage tente de persuader un garde de le laisser passer, mais il est pressé et le garde est méfiant.

**Évaluation** :
- **Action de base** : Persuader un garde (DD 15)
- **Urgence** : Le personnage est pressé (+2 DD)
- **Méfiance** : Le garde est méfiant (+2 DD)
- **DD final** : 19

**Modificateurs** :
- **Désavantage** : À cause de l'urgence et de la méfiance

### Exemple 3 : Crochetage avec aide

**Scénario** : Un personnage tente de crocheter une serrure complexe avec l'aide d'un allié qui l'éclaire.

**Évaluation** :
- **Action de base** : Crocheter une serrure complexe (DD 20)
- **Aide d'un allié** : L'allié éclaire (-2 DD)
- **DD final** : 18

**Modificateurs** :
- **Avantage** : Grâce à l'aide de l'allié
- **Outils de crochetage** : +2 aux tests d'Escamotage
