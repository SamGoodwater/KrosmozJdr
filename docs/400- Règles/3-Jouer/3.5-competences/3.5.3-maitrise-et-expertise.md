# 3.5.3. Maîtrise et expertise

**Description** : Définir la maîtrise/expertise, comment elles s'appliquent aux tests, et leurs sources (classe, spécialisation, équipements).

> **Règles rapides - Maîtrise et expertise**
> - **Maîtrise** : Bonus fixe = 1 + (Niveau / 4) arrondi à l'inférieur (+1 à +6)
> - **Expertise** : Double le bonus de maîtrise (à certains niveaux de spécialisation)
> - **Sources** : Classe, spécialisation, équipements, dons, entraînements
> - **Application** : Ajoute le bonus aux tests de compétence maîtrisée
> - **Limitations** : Compétences non cumulables, équilibre maintenu

## Contenu

- **3.5.3.1. Maîtrise** : Bonus fixe, attribution par classe/spécialisation.
- **3.5.3.2. Expertise** : Double bonus, conditions d'obtention.
- **3.5.3.3. Sources supplémentaires** : Équipements, dons, entraînements.
- **3.5.3.4. Limitations** : Compétences non cumulables, équilibre.

---

## 3.5.3.1. Maîtrise

La **maîtrise** d'une compétence vous permet d'ajouter votre bonus de maîtrise aux tests de cette compétence.

### Bonus fixe

Le bonus de maîtrise est un **bonus fixe** qui augmente avec votre niveau :

**Formule** : Bonus de maîtrise = 1 + (Niveau / 4) arrondi à l'inférieur

**Progression par niveau** :
- Niveau 1-3 : **+1**
- Niveau 4-7 : **+2**
- Niveau 8-11 : **+3**
- Niveau 12-15 : **+4**
- Niveau 16-19 : **+5**
- Niveau 20 : **+6**

### Application

Quand vous maîtrisez une compétence :
- **Ajoutez votre bonus de maîtrise** à tous les tests de cette compétence
- **Le bonus s'ajoute** au modificateur de caractéristique
- **Le bonus augmente** automatiquement avec votre niveau

### Exemple

Un personnage niveau 10 avec :
- Agilité +4
- Maîtrise en Discrétion
- Bonus de maîtrise : +3

**Test de Discrétion** : 1d20 + 4 (Agilité) + 3 (Maîtrise) = 1d20 + 7

---

## 3.5.3.2. Expertise

L'**expertise** double votre bonus de maîtrise pour une compétence spécifique.

### Conditions d'obtention

Vous pouvez obtenir une expertise aux niveaux :
- **Niveau 9** : Première expertise
- **Niveau 15** : Deuxième expertise
- **Niveau 20** : Troisième expertise

**Total** : Maximum **3 expertises** au niveau 20

### Exigence

**Prérequis** : Vous devez déjà **maîtriser** la compétence pour obtenir l'expertise.

### Fonctionnement

**Effet** : Votre bonus de maîtrise est **doublé** pour cette compétence.

**Calcul** : Bonus d'expertise = Bonus de maîtrise × 2

### Exemple

Un personnage niveau 15 avec :
- Agilité +4
- Maîtrise en Discrétion
- Expertise en Discrétion
- Bonus de maîtrise : +4

**Test de Discrétion** : 1d20 + 4 (Agilité) + 8 (Expertise : 2 × 4) = 1d20 + 12

### Choix stratégiques

L'expertise est un choix important :
- **Renforce vos forces** : Doublez vos compétences les plus importantes
- **Compense vos faiblesses** : Expertisez une compétence que vous utilisez souvent mais avec une caractéristique faible
- **Crée des spécialisations** : Devenez expert dans un domaine spécifique

---

## 3.5.3.3. Sources supplémentaires

En plus de la classe et de la spécialisation, vous pouvez obtenir des maîtrises et expertises de sources supplémentaires.

### Équipements

Certains équipements peuvent :
- **Donner des maîtrises** : Conférer la maîtrise d'une compétence
- **Donner des bonus** : Ajouter des bonus aux tests de compétence
- **Donner des expertises** : Conférer l'expertise d'une compétence (rare)

**Exemples** :
- **Outils spécialisés** : Outils qui donnent la maîtrise d'une compétence
- **Livres** : Livres qui donnent des bonus aux tests de connaissances
- **Amulettes** : Amulettes qui renforcent certaines compétences

### Dons

Certains dons peuvent :
- **Donner des maîtrises** : Conférer la maîtrise d'une compétence
- **Donner des expertises** : Conférer l'expertise d'une compétence
- **Améliorer les maîtrises** : Augmenter le bonus de maîtrise

**Note** : Les dons sont des capacités spéciales obtenues à certains niveaux ou par des quêtes spéciales.

### Entraînements

Vous pouvez obtenir des maîtrises par **entraînement** :
- **Temps requis** : Nécessite du temps et des ressources
- **Maître** : Peut nécessiter un maître ou un mentor
- **Limitations** : Généralement limité à quelques compétences

**Exemple** : Un personnage peut s'entraîner avec un maître voleur pour obtenir la maîtrise d'Escamotage.

### Quêtes et récompenses

Certaines quêtes ou récompenses peuvent :
- **Donner des maîtrises** : Conférer la maîtrise d'une compétence
- **Donner des expertises** : Conférer l'expertise d'une compétence
- **Améliorer les compétences** : Augmenter les bonus de compétence

**Exemple** : Une quête spéciale peut récompenser un personnage avec l'expertise en Perception.

## 3.5.3.4. Limitations

Il existe des limitations pour maintenir l'équilibre du jeu.

### Compétences non cumulables

**Même type de bonus** : Les bonus du même type ne se cumulent généralement pas :
- **Meilleur bonus** : On prend le meilleur bonus
- **Exceptions** : Si explicitement indiqué, certains bonus peuvent se cumuler

**Exemple** : Si vous avez +2 de maîtrise et +1 d'équipement, vous avez +2 (pas +3) pour la maîtrise, mais +1 pour l'équipement, donc +3 au total.

### Limites de maîtrise

**Nombre de maîtrises** : Le nombre de maîtrises que vous pouvez avoir est limité :
- **Par classe** : Votre classe vous donne un certain nombre de maîtrises
- **Par spécialisation** : Votre spécialisation vous donne des maîtrises supplémentaires
- **Par niveau** : Certains niveaux peuvent vous donner des maîtrises supplémentaires

**Total** : Généralement, un personnage a entre 4 et 8 maîtrises de compétences au niveau 20.

### Limites d'expertise

**Nombre d'expertises** : Le nombre d'expertises que vous pouvez avoir est limité :
- **Par niveau** : Vous obtenez des expertises aux niveaux 9, 15, et 20
- **Par spécialisation** : Certaines spécialisations peuvent vous donner des expertises supplémentaires
- **Par dons** : Certains dons peuvent vous donner des expertises supplémentaires

**Total** : Généralement, un personnage a entre 3 et 5 expertises au niveau 20.

### Équilibre

**Surveillance du MJ** : Le MJ doit surveiller les combinaisons trop puissantes :
- **Maîtrises multiples** : Trop de maîtrises peuvent créer des déséquilibres
- **Expertises multiples** : Trop d'expertises peuvent créer des déséquilibres
- **Combinaisons** : Certaines combinaisons de maîtrises et expertises peuvent être trop puissantes

**Ajustements** : Le MJ peut ajuster les règles si nécessaire pour maintenir l'équilibre.
