# 3.5.5. Utilisation narrative

**Description** : Proposer des exemples d'utilisation roleplay (social, investigation, craft), comment intégrer les compétences dans la narration.

> **Règles rapides - Utilisation narrative**
> - **Scènes sociales** : Négociations (Persuasion), interrogatoires (Persuasion/Intimidation), discours publics
> - **Enquêtes** : Recherche d'indices (Investigation), analyse de scènes de crime (Perception)
> - **Création et artisanat** : Projets de craft (Artisanat), réalisation de plans (Intelligence)
> - **Exploration immersive** : Approche discrète (Discrétion), survie (Survie), navigation (Survie)
> - **Intégration** : Les compétences enrichissent la narration et créent des opportunités roleplay

## Contenu

- **3.5.5.1. Scènes sociales** : Négociations, interrogatoires, discours publics.
- **3.5.5.2. Enquêtes** : Recherche d'indices, analyse de scènes de crime.
- **3.5.5.3. Création et artisanat** : Projets de craft, réalisation de plans.

## 3.5.5.1. Scènes sociales

Les compétences sociales enrichissent les interactions entre personnages et PNJ.

### Négociations

Les **négociations** utilisent principalement la compétence **Persuasion**.

**Exemple de scène** :
- **Contexte** : Le groupe tente de négocier le prix d'un équipement avec un marchand.
- **Compétence** : Persuasion (Chance)
- **DD** : 15 (négociation normale) ou 20 (marchand réticent)
- **Succès** : Le marchand accepte une réduction de prix
- **Échec** : Le marchand refuse ou augmente même le prix

**Variantes** :
- **Négociation de contrat** : Négocier les termes d'un contrat
- **Négociation de paix** : Négocier un traité de paix entre factions
- **Négociation de rançon** : Négocier la libération d'un otage

### Interrogatoires

Les **interrogatoires** utilisent plusieurs compétences selon l'approche.

**Approche douce** :
- **Compétence** : Persuasion (Chance)
- **DD** : 15-20 selon la résistance de la cible
- **Succès** : La cible révèle des informations

**Approche dure** :
- **Compétence** : Intimidation (Force ou Chance)
- **DD** : 15-20 selon la résistance de la cible
- **Succès** : La cible révèle des informations par peur

**Détection de mensonges** :
- **Compétence** : Perspicacité (Sagesse)
- **DD** : 15-20 selon la capacité de la cible à mentir
- **Succès** : Vous détectez si la cible ment

**Exemple de scène** :
- **Contexte** : Le groupe interroge un suspect sur un vol.
- **Approche** : Le personnage utilise Persuasion pour obtenir des informations
- **Compétence** : Persuasion (Chance) DD 18
- **Succès** : Le suspect révèle des informations importantes
- **Échec** : Le suspect reste silencieux ou ment

### Discours publics

Les **discours publics** utilisent la compétence **Représentation** ou **Persuasion**.

**Représentation** :
- **Compétence** : Représentation (Chance)
- **DD** : 15-20 selon la taille de l'audience
- **Succès** : Le discours inspire ou convainc l'audience
- **Échec** : Le discours ne produit pas l'effet désiré

**Persuasion** :
- **Compétence** : Persuasion (Chance)
- **DD** : 18-25 selon la difficulté de convaincre
- **Succès** : Le discours convainc l'audience
- **Échec** : Le discours ne convainc pas ou crée de l'hostilité

**Exemple de scène** :
- **Contexte** : Un personnage fait un discours pour rallier une ville à sa cause.
- **Compétence** : Persuasion (Chance) DD 20
- **Succès** : La ville se rallie à la cause
- **Échec** : La ville reste indécise ou devient hostile

## 3.5.5.2. Enquêtes

Les compétences d'investigation sont essentielles pour résoudre des mystères.

### Recherche d'indices

La **recherche d'indices** utilise principalement la compétence **Investigation** ou **Perception**.

**Investigation** :
- **Compétence** : Investigation (Intelligence)
- **DD** : 15-20 selon la difficulté de trouver les indices
- **Succès** : Vous trouvez des indices importants
- **Échec** : Vous ne trouvez pas d'indices ou trouvez des indices trompeurs

**Perception** :
- **Compétence** : Perception (Sagesse)
- **DD** : 15-20 selon la difficulté de remarquer
- **Succès** : Vous remarquez des détails importants
- **Échec** : Vous ne remarquez rien

**Exemple de scène** :
- **Contexte** : Le groupe examine une scène de crime pour trouver des indices.
- **Compétence** : Investigation (Intelligence) DD 18
- **Succès** : Vous trouvez des empreintes, des traces de sang, des objets suspects
- **Échec** : Vous ne trouvez rien ou trouvez des indices trompeurs

### Analyse de scènes de crime

L'**analyse de scènes de crime** combine plusieurs compétences.

**Investigation** :
- **Compétence** : Investigation (Intelligence)
- **DD** : 18-25 selon la complexité
- **Succès** : Vous comprenez ce qui s'est passé
- **Échec** : Vous tirez de mauvaises conclusions

**Médecine** :
- **Compétence** : Médecine (Sagesse)
- **DD** : 15-20 selon la complexité
- **Succès** : Vous identifiez la cause de la mort
- **Échec** : Vous ne pouvez pas déterminer la cause

**Exemple de scène** :
- **Contexte** : Le groupe analyse une scène de meurtre pour comprendre ce qui s'est passé.
- **Compétence** : Investigation (Intelligence) DD 20
- **Succès** : Vous comprenez la séquence des événements, identifiez l'arme, trouvez des indices sur l'assassin
- **Échec** : Vous tirez de mauvaises conclusions ou ne trouvez rien

### Résolution d'énigmes

La **résolution d'énigmes** utilise principalement la compétence **Investigation**.

**Investigation** :
- **Compétence** : Investigation (Intelligence)
- **DD** : 15-25 selon la complexité de l'énigme
- **Succès** : Vous résolvez l'énigme
- **Échec** : Vous ne résolvez pas l'énigme ou prenez une mauvaise décision

**Exemple de scène** :
- **Contexte** : Le groupe trouve une énigme dans un donjon qui bloque l'accès à un trésor.
- **Compétence** : Investigation (Intelligence) DD 22
- **Succès** : Vous résolvez l'énigme et ouvrez la porte
- **Échec** : Vous ne résolvez pas l'énigme ou déclenchez un piège

## 3.5.5.3. Création et artisanat

Les compétences d'artisanat permettent de créer, réparer et améliorer des objets.

### Projets de craft

Les **projets de craft** utilisent les compétences d'artisanat appropriées.

**Forge** :
- **Compétence** : Forge (Intelligence ou Force)
- **DD** : 15-25 selon la complexité de l'objet
- **Succès** : Vous créez l'objet avec succès
- **Échec** : Vous échouez et perdez peut-être des matériaux

**Couture** :
- **Compétence** : Couture (Agilité ou Intelligence)
- **DD** : 15-20 selon la complexité
- **Succès** : Vous créez l'objet avec succès
- **Échec** : Vous échouez et perdez peut-être des matériaux

**Exemple de scène** :
- **Contexte** : Un personnage tente de forger une épée de qualité.
- **Compétence** : Forge (Intelligence) DD 20
- **Succès** : Vous créez une épée de qualité supérieure
- **Échec** : Vous échouez et perdez des matériaux, ou créez une épée de qualité inférieure

### Réalisation de plans

La **réalisation de plans** combine plusieurs compétences.

**Investigation** :
- **Compétence** : Investigation (Intelligence)
- **DD** : 15-20 selon la complexité
- **Succès** : Vous comprenez les plans et pouvez les réaliser
- **Échec** : Vous ne comprenez pas les plans

**Compétence d'artisanat** :
- **Compétence** : Compétence d'artisanat appropriée
- **DD** : 18-25 selon la complexité
- **Succès** : Vous réalisez le plan avec succès
- **Échec** : Vous échouez à réaliser le plan

**Exemple de scène** :
- **Contexte** : Un personnage tente de construire un mécanisme complexe selon des plans.
- **Compétence** : Investigation (Intelligence) DD 18 pour comprendre les plans, puis Forge (Intelligence) DD 22 pour construire
- **Succès** : Vous comprenez les plans et construisez le mécanisme
- **Échec** : Vous ne comprenez pas les plans ou échouez à construire

## 3.5.5.4. Exploration immersive

Les compétences d'exploration enrichissent les scènes d'aventure.

### Approche discrète

L'**approche discrète** utilise principalement la compétence **Discrétion**.

**Discrétion** :
- **Compétence** : Discrétion (Agilité)
- **DD** : 15-20 selon la vigilance des gardes
- **Succès** : Vous passez inaperçu
- **Échec** : Vous êtes détecté

**Test opposé** :
- **Votre test** : Discrétion (Agilité)
- **Test du garde** : Perception (Sagesse)
- **Résultat** : Le résultat le plus élevé l'emporte

**Exemple de scène** :
- **Contexte** : Le groupe tente de s'infiltrer dans un château gardé.
- **Compétence** : Discrétion (Agilité) DD 18
- **Succès** : Vous passez inaperçu et entrez dans le château
- **Échec** : Vous êtes détecté et les gardes vous attaquent

### Survie

La **survie** utilise principalement la compétence **Survie**.

**Survie** :
- **Compétence** : Survie (Sagesse)
- **DD** : 15-20 selon la difficulté de l'environnement
- **Succès** : Vous trouvez de la nourriture, de l'eau, un abri
- **Échec** : Vous ne trouvez rien ou vous perdez

**Exemple de scène** :
- **Contexte** : Le groupe est perdu dans une forêt et doit survivre.
- **Compétence** : Survie (Sagesse) DD 15
- **Succès** : Vous trouvez de la nourriture, de l'eau potable, un abri sûr
- **Échec** : Vous ne trouvez rien et subissez les effets de la faim et de la soif

### Navigation

La **navigation** utilise principalement la compétence **Survie** ou **Nature**.

**Survie** :
- **Compétence** : Survie (Sagesse)
- **DD** : 15-20 selon la difficulté de la navigation
- **Succès** : Vous vous orientez correctement
- **Échec** : Vous vous perdez

**Nature** :
- **Compétence** : Nature (Intelligence)
- **DD** : 15-20 selon la connaissance de la région
- **Succès** : Vous connaissez la région et pouvez vous orienter
- **Échec** : Vous ne connaissez pas la région et vous perdez

**Exemple de scène** :
- **Contexte** : Le groupe tente de naviguer dans une forêt dense pour atteindre un objectif.
- **Compétence** : Survie (Sagesse) DD 18
- **Succès** : Vous vous orientez correctement et progressez vers votre objectif
- **Échec** : Vous vous perdez et perdez du temps

### Suivi de pistes

Le **suivi de pistes** utilise principalement la compétence **Survie**.

**Survie** :
- **Compétence** : Survie (Sagesse)
- **DD** : 15-25 selon l'âge des traces et les conditions
- **Succès** : Vous suivez les traces avec succès
- **Échec** : Vous perdez la piste

**Exemple de scène** :
- **Contexte** : Le groupe suit les traces d'une créature qui a volé un objet précieux.
- **Compétence** : Survie (Sagesse) DD 20
- **Succès** : Vous suivez les traces et trouvez la créature
- **Échec** : Vous perdez la piste et ne pouvez pas retrouver la créature

## Conseils pour le MJ

### Intégrer les compétences dans la narration

**Ne pas réduire à un simple jet** :
- **Décrivez l'action** : Décrivez ce que le personnage fait
- **Décrivez le résultat** : Décrivez le résultat de manière narrative
- **Créez des opportunités** : Utilisez les compétences pour créer des opportunités narratives

**Variez les approches** :
- **Différentes compétences** : Permettez l'utilisation de différentes compétences pour résoudre un problème
- **Combinaisons** : Encouragez les combinaisons de compétences
- **Créativité** : Récompensez la créativité dans l'utilisation des compétences

### Créer des scènes engageantes

**Mélangez les compétences** :
- **Plusieurs compétences** : Créez des scènes qui nécessitent plusieurs compétences
- **Coopération** : Encouragez la coopération entre personnages
- **Choix** : Offrez des choix aux joueurs sur les compétences à utiliser

**Rendez les échecs intéressants** :
- **Conséquences narratives** : Les échecs doivent avoir des conséquences narratives
- **Opportunités** : Les échecs peuvent créer de nouvelles opportunités
- **Progression** : Les échecs peuvent mener à de nouvelles scènes
