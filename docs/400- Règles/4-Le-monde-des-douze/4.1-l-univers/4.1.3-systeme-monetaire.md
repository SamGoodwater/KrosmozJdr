# 4.1.3. Système monétaire

**Description** : Fonctionnement des Kamas et des éventuelles monnaies locales, taux d'échange, valeur relative des biens/services.

> **Règles rapides - Système monétaire**
> - **Kamas** : Monnaie officielle du monde des Douze, acceptée partout
> - **Notations** : 1k = 1 000, 1m = 1 million, 1M = 1 milliard
> - **Devises locales** : Jetons de tournois, perles, ressources rares (échangeables)
> - **Troc** : Alternative possible dans les régions reculées
> - **Économie** : Inflation, commerce inter-régions, valeur relative des biens/services

## Contenu

- **4.1.3.1. Kamas** : Standard monétaire, format, conversion.
- **4.1.3.2. Devises locales** : Jetons de tournois, perles, ressources rares.
- **4.1.3.3. Valeur des biens** : Échelles de prix (équipement, consommables, services).
- **4.1.3.4. Économie** : Inflation, commerce inter-régions, troc.

## 4.1.3.1. Kamas

Les **Kamas**** sont la monnaie officielle du monde des Douze, utilisée par tous les royaumes.

### Standard monétaire

**Adoption** : L'ensemble des royaumes du monde des Douze se sont mis d'accord pour utiliser les Kamas officiellement.

**Acceptation** : Les Kamas sont acceptés partout dans le monde des Douze, facilitant le commerce inter-régional.

**Alternative** : Néanmoins, il sera toujours possible d'utiliser le **troc**, particulièrement dans des régions reculées où ce système monétaire n'a que peu d'intérêt pour les locaux.

### Format et conversion

**Notations** :
- **Kamas standards** : 1, 10, 100, 1000 kamas
- **Abréviations** :
  - **1k kamas** = 1 000 kamas
  - **1m kamas** = 1 million de kamas
  - **1M kamas** = 1 milliard de kamas

**Exemples** :
- Une épée coûte **500 kamas**
- Une armure coûte **2k kamas** (2 000 kamas)
- Un objet rare coûte **5m kamas** (5 millions de kamas)

## 4.1.3.2. Devises locales

En plus des Kamas, certaines régions ou événements utilisent des devises locales.

### Jetons de tournois

**Utilisation** : Utilisés lors des tournois organisés dans les grandes villes.

**Valeur** : Généralement échangeables contre des Kamas ou des récompenses spéciales.

**Caractéristiques** :
- **Non transférables** : Ne peuvent être utilisés que dans le contexte du tournoi
- **Récompenses** : Peuvent être échangés contre des prix spéciaux

### Perles

**Utilisation** : Utilisées dans certaines régions côtières ou insulaires.

**Valeur** : Généralement échangeables contre des Kamas selon leur qualité.

**Caractéristiques** :
- **Rareté** : Valeur dépend de la rareté et de la qualité
- **Commerce** : Utilisées dans le commerce maritime

### Ressources rares

**Utilisation** : Certaines ressources rares peuvent servir de monnaie d'échange.

**Exemples** :
- **Fragments de Dofus** : Extrêmement rares et précieux
- **Essences magiques** : Utilisées dans la forgemagie
- **Reliques** : Objets anciens de grande valeur

## 4.1.3.3. Valeur des biens

La valeur des objets dépend de plusieurs facteurs : région, concurrence, rareté, et contexte.

### Facteurs influençant les prix

**Région** :
- **Grandes villes** : Concurrence élevée, prix généralement plus bas
- **Petites villes** : Moins de concurrence, prix généralement plus élevés
- **Régions reculées** : Prix variables, parfois très élevés

**Concurrence** :
- **Marchés animés** : Beaucoup de vendeurs, prix compétitifs
- **Monopoles** : Peu de vendeurs, prix élevés

**Contexte** :
- **Période de guerre** : Les armes coûtent plus cher
- **Pénuries** : Ressources rares, prix élevés
- **Saisons** : Certains biens varient selon les saisons

**Négociation** :
- **Compétence** : La compétence Persuasion peut aider à négocier
- **Risque** : Certains vendeurs peuvent essayer d'escroquer les touristes
- **Relations** : Avoir de bonnes relations peut réduire les prix

### Échelles de prix

#### Armes

**Valeur de base** :
- **Bonus de touche** : 100 kamas par point
- **Bonus de dégâts fixes** : 700 kamas par point
- **Bonus de dégâts multiples** : 900 kamas par point

**Éléments** :
- **Neutre** : Prix de base
- **Terre, Feu, Air, Eau** : Même prix que neutre

**Effets spéciaux** :
- **Avantage au toucher** : +1k kamas minimum
- **Autres effets** : Prix variables selon la puissance

**Exemple** :
- Une épée avec +5 touche et +10 dégâts fixes coûte environ : (5 × 100) + (10 × 700) = 7 500 kamas

#### Équipements

**Caractéristiques principales** :
- **PA** : 1 300 kamas par point
- **PM** : 1 000 kamas par point
- **PO** : 500 kamas par point
- **Vitalité** : 600 kamas par point
- **Force** : 800 kamas par point
- **Intelligence** : 600 kamas par point
- **Sagesse** : 400 kamas par point
- **Agilité** : 70 kamas par point
- **Chance** : 300 kamas par point

**Caractéristiques secondaires** :
- **CA** : 400 kamas par point
- **Résistance fixe** : 700 kamas par point
- **Bonus de touche** : 1 000 kamas par point
- **Initiative** : 300 kamas par point
- **Esquive** : 300 kamas par point
- **Tacle/Fuite** : 300 kamas par point
- **Recharge de PA/PM** : 300 kamas par point
- **Nombre d'invocations** : 1 000 kamas par point

**Compétences** :
- **Acrobaties, Athlétisme, Arcanes, Dressage, Persuasion, etc.** : 300 kamas par point

**Exemple** :
- Un chapeau avec +2 Intelligence et +1 Arcanes coûte environ : (2 × 600) + (1 × 300) = 1 500 kamas

#### Consommables

**Règle générale** : Les consommables coûtent **minimum 10 fois leur niveau**.

**Exemples** :
- **Potion de niveau 5** : Minimum 50 kamas
- **Potion de niveau 10** : Minimum 100 kamas
- **Potion de niveau 20** : Minimum 200 kamas

**Variations** :
- **Potions de soins** : Suivent généralement la règle des 10 × niveau
- **Potions spéciales** : Peuvent coûter beaucoup plus cher
  - **Potion d'invisibilité** : Bien plus chère que 140 kamas (niveau 14)

#### Parchemins d'apprentissage

**Règle** : Les parchemins d'apprentissage de sort coûtent généralement **2 000 fois le niveau du sort**.

**Exemples** :
- **Sort niveau 5** : 10 000 kamas
- **Sort niveau 10** : 20 000 kamas
- **Sort niveau 13** : 26 000 kamas (exemple : Brume)

## 4.1.3.4. Économie

L'économie du monde des Douze est dynamique et influencée par de nombreux facteurs.

### Inflation

**Facteurs d'inflation** :
- **Guerres** : Augmentent les prix des armes et équipements
- **Pénuries** : Ressources rares, prix élevés
- **Croissance économique** : Développement des villes, augmentation des prix

**Gestion** :
- **Stabilité** : Les grandes villes maintiennent généralement une stabilité relative
- **Fluctuations** : Les petites villes peuvent connaître des fluctuations importantes

### Commerce inter-régions

**Routes commerciales** :
- **Terrestres** : Caravanes entre les villes
- **Maritimes** : Navires entre les ports
- **Magiques** : Portails et Zaaps pour le transport rapide

**Avantages** :
- **Diversité** : Accès à des biens de différentes régions
- **Opportunités** : Possibilité de profiter des différences de prix

**Risques** :
- **Brigands** : Attaques sur les routes commerciales
- **Pirates** : Attaques sur les routes maritimes
- **Instabilité** : Conflits régionaux

### Troc

**Utilisation** :
- **Régions reculées** : Où les Kamas ont peu de valeur
- **Échanges directs** : Échange de biens contre biens
- **Négociations** : Accords personnalisés

**Avantages** :
- **Flexibilité** : Pas besoin de monnaie
- **Relations** : Crée des liens entre les parties

**Inconvénients** :
- **Valeur** : Difficile d'estimer la valeur exacte
- **Transport** : Nécessite de transporter les biens
