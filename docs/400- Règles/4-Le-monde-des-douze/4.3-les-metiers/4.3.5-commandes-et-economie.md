# 4.3.5. Commandes et économie

**Description** : Gestion des commandes (PNJ ou PJ), commerce entre joueurs, impact sur l'économie de la campagne.

> **Règles rapides - Commandes et économie**
> - **Commandes PNJ** : Récompenses (prix complet ou main d'œuvre), délais, réputation
> - **Contrats PJ** : Négociation, paiements anticipés, garanties
> - **Marchés** : Mise en vente, taxes, enchères, hôtels de vente
> - **Économie** : Influence sur les prix, abondance/rareté (impact sur la campagne)

## Contenu

- **4.3.5.1. Commandes PNJ** : Récompenses, délais, réputation.
- **4.3.5.2. Contrats PJ** : Négociation, paiements anticipés, garanties.
- **4.3.5.3. Marchés** : Mise en vente, taxes, enchères, hôtels de vente.
- **4.3.5.4. Économie de campagne** : Influence sur les prix, abondance/rareté.

---

## 4.3.5.1. Commandes PNJ

Les PNJ peuvent commander des objets aux personnages artisans.

### Récompenses

**Paiement** :
- **Prix complet** : Prix de l'objet complet
- **Main d'œuvre** : Si ressources fournies, paiement de la main d'œuvre (50% du prix)
- **Bonus** : Bonus possibles pour qualité ou rapidité

**Réputation** :
- **Augmentation** : Augmentation de la réputation avec le PNJ
- **Relations** : Amélioration des relations
- **Opportunités** : Nouvelles opportunités de commandes

**Récompenses spéciales** :
- **Objets rares** : Accès à des objets rares
- **Informations** : Informations précieuses
- **Quêtes** : Accès à de nouvelles quêtes

### Délais

**Délais standard** :
- **Temps** : Délai basé sur le temps de fabrication
- **Négociation** : Délais peuvent être négociés
- **Urgence** : Commandes urgentes avec bonus

**Conséquences du retard** :
- **Réputation** : Réduction de la réputation
- **Pénalités** : Pénalités financières
- **Relations** : Détérioration des relations

### Réputation

**Niveaux** :
- **Neutre** : Réputation de base
- **Positif** : Réputation positive avec bonus
- **Excellent** : Réputation excellente avec avantages
- **Négatif** : Réputation négative avec malus

**Effets** :
- **Prix** : Influence sur les prix proposés
- **Commandes** : Accès à plus de commandes
- **Relations** : Amélioration des relations avec d'autres PNJ

---

## 4.3.5.2. Contrats PJ

Les personnages peuvent passer des contrats entre eux.

### Négociation

**Termes** :
- **Prix** : Négociation du prix
- **Délais** : Négociation des délais
- **Qualité** : Négociation de la qualité attendue

**Garanties** :
- **Qualité** : Garanties de qualité
- **Délais** : Garanties de respect des délais
- **Réparations** : Garanties de réparations en cas de défaut

### Paiements anticipés

**Acomptes** :
- **Montant** : Montant de l'acompte négocié
- **Sécurité** : Sécurité pour l'artisan
- **Engagement** : Engagement pour le client

**Paiements échelonnés** :
- **Étapes** : Paiements à différentes étapes
- **Sécurité** : Sécurité pour les deux parties
- **Flexibilité** : Flexibilité dans le paiement

### Garanties

**Garanties de qualité** :
- **Standards** : Standards de qualité garantis
- **Réparations** : Réparations en cas de défaut
- **Remplacement** : Remplacement en cas de problème

**Garanties de délais** :
- **Respect** : Respect des délais convenus
- **Pénalités** : Pénalités en cas de retard
- **Compensation** : Compensation pour les retards

---

## 4.3.5.3. Marchés

Les personnages peuvent vendre leurs créations sur les marchés.

### Mise en vente

**Hôtels de vente** :
- **Fonction** : Mise en vente d'objets
- **Frais** : Frais de mise en vente
- **Durée** : Durée de mise en vente

**Marchés publics** :
- **Fonction** : Vente directe sur les marchés
- **Négociation** : Négociation directe avec les clients
- **Flexibilité** : Plus de flexibilité dans les prix

### Taxes

**Taxes de vente** :
- **Taux** : Taux de taxe selon les régions
- **Calcul** : Calcul sur le prix de vente
- **Paiement** : Paiement des taxes

**Taxes d'hôtel de vente** :
- **Frais** : Frais de mise en vente
- **Commission** : Commission sur les ventes
- **Gestion** : Gestion par les hôtels de vente

### Enchères

**Système d'enchères** :
- **Fonction** : Vente aux enchères
- **Processus** : Processus d'enchères
- **Avantages** : Prix potentiellement plus élevés

**Enchères spéciales** :
- **Événements** : Enchères lors d'événements spéciaux
- **Objets rares** : Enchères pour objets rares
- **Prestige** : Prestige associé aux enchères

## 4.3.5.4. Économie de campagne

L'économie de la campagne est influencée par les actions des personnages.

### Influence sur les prix

**Abondance** :
- **Effet** : Abondance de ressources réduit les prix
- **Conséquence** : Prix plus bas pour les ressources
- **Opportunités** : Opportunités pour les acheteurs

**Rareté** :
- **Effet** : Rareté de ressources augmente les prix
- **Conséquence** : Prix plus élevés pour les ressources
- **Opportunités** : Opportunités pour les vendeurs

**Demande** :
- **Effet** : Demande élevée augmente les prix
- **Conséquence** : Prix plus élevés pour les objets demandés
- **Opportunités** : Opportunités pour les producteurs

### Abondance/rareté

**Facteurs** :
- **Récoltes** : Récoltes abondantes ou rares
- **Créatures** : Disponibilité des créatures
- **Événements** : Événements affectant les ressources

**Conséquences** :
- **Prix** : Variation des prix
- **Disponibilité** : Disponibilité des ressources
- **Opportunités** : Nouvelles opportunités économiques

### Dynamique économique

**Cycles** :
- **Saisons** : Cycles saisonniers
- **Événements** : Cycles d'événements
- **Tendances** : Tendances économiques

**Adaptation** :
- **Stratégies** : Adaptation des stratégies économiques
- **Investissements** : Investissements opportuns
- **Risques** : Gestion des risques économiques
