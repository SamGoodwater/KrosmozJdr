# 4.3.4. Forgemagie

**Description** : Système d'amélioration/modification d'équipements, risques de casser l'objet, ressources spécifiques.

> **Règles rapides - Forgemagie**
> - **Améliorations** : Bonus de stats (PA, PM, PO, PV, caractéristiques), effets spéciaux, runes
> - **Risques** : Échec (perte de ressources), casse (objet détruit), répercussions
> - **Ressources** : Runes, essences, catalyseurs (coût variable selon amélioration)
> - **Processus** : Étapes spécifiques, temps requis, assistance de PNJ possible

## Contenu

- **4.3.4.1. Types d'améliorations** : Bonus de stats, effets spéciaux, runes.
- **4.3.4.2. Risques** : Échec, casse, répercussions.
- **4.3.4.3. Ressources nécessaires** : Runes, essences, catalyseurs.
- **4.3.4.4. Processus** : Étapes, temps, assistance de PNJ.

---

## 4.3.4.1. Types d'améliorations

La forgemagie permet d'améliorer les équipements en ajoutant des bonus via des runes.

### Bonus de caractéristiques

**Caractéristiques principales** :
- **PA** : +1 maximum (3900 kamas), Amulettes uniquement
- **PM** : +1 maximum (3000 kamas), Bottes uniquement
- **PO** : +1 maximum (2400 kamas), Anneaux uniquement
- **Nombre d'invocations** : +1 maximum (2400 kamas), Anneaux uniquement
- **Points de vie maximum** : +20 maximum (150 kamas), Tous équipements

**Caractéristiques secondaires** :
- **Vitalité** : +1 maximum (3600 kamas), Chapeaux uniquement
- **Sagesse** : +1 maximum (3600 kamas), Chapeaux uniquement
- **Initiative** : +3 maximum (210 kamas), Tous équipements
- **Force, Intelligence, Agilité, Chance** : +1 maximum (3000 kamas), Capes uniquement

**Résistances** :
- **Résistance neutre** : +3 maximum (2100 kamas), Boucliers uniquement
- **Résistance terre** : +3 maximum (2100 kamas), Boucliers uniquement
- **Résistance feu** : +3 maximum (2100 kamas), Boucliers uniquement
- **Résistance air** : +3 maximum (2100 kamas), Boucliers uniquement
- **Résistance eau** : +3 maximum (2100 kamas), Boucliers uniquement

### Bonus de compétences

**Compétences physiques** :
- **Acrobatie** : +3 maximum (1000 kamas), Tous équipements
- **Athlétisme** : +3 maximum (1000 kamas), Tous équipements
- **Discrétion** : +3 maximum (1000 kamas), Tous équipements
- **Escamotage** : +3 maximum (1000 kamas), Tous équipements

**Compétences mentales** :
- **Arcanes** : +3 maximum (1000 kamas), Tous équipements
- **Histoire** : +3 maximum (1000 kamas), Tous équipements
- **Investigation** : +3 maximum (1000 kamas), Outils uniquement
- **Nature** : +3 maximum (1000 kamas), Tous équipements
- **Religion** : +3 maximum (1000 kamas), Tous équipements

**Compétences sociales** :
- **Persuasion** : +3 maximum (1000 kamas), Tous équipements
- **Représentation** : +3 maximum (1000 kamas), Tous équipements
- **Supercherie** : +3 maximum (1000 kamas), Tous équipements

**Compétences de sagesse** :
- **Dressage** : +3 maximum (1000 kamas), Tous équipements
- **Médecine** : +3 maximum (1000 kamas), Tous équipements
- **Perception** : +3 maximum (1000 kamas), Tous équipements
- **Perspicacité** : +3 maximum (1000 kamas), Tous équipements
- **Survie** : +3 maximum (1000 kamas), Tous équipements

### Bonus de combat

**Esquive** :
- **Esquive PA** : +2 maximum (900 kamas), Tous équipements
- **Esquive PM** : +2 maximum (900 kamas), Tous équipements

**Tacle/Fuite** :
- **Tacle** : +2 maximum (900 kamas), Tous équipements
- **Fuite** : +2 maximum (900 kamas), Tous équipements

**Dégâts** :
- **Dommages** : +2d6 maximum (4000 kamas), Armes uniquement
- **Critiques** : +1 maximum (4000 kamas), Armes uniquement
- **Dommage fixe** : +3 à +5 maximum (1500-1800 kamas), Armes uniquement

### Limitations

**Une seule rune par objet** :
- **Règle** : Un objet ne peut être forgemagé qu'avec une seule rune
- **Conséquence** : Un seul bonus par équipement
- **Choix** : Choix stratégique de la rune

**Pas de cumul** :
- **Règle** : Une créature ne peut pas cumuler les bonus d'un même type
- **Exemple** : Si un équipement a une rune +PA, un autre équipement ne peut pas avoir une rune +PA
- **Conséquence** : Diversification nécessaire

**Limites de caractéristiques** :
- **Règle** : La forgemagie ne permet pas de dépasser les limites d'une caractéristique
- **Exemple** : Les PA sont limités à 12, même avec une rune +PA
- **Conséquence** : Planification nécessaire

## 4.3.4.2. Risques

La forgemagie comporte des risques importants.

### Échec

**Test de forgemagie** :
- **Formule** : 1d20 + Modificateur + Bonus
- **DD** : Niveau de l'équipement + 6
- **Exemple** : Objet niveau 10 = DD 16

**Conséquences de l'échec** :
- **Runes perdues** : Les runes utilisées sont perdues
- **Équipement** : L'équipement peut être endommagé
- **Temps** : Le temps écoulé est perdu

### Casse

**Échec critique** :
- **Seuil** : Échec critique sur un 1 naturel
- **Conséquence** : L'équipement peut être complètement détruit
- **Runes** : Les runes sont perdues

**Dégâts** :
- **Endommagement** : L'équipement peut être endommagé
- **Réparation** : Nécessite des réparations coûteuses
- **Restauration** : Peut nécessiter des ressources rares

### Répercussions

**Coût financier** :
- **Runes** : Runes très chères et rares
- **Réparations** : Coûts de réparation élevés
- **Remplacement** : Coût de remplacement de l'équipement

**Temps** :
- **Temps perdu** : Temps investi perdu en cas d'échec
- **Retard** : Retard dans les projets
- **Opportunités** : Opportunités manquées

## 4.3.4.3. Ressources nécessaires

La forgemagie nécessite des ressources spécifiques et rares.

### Runes

**Rareté** :
- **Très rares** : Les runes sont extrêmement rares
- **Prix** : Environ 3 fois le prix de la caractéristique sur équipement de base
- **Disponibilité** : Ne peuvent pas être trouvées ou achetées n'importe où

**Obtention** :
- **Quêtes** : Souvent l'objet de quêtes
- **Boss** : Récompenses de boss
- **Marchands spéciaux** : Marchands spécialisés et rares
- **Échange** : Souvent échangées plutôt que vendues

### Essences

**Types** :
- **Essences élémentaires** : Feu, eau, terre, air
- **Essences magiques** : Essences de Wakfu
- **Essences rares** : Essences légendaires

**Usage** :
- **Catalyseurs** : Utilisées comme catalyseurs
- **Stabilisation** : Stabilisent le processus
- **Amélioration** : Améliorent les chances de réussite

### Catalyseurs

**Types** :
- **Catalyseurs de base** : Catalyseurs communs
- **Catalyseurs rares** : Catalyseurs rares
- **Catalyseurs épiques** : Catalyseurs légendaires

**Usage** :
- **Réduction des risques** : Réduisent les risques d'échec
- **Amélioration** : Améliorent les bonus
- **Stabilisation** : Stabilisent le processus

## 4.3.4.4. Processus

Le processus de forgemagie suit des étapes précises.

### Étapes

**1. Préparation** :
- **Équipement** : Préparer l'équipement à améliorer
- **Runes** : Préparer les runes nécessaires
- **Atelier** : Accéder à un atelier de forgemagie

**2. Activation** :
- **Runes** : Activer les runes
- **Équipement** : Lier les runes à l'équipement
- **Magie** : Canaliser la magie nécessaire

**3. Test** :
- **Test de forgemagie** : Effectuer le test de forgemagie
- **DD** : Niveau de l'équipement + 6
- **Résultat** : Succès ou échec

**4. Finalisation** :
- **Succès** : L'équipement est amélioré
- **Échec** : Conséquences selon le type d'échec

### Temps

**Durée** :
- **Règle** : Temps égal au niveau de l'équipement en heures
- **Exemple** : Objet niveau 10 = 10 heures
- **Modificateurs** : Réduction possible avec expérience et outils

**Interruptions** :
- **Risques** : Interruptions peuvent causer des échecs
- **Protection** : Nécessite un environnement calme et sécurisé
- **Concentration** : Nécessite de la concentration

### Assistance de PNJ

**Maîtres forgemageurs** :
- **Guidance** : Guidance d'un maître peut améliorer les chances
- **Réduction des risques** : Réduction des risques d'échec
- **Coût** : Nécessite généralement un paiement

**Ateliers spécialisés** :
- **Bonus** : Bonus aux tests de forgemagie
- **Sécurité** : Réduction des risques de casse
- **Accès** : Nécessitent souvent une réputation ou des relations
