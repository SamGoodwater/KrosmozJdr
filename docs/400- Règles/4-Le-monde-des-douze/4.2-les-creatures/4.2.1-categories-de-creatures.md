# 4.2.1. Catégories de créatures

**Description** : Classer les créatures par type : animaux, humanoïdes, aberrations, boss, invocations, etc.

> **Règles rapides - Créatures**
> - **Faune sauvage** : Généralement neutre, peut devenir agressive si menacée
> - **Humanoïdes** : Intelligents, peuvent être alliés ou ennemis
> - **Aberrations/Boss** : Puissantes, souvent uniques, gardiens de donjons
> - **Invocations** : Créatures contrôlées par les joueurs (Osamodas, Sadida)
> - **Comportement** : Neutre, Agressif, Territorial, Protecteur

## Contenu

- **4.2.1.1. Faune sauvage** : Bouftous, Dragodindes, créatures neutres.
- **4.2.1.2. Humanoïdes** : Bandits, miliciens, cultistes.
- **4.2.1.3. Aberrations et boss** : Gardiens de donjons, entités wakfu corrompues.
- **4.2.1.4. Invocations** : Créatures contrôlées par les classes (Osamodas, Sadida).

---

## 4.2.1.1. Faune sauvage

La faune sauvage regroupe les créatures naturelles qui peuplent le monde des Douze.

### Bouftous

**Description** : Créatures herbivores paisibles, généralement de taille moyenne.

**Caractéristiques** :
- **Taille** : Moyenne (1,50 m × 1,50 m)
- **Comportement** : Généralement neutre, peut devenir agressif si menacé
- **Habitat** : Plaines, forêts, prairies
- **Ressources** : Laine, viande, cornes

**Variantes** :
- **Bouftou** : Version standard (niveau 1-3)
- **Bouftou royal** : Version plus puissante (niveau 5-7)
- **Bouftou noir** : Version plus agressive (niveau 8-10)

| Type | Niveau | PV | Comportement | Ressources |
|------|-------|----|--------------|------------|
| **Bouftou** | 1-3 | 20-40 | Neutre | Laine, viande |
| **Bouftou royal** | 5-7 | 60-100 | Agressif si menacé | Laine rare, cornes |
| **Bouftou noir** | 8-10 | 120-180 | Agressif | Laine noire, cornes précieuses |

### Dragodindes

**Description** : Créatures reptiliennes utilisées comme montures et pour leur viande.

**Caractéristiques** :
- **Taille** : Grande (3 m × 3 m) à Très Grande (4,50 m × 4,50 m)
- **Comportement** : Peut être domestiqué, généralement neutre
- **Habitat** : Plaines, déserts, zones chaudes
- **Ressources** : Viande, écailles, griffes

**Variantes** :
- **Dragodinde** : Version standard
- **Dragodinde dorée** : Version rare et précieuse
- **Dragodinde sauvage** : Version agressive

### Tofus

**Description** : Petites créatures magiques, généralement inoffensives.

**Caractéristiques** :
- **Taille** : Très Petite (75 cm × 75 cm)
- **Comportement** : Généralement neutre, curieux
- **Habitat** : Forêts, zones magiques
- **Ressources** : Peau, essence magique

**Variantes** :
- **Tofu** : Version standard
- **Tofu royal** : Version plus puissante
- **Tofu noir** : Version plus agressive

### Autres créatures neutres

**Wabbits** :
- **Description** : Petites créatures rapides et agiles
- **Taille** : Petite (1,50 m × 1,50 m)
- **Comportement** : Généralement neutre, peut fuir rapidement

**Chafers** :
- **Description** : Créatures insectoïdes agressives
- **Taille** : Moyenne à Grande
- **Comportement** : Agressif, territorial

**Craqueleurs** :
- **Description** : Grandes créatures puissantes
- **Taille** : Très Grande (4,50 m × 4,50 m) à Gigantesque (6 m × 6 m)
- **Comportement** : Agressif, très dangereux

## 4.2.1.2. Humanoïdes

Les humanoïdes sont des créatures intelligentes qui peuvent être des alliés ou des ennemis.

### Bandits

**Description** : Criminels et hors-la-loi qui attaquent les voyageurs.

**Caractéristiques** :
- **Taille** : Moyenne (1,50 m × 1,50 m)
- **Comportement** : Agressif, opportuniste
- **Habitat** : Routes, forêts, zones isolées
- **Ressources** : Équipement, argent, objets volés

**Variantes** :
- **Bandit** : Version standard
- **Bandit de grand chemin** : Version plus puissante
- **Chef de bandits** : Version leader

### Miliciens

**Description** : Soldats et gardes au service des royaumes.

**Caractéristiques** :
- **Taille** : Moyenne (1,50 m × 1,50 m)
- **Comportement** : Neutre à hostile selon l'allégeance
- **Habitat** : Villes, forteresses, routes gardées
- **Ressources** : Équipement militaire, armes, armures

**Variantes** :
- **Milice** : Version standard
- **Garde royal** : Version plus puissante
- **Capitaine** : Version leader

### Cultistes

**Description** : Adeptes de cultes maléfiques ou mystérieux.

**Caractéristiques** :
- **Taille** : Moyenne (1,50 m × 1,50 m)
- **Comportement** : Hostile, fanatique
- **Habitat** : Temples secrets, ruines, zones isolées
- **Ressources** : Objets rituels, reliques, parchemins

**Variantes** :
- **Cultiste** : Version standard
- **Prêtre du culte** : Version plus puissante
- **Grand prêtre** : Version leader

### Autres humanoïdes

**Marchands** :
- **Description** : Commerçants itinérants ou sédentaires
- **Comportement** : Neutre, intéressé par le commerce

**Aventuriers** :
- **Description** : Autres aventuriers explorant le monde
- **Comportement** : Variable, peut être allié ou ennemi

**Villageois** :
- **Description** : Habitants des villages et villes
- **Comportement** : Généralement neutre, peut demander de l'aide

## 4.2.1.3. Aberrations et boss

Les aberrations et boss sont des créatures puissantes et souvent uniques.

### Gardiens de donjons

**Description** : Créatures puissantes qui gardent les donjons et leurs trésors.

**Caractéristiques** :
- **Taille** : Grande à Gigantesque
- **Comportement** : Hostile, territorial
- **Habitat** : Donjons, zones souterraines
- **Ressources** : Trésors rares, équipements puissants

**Exemples** :
- **Kardorim** : Gardien de donjon de glace
- **Groloum** : Gardien de donjon volcanique
- **Comte Harebourg** : Gardien de donjon temporel

### Entités wakfu corrompues

**Description** : Créatures corrompues par des forces maléfiques ou des perturbations magiques.

**Caractéristiques** :
- **Taille** : Variable, souvent Grande à Gigantesque
- **Comportement** : Hostile, chaotique
- **Habitat** : Zones corrompues, lieux maudits
- **Ressources** : Essences corrompues, fragments de Wakfu

**Types** :
- **Créatures corrompues** : Créatures normales transformées
- **Entités pures** : Créatures nées de la corruption
- **Gardiens corrompus** : Gardiens transformés par la corruption

### Boss légendaires

**Description** : Créatures légendaires et uniques, souvent liées à des quêtes épiques.

**Caractéristiques** :
- **Taille** : Généralement Gigantesque
- **Comportement** : Hostile, très puissant
- **Habitat** : Zones spéciales, lieux uniques
- **Ressources** : Trésors légendaires, équipements uniques

**Exemples** :
- **Dragons** : Créatures légendaires puissantes
- **Démons** : Entités maléfiques puissantes
- **Entités divines** : Créatures liées aux dieux

## 4.2.1.4. Invocations

Les invocations sont des créatures créées ou contrôlées par certaines classes.

### Invocations d'Osamodas

**Description** : Créatures invoquées et contrôlées par les Osamodas.

**Caractéristiques** :
- **Taille** : Variable selon le sort
- **Comportement** : Contrôlé par l'invocateur
- **Durée** : Temporaire, selon le sort
- **Types** : Diverses créatures selon les sorts

**Exemples** :
- **Bouftous invoqués** : Bouftous contrôlés
- **Dragodindes invoquées** : Dragodindes contrôlées
- **Créatures spéciales** : Créatures uniques aux Osamodas

### Poupées de Sadida

**Description** : Poupées magiques créées et contrôlées par les Sadida.

**Caractéristiques** :
- **Taille** : Généralement Petite à Moyenne
- **Comportement** : Contrôlé par le Sadida
- **Durée** : Temporaire ou permanente selon le sort
- **Types** : Diverses poupées selon les sorts

**Exemples** :
- **Poupées d'attaque** : Poupées offensives
- **Poupées de soutien** : Poupées défensives
- **Poupées spéciales** : Poupées avec des capacités uniques

### Autres invocations

**Invocations d'autres classes** :
- **Xélor** : Invocations temporelles
- **Eliotrope** : Invocations dimensionnelles
- **Autres** : Diverses invocations selon les classes

**Caractéristiques communes** :
- **Contrôle** : Généralement contrôlées par l'invocateur
- **Durée** : Temporaire selon le sort
- **Limites** : Nombre limité d'invocations simultanées

| **Grande (G)** | 3 m × 3 m | - |
| **Très Grande (TG)** | 4,50 m × 4,50 m | Craqueleur |
| **Gigantesque (Gig)** | 6 m × 6 m ou plus | - |

### Espace en combat

**Définition** : L'espace d'une créature est la surface qu'elle contrôle effectivement durant le combat, ce n'est pas une mesure de sa dimension physique.

**Encerclement** :
- **Créature Moyenne** : Jusqu'à 8 créatures peuvent l'entourer
- **Créature Grande** : Moins de créatures peuvent l'entourer
- **Créature Gigantesque** : Jusqu'à 20 créatures Moyennes peuvent l'entourer

### Passage dans un espace étroit

**Règle** : Une créature peut passer dans un espace assez large pour une créature une taille plus petite qu'elle.

**Exemple** : Des créatures de taille Grande peuvent traverser un passage de 1,50 mètre de large.

**Pénalités** :
- **Coût de mouvement** : Chaque mètre dans un passage étroit coûte un mètre supplémentaire
- **Désavantage** : Désavantage aux jets d'attaque et jets d'Agilité
- **Avantage ennemi** : Les jets d'attaque contre cette créature ont un avantage
