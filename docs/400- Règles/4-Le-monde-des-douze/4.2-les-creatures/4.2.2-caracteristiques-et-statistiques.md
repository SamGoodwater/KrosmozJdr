**Description** : Décrire les attributs propres aux créatures (caractéristiques, PV, résistances) et leur adaptation depuis le jeu vidéo.

> **Règles rapides - Caractéristiques et statistiques**
> - **Caractéristiques** : Force, Intelligence, Sagesse, Agilité, Chance, Vitalité (adaptées aux créatures)
> - **Ressources de combat** : PA, PM, PO, initiatives spécifiques
> - **Défenses** : PV, boucliers, résistances réduites/augmentées
> - **Adaptation Dofus** : Conversion rapide depuis les stats du jeu vidéo
> - **Ajustements MJ** : Possibilité d'ajuster selon le niveau des personnages

- **4.2.2.1. Caractéristiques de base** : Force, Intelligence, etc. adaptées aux créatures.

## 4.2.2.1. Caractéristiques de base

Les créatures utilisent les mêmes caractéristiques que les personnages, mais avec des valeurs adaptées à leur nature.

### Les six caractéristiques

**Force** :
- **Usage** : Dégâts physiques, tests d'Athlétisme
- **Créatures fortes** : Bouftous, Craqueleurs, créatures de combat
- **Valeurs typiques** : Variable selon le type de créature

**Intelligence** :
- **Usage** : Sorts, tests de connaissances
- **Créatures intelligentes** : Mages, créatures magiques, humanoïdes
- **Valeurs typiques** : Variable selon le type de créature

**Sagesse** :
- **Usage** : Perception, tests de Sagesse
- **Créatures sages** : Créatures anciennes, gardiens, sages
- **Valeurs typiques** : Variable selon le type de créature

**Agilité** :
- **Usage** : Vitesse, esquive, tests d'Agilité
- **Créatures agiles** : Wabbits, créatures rapides, furtives
- **Valeurs typiques** : Variable selon le type de créature

**Chance** :
- **Usage** : Tests de Chance, effets aléatoires
- **Créatures chanceuses** : Créatures magiques, créatures rares
- **Valeurs typiques** : Variable selon le type de créature

**Vitalité** :
- **Usage** : Points de vie, résistance aux effets
- **Toutes les créatures** : Toutes les créatures ont de la Vitalité
- **Valeurs typiques** : Variable selon le type et le niveau

### Adaptation depuis Dofus

**Conversion** :
- **Niveau** : Le niveau de la créature dans Dofus correspond généralement au niveau dans le JDR
- **Caractéristiques** : Les caractéristiques sont adaptées proportionnellement
- **Équilibrage** : Les valeurs sont ajustées pour l'équilibre du JDR

**Exemples** :
- **Bouftou niveau 10** : Caractéristiques adaptées pour un niveau 10
- **Boss niveau 50** : Caractéristiques adaptées pour un niveau 50
- **Créature faible** : Caractéristiques réduites pour les créatures faibles

## 4.2.2.2. Ressources de combat

Les créatures utilisent les mêmes ressources de combat que les personnages.

### Points d'Action (PA)

**Fonctionnement** :
- **Utilisation** : Identique aux personnages
- **Valeurs** : Variable selon le type de créature
- **Régénération** : Généralement régénérés chaque tour

**Exemples** :
- **Créature faible** : 3-4 PA
- **Créature moyenne** : 5-6 PA
- **Créature puissante** : 7-8 PA
- **Boss** : 8-10 PA ou plus

### Points de Mouvement (PM)

**Fonctionnement** :
- **Utilisation** : Identique aux personnages
- **Valeurs** : Variable selon le type de créature
- **Régénération** : Généralement régénérés chaque tour

**Exemples** :
- **Créature lente** : 2-3 PM
- **Créature moyenne** : 4-5 PM
- **Créature rapide** : 6-7 PM
- **Créature très rapide** : 8+ PM

### Points d'Opportunité (PO)

**Fonctionnement** :
- **Utilisation** : Identique aux personnages
- **Valeurs** : Variable selon le type de créature
- **Régénération** : Généralement régénérés chaque tour

**Exemples** :
- **Créature standard** : 1-2 PO
- **Créature spécialisée** : 3-4 PO
- **Boss** : 5+ PO

### Initiative

**Fonctionnement** :
- **Calcul** : Identique aux personnages (Agilité + modificateurs)
- **Valeurs** : Variable selon le type de créature
- **Ordre de tour** : Détermine l'ordre d'action en combat

**Exemples** :
- **Créature lente** : Initiative faible
- **Créature rapide** : Initiative élevée
- **Boss** : Initiative généralement élevée

## 4.2.2.3. Défenses

Les créatures ont diverses défenses pour survivre au combat.

### Points de Vie (PV)

**Fonctionnement** :
- **Calcul** : Vitalité + modificateurs
- **Valeurs** : Variable selon le type et le niveau
- **Régénération** : Généralement pas de régénération automatique

**Exemples** :
- **Créature faible** : 20-50 PV
- **Créature moyenne** : 50-100 PV
- **Créature puissante** : 100-200 PV
- **Boss** : 200+ PV

### Boucliers

**Fonctionnement** :
- **Types** : Boucliers magiques, boucliers physiques
- **Valeurs** : Variable selon le type de créature
- **Régénération** : Certains boucliers se régénèrent

**Exemples** :
- **Créature avec bouclier** : 10-50 points de bouclier
- **Créature magique** : Boucliers magiques régénérables
- **Boss** : Boucliers puissants et régénérables

### Résistances

**Résistances réduites** :
- **Fonctionnement** : Réduction des dégâts d'un type spécifique
- **Types** : Résistance au feu, à l'eau, à la terre, à l'air, neutre
- **Valeurs** : Variable selon le type de créature

**Résistances augmentées** :
- **Fonctionnement** : Augmentation des dégâts d'un type spécifique
- **Types** : Vulnérabilité au feu, à l'eau, à la terre, à l'air, neutre
- **Valeurs** : Variable selon le type de créature

**Exemples** :
- **Créature de feu** : Résistance au feu, vulnérabilité à l'eau
- **Créature de glace** : Résistance à l'eau, vulnérabilité au feu
- **Créature neutre** : Pas de résistances particulières

### Classe d'Armure (CA)

**Fonctionnement** :
- **Calcul** : Identique aux personnages
- **Valeurs** : Variable selon le type de créature
- **Sources** : Armure naturelle, équipement, magie

**Exemples** :
- **Créature sans armure** : CA basse
- **Créature avec armure naturelle** : CA moyenne à élevée
- **Créature avec équipement** : CA selon l'équipement

## 4.2.2.4. Ajustements MJ

Le MJ peut ajuster les statistiques des créatures pour s'adapter à son groupe.

### Conversion rapide depuis Dofus

**Méthode simple** :
1. **Niveau** : Utiliser le niveau de la créature dans Dofus
2. **Caractéristiques** : Multiplier par un facteur (ex: niveau × 2 pour les PV)
3. **Ressources** : Adapter selon le type de créature
4. **Ajustements** : Ajuster selon l'équilibre du groupe

**Facteurs de conversion** :
- **PV** : Niveau × 5-10 selon la robustesse
- **PA** : 3-6 selon le type
- **PM** : 3-6 selon la mobilité
- **Caractéristiques** : Niveau × 1-2 selon le type

### Ajustements selon le groupe

**Groupe faible** :
- **Réduire** : Réduire les PV, les dégâts, les résistances
- **Simplifier** : Simplifier les capacités spéciales
- **Ajuster** : Ajuster le nombre de créatures

**Groupe puissant** :
- **Augmenter** : Augmenter les PV, les dégâts, les résistances
- **Complexifier** : Ajouter des capacités spéciales
- **Ajuster** : Augmenter le nombre de créatures ou ajouter des boss

**Groupe équilibré** :
- **Standard** : Utiliser les statistiques de base
- **Varier** : Varier les types de créatures
- **Adapter** : Adapter selon les forces et faiblesses du groupe

### Exemples de conversion

**Exemple 1 : Bouftou niveau 10** :
- **PV** : 10 × 5 = 50 PV
- **PA** : 4 PA
- **PM** : 3 PM
- **Force** : +2
- **Vitalité** : +3

**Exemple 2 : Boss niveau 50** :
- **PV** : 50 × 8 = 400 PV
- **PA** : 8 PA
- **PM** : 5 PM
- **Caractéristiques** : +5 à +8 selon le type
- **Capacités** : Capacités spéciales puissantes
