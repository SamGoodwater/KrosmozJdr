## 4.4.2.1. Échelle de rareté

Les ressources sont classées selon leur rareté, déterminant leur valeur et leur disponibilité.

### Commun

**Caractéristiques** :
- **Disponibilité** : Très facile à trouver
- **Probabilité** : 60-80% de chance d'obtenir
- **Valeur** : 10-50 kamas par unité
- **Usage** : Utilisé dans de nombreuses recettes de base

**Exemples** :
- **Laine de Bouftou** : 20-30 kamas
- **Bois d'Amakna** : 15-25 kamas
- **Minerai de fer** : 30-50 kamas

### Rare

**Caractéristiques** :
- **Disponibilité** : Difficile à trouver
- **Probabilité** : 15-25% de chance d'obtenir
- **Valeur** : 50-200 kamas par unité
- **Usage** : Utilisé dans des recettes avancées

**Exemples** :
- **Laine de qualité** : 80-120 kamas
- **Essence magique** : 100-150 kamas
- **Minerai d'or** : 150-200 kamas

### Épique

**Caractéristiques** :
- **Disponibilité** : Très difficile à trouver
- **Probabilité** : 5-10% de chance d'obtenir
- **Valeur** : 200-1000 kamas par unité
- **Usage** : Utilisé dans des recettes épiques

**Exemples** :
- **Laine royale** : 300-500 kamas
- **Essence pure** : 400-600 kamas
- **Cristal wakfu pur** : 500-800 kamas

### Légendaire

**Caractéristiques** :
- **Disponibilité** : Extrêmement rare
- **Probabilité** : 1-5% de chance d'obtenir
- **Valeur** : 1000+ kamas par unité, parfois inestimables
- **Usage** : Utilisé dans des recettes légendaires

**Exemples** :
- **Fragment de Dofus** : 2000-5000 kamas
- **Relique légendaire** : 5000+ kamas
- **Dofus complet** : Inestimable

## 4.4.2.2. Valeurs indicatives

Les valeurs des ressources varient selon plusieurs facteurs.

### Prix moyens

**Ressources communes** :
- **Prix** : 10-50 kamas par unité
- **Variation** : Peu de variation selon les régions
- **Marché** : Facilement disponibles

**Ressources rares** :
- **Prix** : 50-200 kamas par unité
- **Variation** : Variation selon les régions et la demande
- **Marché** : Disponibles mais plus chères

**Ressources épiques** :
- **Prix** : 200-1000 kamas par unité
- **Variation** : Grande variation selon les régions
- **Marché** : Difficiles à trouver

**Ressources légendaires** :
- **Prix** : 1000+ kamas par unité, parfois inestimables
- **Variation** : Variation extrême, parfois uniques
- **Marché** : Très rares, souvent troquées

### Variations régionales

**Grandes villes** :
- **Prix** : Généralement plus bas en raison de la concurrence
- **Disponibilité** : Plus grande disponibilité
- **Variété** : Plus grande variété

**Petites villes** :
- **Prix** : Généralement plus élevés
- **Disponibilité** : Disponibilité limitée
- **Variété** : Variété limitée

**Régions reculées** :
- **Prix** : Très variables
- **Disponibilité** : Disponibilité très limitée
- **Variété** : Variété très limitée

## 4.4.2.3. Conditions de collecte

Certaines ressources ont des conditions de collecte spécifiques.

### Saisonniers

**Ressources saisonnières** :
- **Printemps** : Herbes médicinales, fruits précoces
- **Été** : Fruits, légumes, récoltes principales
- **Automne** : Récoltes tardives, champignons
- **Hiver** : Ressources limitées, certaines indisponibles

**Gestion** :
- **Planification** : Planifier la collecte selon les saisons
- **Stockage** : Stocker les ressources pour les saisons difficiles
- **Opportunités** : Profiter des saisons favorables

### Zones protégées

**Zones protégées** :
- **Réserves** : Zones protégées par les autorités
- **Sanctuaires** : Zones sacrées protégées
- **Territoires** : Territoires de créatures puissantes

**Risques** :
- **Légaux** : Risques légaux de collecte illégale
- **Créatures** : Risques de rencontres avec des créatures puissantes
- **Magie** : Risques magiques dans certaines zones

### Dangers

**Zones dangereuses** :
- **Donjons** : Zones souterraines dangereuses
- **Terres sauvages** : Zones non civilisées
- **Zones corrompues** : Terres affectées par la magie maléfique

**Préparations** :
- **Équipement** : Équipement approprié nécessaire
- **Compagnons** : Voyager en groupe recommandé
- **Précautions** : Prendre des précautions supplémentaires
