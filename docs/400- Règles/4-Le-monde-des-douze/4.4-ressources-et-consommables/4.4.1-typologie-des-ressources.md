# 4.4.1. Typologie des ressources

**Description** : Distinguer les ressources provenant des créatures, de la récolte, des minerais, des reliques, etc.

> **Règles rapides - Typologie des ressources**
> - **Ressources animales** : Peaux, poils, os, venins (provenant des créatures)
> - **Ressources végétales** : Herbes, bois, champignons (récolte)
> - **Ressources minérales** : Minerais, pierres précieuses, cristaux wakfu (minage)
> - **Ressources mystiques** : Essences, reliques, fragments de Dofus (sources magiques)

## Contenu

- **4.4.1.1. Ressources animales** : Peaux, poils, os, venins.
- **4.4.1.2. Ressources végétales** : Herbes, bois, champignons.
- **4.4.1.3. Ressources minérales** : Minerais, pierres précieuses, cristaux wakfu.
- **4.4.1.4. Ressources mystiques** : Essences, reliques, fragments de Dofus.

## 4.4.1.1. Ressources animales

Les ressources animales proviennent des créatures vaincues ou chassées.

### Peaux

**Types** :
- **Peaux communes** : Peaux de créatures communes (Bouftous, Wabbits)
- **Peaux rares** : Peaux de créatures rares (Créatures spéciales)
- **Peaux épiques** : Peaux de créatures épiques (Boss, créatures légendaires)

**Usage** :
- **Cordonnerie** : Fabrication de bottes et chaussures
- **Couture** : Fabrication de vêtements et capes
- **Armurerie** : Renforcement d'armures

**Exemples** :
- **Peau de Bouftou** : Peau commune, utilisée en cordonnerie
- **Peau de Tofu** : Peau magique, utilisée en couture
- **Peau de Craqueleur** : Peau épique, utilisée en armurerie

### Poils

**Types** :
- **Poils communs** : Poils de créatures communes
- **Poils rares** : Poils de créatures rares
- **Poils épiques** : Poils de créatures épiques

**Usage** :
- **Couture** : Fabrication de vêtements
- **Artisanat** : Fabrication d'objets divers
- **Alchimie** : Composants pour potions

**Exemples** :
- **Laine de Bouftou** : Laine commune, utilisée en couture
- **Poils magiques** : Poils rares, utilisés en alchimie

### Os

**Types** :
- **Os communs** : Os de créatures communes
- **Os rares** : Os de créatures rares
- **Os épiques** : Os de créatures épiques

**Usage** :
- **Artisanat** : Fabrication d'objets
- **Alchimie** : Composants pour potions
- **Forge** : Renforcement d'armes

**Exemples** :
- **Os de créature** : Os communs, utilisés en artisanat
- **Os magiques** : Os rares, utilisés en alchimie

### Venins

**Types** :
- **Venins communs** : Venins de créatures communes
- **Venins rares** : Venins de créatures rares
- **Venins épiques** : Venins de créatures épiques

**Usage** :
- **Alchimie** : Fabrication de poisons
- **Artisanat** : Traitement d'armes
- **Médecine** : Composants pour antidotes

**Exemples** :
- **Venin de Chafers** : Venin commun, utilisé en alchimie
- **Venin pur** : Venin rare, utilisé en médecine

Les ressources végétales proviennent de la récolte dans la nature.

**Types** :
- **Herbes communes** : Herbes médicinales de base

- **Alchimie** : Fabrication de potions
- **Médecine** : Soins et traitements
- **Cuisine** : Assaisonnement et préparation

**Exemples** :
- **Herbe médicinale** : Herbe commune, utilisée en alchimie
- **Herbe magique** : Herbe rare, utilisée en potions puissantes

### Bois

**Types** :
- **Bois communs** : Bois de base
- **Bois rares** : Bois de qualité supérieure
- **Bois épiques** : Bois légendaires

**Usage** :
- **Forge** : Fabrication d'armes et d'outils

**Exemples** :
- **Bois d'Amakna** : Bois commun, utilisé en forge

### Champignons

**Types** :

- **Champignons rares** : Champignons magiques
- **Champignons épiques** : Champignons légendaires

**Usage** :
- **Alchimie** : Fabrication de potions

- **Médecine** : Composants pour traitements

**Exemples** :
- **Champignon comestible** : Champignon commun, utilisé en cuisine
- **Champignon magique** : Champignon rare, utilisé en alchimie

### Fruits et légumes

**Types** :
- **Fruits communs** : Fruits de base
- **Légumes communs** : Légumes de base
- **Variétés rares** : Fruits et légumes rares

**Usage** :
- **Cuisine** : Préparation de plats
- **Alchimie** : Composants pour potions
- **Commerce** : Vente directe

**Exemples** :
- **Pommes** : Fruits communs, utilisés en cuisine
- **Légumes** : Légumes communs, utilisés en cuisine

**Exemples** :

## 4.4.1.4. Ressources mystiques

Les ressources mystiques sont des objets magiques et rares.

### Essences

**Types** :
- **Essences élémentaires** : Feu, eau, terre, air
- **Essences magiques** : Essences de Wakfu
- **Essences légendaires** : Essences divines

**Usage** :
- **Alchimie** : Fabrication de potions puissantes
- **Forgemagie** : Composants pour runes
- **Artisanat magique** : Fabrication d'objets magiques

**Exemples** :
- **Essence de feu** : Essence élémentaire, utilisée en alchimie
- **Essence de Wakfu** : Essence magique, utilisée en forgemagie

### Reliques

**Types** :
- **Reliques communes** : Reliques de base
- **Reliques rares** : Reliques de qualité
- **Reliques épiques** : Reliques légendaires

**Usage** :
- **Artisanat magique** : Fabrication d'objets magiques
- **Forgemagie** : Composants pour runes puissantes
- **Commerce** : Vente de très grande valeur

**Exemples** :
- **Relique ancienne** : Relique rare, utilisée en artisanat
- **Relique légendaire** : Relique épique, utilisée en forgemagie

### Fragments de Dofus

**Types** :
- **Fragments** : Fragments de Dofus
- **Fragments purs** : Fragments de grande qualité
- **Dofus complets** : Dofus entiers (extrêmement rares)

**Usage** :
- **Artisanat légendaire** : Fabrication d'objets légendaires
- **Forgemagie** : Composants pour runes légendaires
- **Quêtes** : Objets de quêtes épiques

**Exemples** :
- **Fragment de Dofus** : Fragment rare, utilisé en artisanat
- **Dofus complet** : Dofus légendaire, objet de quête
