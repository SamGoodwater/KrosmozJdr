# 4.4.3. Transformation et stockage

**Description** : Règles sur la transformation des ressources (raffinage, préparation) et leur conservation (durée, conditions).

> **Règles rapides - Transformation et stockage**
> - **Raffinage** : Purification des ressources (10-25% de perte), processus spécifiques
> - **Préparation** : Kits, ateliers requis, tests de compétence
> - **Stockage** : Conditions (température, humidité), dégradation, conservation magique
> - **Transport** : Poids, volumes, sécurisation (risques de vol/perte)

## Contenu

- **4.4.3.1. Raffinage** : Processus de purification, pertes éventuelles.
- **4.4.3.2. Préparation** : Kits, ateliers requis, tests.

## 4.4.3.1. Raffinage

Le raffinage permet de purifier et améliorer les ressources brutes.

### Processus de purification

**Minerais** :
- **Processus** : Fonte et purification des minerais
- **Résultat** : Métaux purs utilisables
- **Pertes** : 10-20% de perte lors du raffinage

**Essences** :
- **Processus** : Distillation et purification des essences
- **Résultat** : Essences pures
- **Pertes** : 15-25% de perte lors du raffinage

**Herbes** :
- **Processus** : Séchage et préparation des herbes
- **Résultat** : Herbes préparées
- **Pertes** : 5-15% de perte lors du raffinage

### Pertes éventuelles

**Pertes normales** :
- **Taux** : 10-25% selon le type de ressource
- **Causes** : Imperfections du processus, impuretés
- **Réduction** : Meilleurs outils et compétences réduisent les pertes

**Pertes critiques** :
- **Échec critique** : Échec critique peut causer des pertes importantes
- **Taux** : 50-100% de perte en cas d'échec critique
- **Prévention** : Tests de compétence pour éviter les échecs critiques

---

## 4.4.3.2. Préparation

La préparation transforme les ressources en matériaux utilisables.

### Kits

**Kits de préparation** :
- **Types** : Kits spécialisés selon le type de ressource
- **Usage** : Facilite la préparation
- **Bonus** : Bonus aux tests de préparation

**Exemples** :
- **Kit d'alchimie** : Pour préparer les herbes et essences
- **Kit de forge** : Pour préparer les minerais
- **Kit de cuisine** : Pour préparer les aliments

### Ateliers requis

**Ateliers de base** :
- **Accès** : Accès à un atelier équipé nécessaire
- **Bonus** : Bonus aux tests de préparation
- **Coût** : Peut nécessiter un paiement ou une location

**Ateliers spécialisés** :
- **Bonus** : Bonus supplémentaires aux tests
- **Rareté** : Plus rares et chers
- **Accès** : Nécessitent souvent une réputation ou des relations

### Tests

**Tests de préparation** :
- **Formule** : 1d20 + Modificateur + Bonus
- **DD** : Variable selon la complexité
- **Résultat** : Succès ou échec avec conséquences

**Exemples** :
- **Préparation simple** : DD 10-15
- **Préparation complexe** : DD 15-20
- **Préparation épique** : DD 20-25

## 4.4.3.3. Stockage

Le stockage des ressources nécessite des conditions appropriées.

### Conditions

**Température** :
- **Ressources sensibles** : Nécessitent une température contrôlée
- **Conservation** : Meilleure conservation dans des conditions optimales
- **Dégradation** : Dégradation accélérée dans de mauvaises conditions

**Humidité** :
- **Ressources sensibles** : Nécessitent un contrôle de l'humidité
- **Conservation** : Meilleure conservation avec un contrôle de l'humidité
- **Dégradation** : Dégradation accélérée avec trop ou pas assez d'humidité

**Protection** :
- **Conteneurs** : Conteneurs appropriés pour chaque type de ressource
- **Sécurité** : Protection contre le vol et la détérioration
- **Organisation** : Organisation pour faciliter l'accès

### Dégradation

**Dégradation normale** :
- **Taux** : Dégradation lente avec le temps
- **Prévention** : Bonnes conditions de stockage ralentissent la dégradation
- **Gestion** : Utiliser les ressources avant qu'elles ne se dégradent

**Dégradation accélérée** :
- **Causes** : Mauvaises conditions, négligence
- **Taux** : Dégradation accélérée
- **Prévention** : Maintenir de bonnes conditions de stockage

### Conservation magique

**Sorts de conservation** :
- **Types** : Sorts qui préservent les ressources
- **Durée** : Durée limitée selon le sort
- **Coût** : Coût en ressources magiques

**Artefacts** :
- **Types** : Artefacts qui préservent les ressources
- **Durée** : Conservation permanente
- **Rareté** : Très rares et chers

**Avantages** :
- **Préservation** : Préservation parfaite des ressources
- **Durée** : Pas de dégradation
- **Valeur** : Maintien de la valeur

## 4.4.3.4. Transport

Le transport des ressources nécessite une gestion appropriée.

### Poids

**Poids des ressources** :
- **Ressources légères** : Herbes, essences (0,1-0,5 kg par unité)
- **Ressources moyennes** : Minerais, bois (1-5 kg par unité)
- **Ressources lourdes** : Métaux, pierres (5-20 kg par unité)

**Gestion** :
- **Capacité** : Limite de capacité de transport
- **Montures** : Utilisation de montures pour transporter
- **Optimisation** : Optimiser le transport selon le poids

### Volumes

**Volumes des ressources** :
- **Ressources compactes** : Minerais, métaux (petit volume)
- **Ressources volumineuses** : Bois, peaux (grand volume)
- **Ressources fragiles** : Herbes, essences (nécessitent protection)

**Gestion** :
- **Conteneurs** : Utilisation de conteneurs appropriés
- **Organisation** : Organisation pour optimiser l'espace
- **Protection** : Protection des ressources fragiles

### Sécurisation

**Protection physique** :
- **Conteneurs** : Conteneurs sécurisés
- **Verrouillage** : Verrouillage des conteneurs
- **Surveillance** : Surveillance des ressources

**Protection magique** :
- **Sorts** : Sorts de protection
- **Artefacts** : Artefacts de protection
- **Coût** : Coût en ressources magiques

**Risques** :
- **Vol** : Risque de vol
- **Détérioration** : Risque de détérioration
- **Perte** : Risque de perte
